= Katello {KatelloVersion} Release Notes

[id="katello-headline-features"]
== Headline Features

=== Transient Package Installation on Image Mode Hosts via REX

Remote Execution job templates for package and errata installation now support image mode hosts by applying updates as transient changes, compatible with their immutable design.


=== Container Gateway Support for Load-Balanced setups

The container gateway can now redirect clients to Pulp content using a configurable FQDN, enabling proper routing in load-balanced setups. This ensures clients are redirected to the load balancer instead of the originating host, allowing traffic to reach the correct capsule.


=== Rolling Content Views

Introduces a new type of content view for direct, always-up-to-date access to repository content from Library—without versions, filters, or publishing overhead. Ideal for lightweight setups, this reduces storage and sync time by avoiding content duplication across Pulp repositories.
