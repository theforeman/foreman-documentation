= Editing a Compliance Policy

In the Satellite web UI, edit the compliance policy.

.Procedure

. Navigate to *Hosts* > *Policies*.
. From the drop-down list to the right of the policy's name, select *Edit*.
. Edit the necessary attributes.
. Click *Submit*.

An edited policy is applied to the host when its Puppet agent next checks with the Satellite{nbsp}Server for updates. By default this occurs every 30 minutes.
