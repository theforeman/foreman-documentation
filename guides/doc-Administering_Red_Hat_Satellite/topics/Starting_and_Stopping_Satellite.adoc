== Starting and Stopping {ProjectName}

{Project} provides the `{foreman-maintain} service` command to manage {Project} services from the command line. This is useful when creating a backup of {Project}. For more information on creating backups, see xref:backing-up-satellite-server-and-capsule-server[].

After installing {Project} with the `{foreman-installer}` command, all {Project} services are started and enabled automatically. View the list of these services by executing:

[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-maintain} service list
----

To see the status of running services, execute:

[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-maintain} service status
----

To stop the `{foreman-maintain}` services, execute:

[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-maintain} service stop
----

To start the `{foreman-maintain}` services, execute:

[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-maintain} service start
----

To restart the `{foreman-maintain}` services, execute:

[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-maintain} service restart
----
