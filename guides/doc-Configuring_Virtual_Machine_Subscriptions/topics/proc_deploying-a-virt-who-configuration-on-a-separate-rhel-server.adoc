= Deploying a virt-who configuration on a separate Red{nbsp}Hat Enterprise Linux server

Use this procedure to deploy a vCenter or Hyper-V virt-who configuration on a dedicated Red{nbsp}Hat Enterprise Linux 7 server. The server can be physical or virtual.

Global configuration values apply to all virt-who configurations on this server, and are overwritten each time a new virt-who configuration is deployed.

.Prerequisites

* Create a read-only virt-who user on the hypervisor or virtualization manager.
* If you are deploying a Hyper-V virt-who configuration, enable remote management on the Hyper-V hypervisor.
* Create a virt-who configuration for your virtualization platform.

.Procedure

. On the Red{nbsp}Hat Enterprise Linux server, install {ProjectServer}'s CA certificate:
+
[options="nowrap", subs="+quotes,attributes"]
----
# rpm -ivh http://__{foreman-example-com}__/pub/katello-ca-consumer-latest.noarch.rpm
----

. Register the Red{nbsp}Hat Enterprise{nbsp}Linux server to {ProjectServer}:
+
[options="nowrap", subs="+quotes,attributes"]
----
# subscription-manager register \
--org=_organization_label_ \
--auto-attach
----

. Open a network port for communication between virt-who and {ProjectServer}:
+
[options="nowrap"]
----
# firewall-cmd --add-port="443/tcp"
# firewall-cmd --add-port="443/tcp" --permanent
----

. Open a network port for communication between virt-who and each hypervisor or virtualization manager:

* VMware vCenter: TCP port 443
* Microsoft Hyper-V: TCP port 5985

. In the {ProjectWebUI}, navigate to *Infrastructure* > *Virt-who configurations*.
. Click the name of the virt-who configuration file.
. Click the *Deploy* tab.
. Under *Configuration script*, click *Download the script*.
. Copy the script to the Red{nbsp}Hat Enterprise Linux server:
+
[options="nowrap", subs="+quotes,attributes"]
----
# scp _deploy_virt_who_config_1_.sh root@_rhel.example.com_:
----

. Make the deployment script executable and run it:
+
[options="nowrap", subs="+quotes,attributes"]
----
# chmod +x _deploy_virt_who_config_1_.sh
# sh _deploy_virt_who_config_1_.sh
----

. After the deployment is complete, delete the script:
+
[options="nowrap", subs="+quotes,attributes"]
----
# rm _deploy_virt_who_config_1_
----
