= Deploying a virt-who configuration on {ProjectServer}

Use this procedure to deploy a vCenter or Hyper-V virt-who configuration on {ProjectServer}.

Global configuration values apply to all virt-who configurations on {ProjectServer}, and are overwritten each time a new virt-who configuration is deployed.

.Prerequisites

* Create a read-only virt-who user on the hypervisor or virtualization manager.
* If you are deploying a Hyper-V virt-who configuration, enable remote management on the Hyper-V hypervisor.
* Create a virt-who configuration for your virtualization platform.

.Procedure

. In the {ProjectWebUI}, navigate to *Infrastructure* > *Virt-who configurations*.
. Click the name of the virt-who configuration.
. Under *Hammer command*, click *Copy to clipboard*.
. On {ProjectServer}, paste the Hammer command into your terminal.
