[id="hammer-report-template-schedule"]
= report-template schedule

[role="_abstract"]
Schedule generating of a report

*Usage*

----
# hammer report-template schedule [OPTIONS]
----

*Options*

* `--generate-at VALUE` {endash} UTC time to generate report at
* `--gzip BOOLEAN` {endash} Compress the report using gzip
* `--id VALUE`
* `--inputs KEY_VALUE_LIST` {endash} Specify inputs
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-title VALUE` {endash} Set the current location context for the request
* `--mail-to VALUE` {endash} If set, scheduled report will be delivered via e-mail. Use `,` to separate
multiple email addresses.
* `--name VALUE` {endash} Name to search by
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--path VALUE` {endash} Path to directory where downloaded content will be saved. Only usable if wait is
specified
* `--report-format ENUM` {endash} Report format, defaults to `csv`
Possible value(s): `csv`, `json`, `yaml`, `html`
* `--wait` {endash} Turns a command to be active, wait for the result and download it right away
* `-h`, `--help` {endash} Print help
