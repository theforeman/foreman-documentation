[id="hammer-settings-set"]
= settings set

[role="_abstract"]
Update a setting

*Usage*

----
# hammer settings set [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Setting name
* `--value VALUE`
* `-h`, `--help` {endash} Print help
