[id="hammer-os-remove-architecture"]
= os remove-architecture

[role="_abstract"]
Disassociate an architecture

*Usage*

----
# hammer os remove-architecture [OPTIONS]
----

*Options*

* `--architecture VALUE` {endash} Architecture name
* `--architecture-id NUMBER`
* `--id VALUE`
* `--title VALUE` {endash} Operating system title
* `-h`, `--help` {endash} Print help
