[id="hammer-activation-key-create"]
= activation-key create

[role="_abstract"]
Create an activation key

*Usage*

----
# hammer activation-key create [OPTIONS]
----

*Options*

* `--auto-attach BOOLEAN` {endash} Auto attach subscriptions upon registration
* `--content-view VALUE` {endash} Content view name to search by
* `--content-view-environment-ids LIST` Array of content view environment ids to be associated with the activation key.
Ignored if content_view_id and lifecycle_environment_id are specified. Requires
allow_multiple_content_views setting to be on.
* `--content-view-environments LIST` {endash} Comma-separated list of content view environment labels to be associated with
the activation key, in the format of
`lifecycle_environment_label/content_view_label`. Ignored if
content_view_environment_ids is specified, or if content_view_id and
lifecycle_environment_id are specified. Requires allow_multiple_content_views
setting to be on.
* `--content-view-id NUMBER` {endash} Id of the single content view to be associated with the activation key.
* `--description VALUE` {endash} Description
* `--environment VALUE` {endash} Lifecycle environment name to search by (--environment is deprecated: Use
* `--lifecycle-environment` instead)
* `--environment-id NUMBER` {endash} (--environment-id is deprecated: Use `--lifecycle-environment-id` instead)
* `--lifecycle-environment VALUE` {endash} Lifecycle environment name to search by
* `--lifecycle-environment-id NUMBER` {endash} Id of the single lifecycle environment to be associated with the activation key.
* `--max-hosts NUMBER` {endash} Maximum number of registered content hosts
* `--name VALUE` {endash} Name
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization identifier
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--purpose-role VALUE` {endash} Sets the system purpose usage
* `--purpose-usage VALUE` {endash} Sets the system purpose usage
* `--release-version VALUE` {endash} Content release version
* `--service-level VALUE` {endash} Service level
* `--unlimited-hosts` {endash} Set hosts max to unlimited
* `-h`, `--help` {endash} Print help
