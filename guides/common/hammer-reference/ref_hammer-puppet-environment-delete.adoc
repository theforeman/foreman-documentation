= puppet-environment delete

[role="_abstract"]
Delete an environment

*Usage*

----
# hammer puppet-environment <delete|destroy> [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Puppet environment name
* `-h`, `--help` {endash} Print help
