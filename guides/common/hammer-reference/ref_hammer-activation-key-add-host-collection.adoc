[id="hammer-activation-key-add-host-collection"]
= activation-key add-host-collection

[role="_abstract"]
Associate a resource

*Usage*

----
# hammer activation-key add-host-collection [OPTIONS]
----

*Options*

* `--host-collection VALUE` {endash} Host collection name to search by
* `--host-collection-id NUMBER` {endash} Id of the host collection
* `--id VALUE` {endash} ID of the activation key
* `--name VALUE` {endash} Activation key name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization ID
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `-h`, `--help` {endash} Print help
