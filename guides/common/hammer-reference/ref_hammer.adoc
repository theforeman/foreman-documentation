[id="hammer"]
= hammer

[role="_abstract"]

*Usage*

----
# hammer [OPTIONS] SUBCOMMAND [ARG] ...
----



*Options*

* `--[no-]use-defaults` {endash} Enable/disable stored defaults. Enabled by default
* `--autocomplete VALUE` {endash} Get list of possible endings
* `--csv` {endash} Output as CSV (same as `--output=csv)`
* `--csv-separator VALUE` {endash} Character to separate the values
* `--fetch-ca-cert VALUE` {endash} Fetch CA certificate from server and exit
* `--interactive BOOLEAN` {endash} Explicitly turn interactive mode on/off
* `--no-headers` {endash} Hide headers from output
* `--output ENUM` {endash} Set output format
Possible value(s): `base`, `table`, `silent`, `csv`, `yaml`, `json`
* `--output-file VALUE` {endash} Path to custom output file
* `--show-ids` {endash} Show ids of associated resources
* `--ssl-ca-file VALUE` {endash} Configure the file containing the CA certificates
* `--ssl-ca-path VALUE` {endash} Configure the directory containing the CA certificates
* `--ssl-client-cert VALUE` {endash} Configure the client`s public certificate
* `--ssl-client-key VALUE` {endash} Configure the client`s private key
* `--ssl-with-basic-auth` {endash} Use standard authentication in addition to client certificate authentication
* `--verify-ssl BOOLEAN` {endash} Configure SSL verification of remote system
* `--version` {endash} Show version
* `-c`, `--config VALUE` {endash} Path to custom config file
* `-d`, `--debug` {endash} Show debugging output
* `-h`, `--help` {endash} Print help
* `-p`, `--password VALUE` {endash} Password to access the remote system
* `-q`, `--quiet` {endash} Completely silent
* `-r`, `--reload-cache` {endash} Force reload of Apipie cache
* `-s`, `--server VALUE` {endash} Remote system address
* `-u`, `--username VALUE` {endash} Username to access the remote system
* `-v`, `--[no-]verbose` {endash} Be verbose (or not). True by default
