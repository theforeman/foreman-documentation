[id="hammer-organization-add-domain"]
= organization add-domain

[role="_abstract"]
Associate a domain

*Usage*

----
# hammer organization add-domain [OPTIONS]
----

*Options*

* `--domain VALUE` {endash} Domain name
* `--domain-id NUMBER` {endash} Numerical ID or domain name
* `--id VALUE` {endash} Organization ID
* `--name VALUE` {endash} Set the current organization context for the request
* `--title VALUE` {endash} Set the current organization context for the request
* `-h`, `--help` {endash} Print help
