= user-group remove-user

[role="_abstract"]
Disassociate an user

*Usage*

----
# hammer user-group remove-user [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Name to search by
* `--user VALUE` {endash} User`s login to search by
* `--user-id NUMBER`
* `-h`, `--help` {endash} Print help
