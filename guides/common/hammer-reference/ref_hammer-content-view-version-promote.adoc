[id="hammer-content-view-version-promote"]
= content-view version promote

[role="_abstract"]
Promote a content view version

*Usage*

----
# hammer content-view version promote [OPTIONS]
----

*Options*

* `--async` {endash} Do not wait for the task
* `--content-view VALUE` {endash} Content view name to search by
* `--content-view-id NUMBER` {endash} Content view numeric identifier
* `--description VALUE` {endash} The description for the content view version promotion
* `--force` {endash} Force content view promotion and bypass lifecycle environment restriction
* `--from-lifecycle-environment VALUE` {endash} Environment name from where to promote its version from (if version is unknown)
* `--from-lifecycle-environment-id VALUE` Id of the environment from where to promote its version from (if version is
unknown)
* `--id NUMBER` {endash} Content view version identifier
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization ID
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--to-lifecycle-environment VALUE` {endash} Name of the target environment
* `--to-lifecycle-environment-id VALUE` {endash} Id of the target environment
* `--version VALUE` {endash} Content view version number
* `-h`, `--help` {endash} Print help
