[id="hammer-content-view-filter-list"]
= content-view filter list

[role="_abstract"]
list filters

*Usage*

----
# hammer content-view filter <list|index> [OPTIONS]
----

*Options*

* `--content-view VALUE` {endash} Content view name to search by
* `--content-view-id NUMBER` {endash} Content view identifier
* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--full-result BOOLEAN` {endash} Whether or not to show all results
* `--name VALUE` {endash} Filter content view filters by name
* `--order VALUE` {endash} Sort field and order, eg. `id DESC`
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id VALUE` {endash} Organization ID to search by
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title to search by
* `--page NUMBER` {endash} Page number, starting at 1
* `--per-page NUMBER` {endash} Number of results per page to return
* `--search VALUE` {endash} Search string
* `--types LIST` {endash} Types of filters
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS      | ALL | DEFAULT | THIN

| Filter id   | x   | x       | x
| Name        | x   | x       | x
| Description | x   | x       |
| Type        | x   | x       |
| Inclusion   | x   | x       |
|===

*Search and order fields*

* `content_type` {endash} Values: rpm, deb, package_group, erratum, docker, modulemd
* `inclusion_type` {endash} Values: include, exclude
* `name` {endash} string
