[id="hammer-location-remove-user"]
= location remove-user

[role="_abstract"]
Disassociate an user

*Usage*

----
# hammer location remove-user [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Set the current location context for the request
* `--title VALUE` {endash} Set the current location context for the request
* `--user VALUE` {endash} User`s login to search by
* `--user-id NUMBER`
* `-h`, `--help` {endash} Print help
