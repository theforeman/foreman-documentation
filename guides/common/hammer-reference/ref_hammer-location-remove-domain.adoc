= location remove-domain

[role="_abstract"]
Disassociate a domain

*Usage*

----
# hammer location remove-domain [OPTIONS]
----

*Options*

* `--domain VALUE` {endash} Domain name
* `--domain-id NUMBER` {endash} Numerical ID or domain name
* `--id VALUE`
* `--name VALUE` {endash} Set the current location context for the request
* `--title VALUE` {endash} Set the current location context for the request
* `-h`, `--help` {endash} Print help
