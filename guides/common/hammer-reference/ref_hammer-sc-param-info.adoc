[id="hammer-sc-param-info"]
= sc-param info

[role="_abstract"]
Show a smart class parameter

*Usage*

----
# hammer sc-param <info|show> [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--id VALUE`
* `--name VALUE` {endash} Smart class parameter name
* `--puppet-class VALUE` {endash} Puppet class name
* `--puppet-class-id NUMBER` {endash} ID of Puppet class
* `--show-hidden BOOLEAN` {endash} Display hidden values
* `-h`, `--help` {endash} Print help
