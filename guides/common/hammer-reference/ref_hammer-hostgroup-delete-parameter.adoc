[id="hammer-hostgroup-delete-parameter"]
= hostgroup delete-parameter

[role="_abstract"]
Delete parameter for a hostgroup

*Usage*

----
# hammer hostgroup delete-parameter [OPTIONS]
----

*Options*

* `--hostgroup VALUE` {endash} Hostgroup name
* `--hostgroup-id NUMBER`
* `--hostgroup-title VALUE` {endash} Hostgroup title
* `--name VALUE` {endash} Parameter name
* `-h`, `--help` {endash} Print help
