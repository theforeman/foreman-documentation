[id="hammer-content-view-purge"]
= content-view purge

[role="_abstract"]
Delete old versions of a content view

*Usage*

----
# hammer content-view purge [OPTIONS]
----

*Options*

* `--async` {endash} Do not wait for the task
* `--count NUMBER` {endash} (deprecated) Number of versions to keep
* `--id VALUE` {endash} Content View numeric identifier
* `--name VALUE` {endash} Content View name
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id VALUE` {endash} Organization ID to search by
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title to search by
* `--versions-to-keep NUMBER` {endash} Number of unused versions to keep
Default: 3
* `-h`, `--help` {endash} Print help
