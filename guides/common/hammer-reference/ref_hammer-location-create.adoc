[id="hammer-location-create"]
= location create

[role="_abstract"]
Create a location

*Usage*

----
# hammer location create [OPTIONS]
----

*Options*

* `--compute-resource-ids LIST` {endash} Compute resource IDs
* `--compute-resources LIST`
* `--description VALUE`
* `--domain-ids LIST` {endash} Domain IDs
* `--domains LIST`
* `--environment-ids LIST` {endash} Environment IDs
* `--hostgroup-ids LIST` {endash} Host group IDs
* `--hostgroup-titles LIST`
* `--hostgroups LIST`
* `--ignore-types LIST` {endash} List of resources types that will be automatically associated
* `--media LIST`
* `--medium-ids LIST` {endash} Medium IDs
* `--name VALUE`
* `--organization-ids LIST` {endash} Associated organization IDs
* `--parent-id NUMBER` {endash} Parent ID
* `--partition-table-ids LIST` {endash} Partition template IDs
* `--partition-tables LIST`
* `--provisioning-template-ids LIST` Provisioning template IDs
* `--provisioning-templates LIST`
* `--realm-ids LIST` {endash} Realm IDs
* `--realms LIST`
* `--smart-proxies LIST`
* `--smart-proxy-ids LIST` {endash} Capsule IDs
* `--subnet-ids LIST` {endash} Subnet IDs
* `--subnets LIST`
* `--user-ids LIST` {endash} User IDs
* `--users LIST`
* `-h`, `--help` {endash} Print help
