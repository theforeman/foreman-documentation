[id="hammer-activation-key-add-subscription"]
= activation-key add-subscription

[role="_abstract"]
Add subscription

*Usage*

----
# hammer activation-key add-subscription [OPTIONS]
----

*Options*

* `--id NUMBER` {endash} ID of the activation key
* `--name VALUE` {endash} Activation key name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER`
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--quantity NUMBER` {endash} Quantity of this subscription to add
* `--subscription VALUE` {endash} Subscription name to search by
* `--subscription-id NUMBER` {endash} Subscription identifier
* `--subscriptions SCHEMA` {endash} Array of subscriptions to add
* `-h`, `--help` {endash} Print help

Following parameters accept format defined by its schema (bold are required; <> contains acceptable type; [] contains acceptable value):

* `--subscriptions` {endash} "id=<string>,quantity=<numeric>, ... "
