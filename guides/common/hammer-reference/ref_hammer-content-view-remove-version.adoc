[id="hammer-content-view-remove-version"]
= content-view remove-version

[role="_abstract"]
Remove a content view version from a composite view

*Usage*

----
# hammer content-view remove-version [OPTIONS]
----

*Options*

* `--content-view VALUE` {endash} Content view name to search by
* `--content-view-id NUMBER` {endash} Content view numeric identifier
* `--content-view-version VALUE` {endash} Content view version number
* `--content-view-version-id NUMBER` Content view version identifier
* `--id VALUE` {endash} Content view numeric identifier
* `--name VALUE` {endash} Content view name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id VALUE` {endash} Organization ID to search by
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title to search by
* `-h`, `--help` {endash} Print help
