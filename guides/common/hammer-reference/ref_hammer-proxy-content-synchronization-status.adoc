[id="hammer-proxy-content-synchronization-status"]
= proxy content synchronization-status

[role="_abstract"]
Get current Capsule synchronization status

*Usage*

----
# hammer proxy content synchronization-status [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--id NUMBER` {endash} Id of the Capsule
* `--name VALUE` {endash} Name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Id of the organization to get the status for
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS                                | ALL | DEFAULT

| Last sync                             | x   | x
| Status                                | x   | x
| Currently running sync tasks/task id  | x   | x
| Currently running sync tasks/progress | x   | x
| Last failure/task id                  | x   | x
| Last failure/messages                 | x   | x
|===
