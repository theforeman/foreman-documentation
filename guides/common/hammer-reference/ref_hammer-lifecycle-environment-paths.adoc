[id="hammer-lifecycle-environment-paths"]
= lifecycle-environment paths

[role="_abstract"]
List environment paths

*Usage*

----
# hammer lifecycle-environment paths [OPTIONS]
----

*Options*

* `--content-source-id NUMBER` {endash} Show whether each lifecycle environment is associated with the given Capsule id.
* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization identifier
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--permission-type VALUE` {endash} The associated permission type. One of (readable | promotable) Default: readable
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS         | ALL | DEFAULT

| Lifecycle path | x   | x
|===
