[id="hammer-job-template-import"]
= job-template import

[role="_abstract"]
Import a job template from ERB

*Usage*

----
# hammer job-template import [OPTIONS]
----

*Options*

* `--associate ENUM` {endash} Determines when the template should associate objects based on metadata, new
means only when new template is being created, always means both for new and
existing template which is only being updated, never ignores metadata
Possible value(s): `new`, `always`, `never`
* `--default BOOLEAN` {endash} Makes the template default meaning it will be automatically associated with
newly created organizations and locations (false by default)
* `--file FILE` {endash} Path to a file that contains the template - must include ERB metadata
* `--force BOOLEAN` {endash} Use if you want update locked templates
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-title VALUE` {endash} Set the current location context for the request
* `--lock BOOLEAN` {endash} Lock imported templates (false by default)
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--overwrite BOOLEAN` {endash} Overwrite template if it already exists
* `-h`, `--help` {endash} Print help
