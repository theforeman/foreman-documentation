[id="hammer-organization-configure-cdn"]
= organization configure-cdn

[role="_abstract"]
Update the CDN configuration

*Usage*

----
# hammer organization configure-cdn [OPTIONS]
----

*Options*

* `--custom-cdn-auth-enabled BOOLEAN` {endash} If product certificates should be used to authenticate to a custom CDN.
* `--id VALUE` {endash} ID of the Organization
* `--label VALUE` {endash} Organization label to search by
* `--name VALUE` {endash} Organization name to search by
* `--password VALUE` {endash} Password for authentication. Relevant only for `upstream_server` type.
* `--ssl-ca-credential-id NUMBER` {endash} Content Credential to use for SSL CA. Relevant only for `upstream_server` type.
* `--title VALUE` {endash} Organization title
* `--type VALUE` {endash} CDN configuration type. One of redhat_cdn, network_sync, export_sync,
custom_cdn.
* `--upstream-content-view-label VALUE` {endash} Upstream Content View Label, default: Default_Organization_View. Relevant only
for `upstream_server` type.
* `--upstream-lifecycle-environment-label VALUE` Upstream Lifecycle Environment, default: Library. Relevant only for
`upstream_server` type.
* `--upstream-organization-label VALUE` {endash} Upstream organization to sync CDN content from. Relevant only for
`upstream_server` type.
* `--url VALUE` {endash} Upstream Satellite server to sync CDN content from. Relevant only for
`upstream_server` type.
* `--username VALUE` {endash} Username for authentication. Relevant only for `upstream_server` type.
* `-h`, `--help` {endash} Print help
