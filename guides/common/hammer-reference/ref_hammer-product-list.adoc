[id="hammer-product-list"]
= product list

[role="_abstract"]
List products

*Usage*

----
# hammer product <list|index> [OPTIONS]
----

*Options*

* `--available-for VALUE` {endash} Interpret specified object to return only Products that can be associated with
specified object.  Only `sync_plan` is supported.
* `--custom BOOLEAN` {endash} Return custom products only
* `--enabled BOOLEAN` {endash} Return enabled products only
* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--full-result BOOLEAN` {endash} Whether or not to show all results
* `--include-available-content BOOLEAN` Whether to include available content attribute in results
* `--name VALUE` {endash} Filter products by name
* `--order VALUE` {endash} Sort field and order, eg. `id DESC`
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Filter products by organization
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--page NUMBER` {endash} Page number, starting at 1
* `--per-page NUMBER` {endash} Number of results per page to return
* `--redhat-only BOOLEAN` {endash} Return Red Hat (non-custom) products only
* `--search VALUE` {endash} Search string
* `--subscription VALUE` {endash} Subscription name to search by
* `--subscription-id NUMBER` {endash} Filter products by subscription
* `--sync-plan VALUE` {endash} Sync plan name to search by
* `--sync-plan-id NUMBER` {endash} Filter products by sync plan id
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS       | ALL | DEFAULT | THIN

| Id           | x   | x       | x
| Name         | x   | x       | x
| Description  | x   | x       |
| Organization | x   | x       |
| Repositories | x   | x       |
| Sync state   | x   | x       |
|===

*Search and order fields*

* `description` {endash} text
* `label` {endash} string
* `name` {endash} string
* `organization_id` {endash} integer
* `redhat` {endash} Values: true, false
