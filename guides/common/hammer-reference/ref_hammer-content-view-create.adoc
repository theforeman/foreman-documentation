[id="hammer-content-view-create"]
= content-view create

[role="_abstract"]
Create a content view

*Usage*

----
# hammer content-view create [OPTIONS]
----

*Options*

* `--auto-publish BOOLEAN` {endash} Enable/Disable auto publish of composite view
* `--component-ids LIST` {endash} List of component content view version ids for composite views
* `--composite` {endash} Create a composite content view
* `--description VALUE` {endash} Description for the content view
* `--import-only` {endash} Designate this Content View for importing from upstream servers only.
* `--label VALUE` {endash} Content view label
* `--name VALUE` {endash} Name of the content view
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization identifier
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--repository-ids LIST` {endash} List of repository ids
* `--rolling` {endash} Create a rolling content view
* `--solve-dependencies BOOLEAN`  Solve RPM dependencies by default on Content View publish, defaults to false
* `-h`, `--help` {endash} Print help
