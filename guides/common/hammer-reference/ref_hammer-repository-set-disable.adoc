[id="hammer-repository-set-disable"]
= repository-set disable

[role="_abstract"]
Disable a repository from the set

*Usage*

----
# hammer repository-set disable [OPTIONS]
----

*Options*

* `--basearch VALUE` {endash} Basearch to disable
* `--id NUMBER` {endash} ID of the repository set to disable
* `--name VALUE` {endash} Repository set name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization identifier
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--product VALUE` {endash} Product name to search by
* `--product-id NUMBER` {endash} ID of the product containing the repository set
* `--releasever VALUE` {endash} Releasever to disable
* `--repository VALUE` {endash} Repository name to search by
* `--repository-id NUMBER` {endash} ID of the repository within the set to disable
* `-h`, `--help` {endash} Print help
