[id="hammer-job-template-create"]
= job-template create

[role="_abstract"]
Create a job template

*Usage*

----
# hammer job-template create [OPTIONS]
----

*Options*

* `--ansible-callback-enabled BOOLEAN` Enable the callback plugin for this template
* `--ansible-check-mode BOOLEAN` {endash} Enable Ansible Check Mode for this template
* `--audit-comment VALUE`
* `--current-user BOOLEAN` {endash} Whether the current user login should be used as the effective user
* `--description VALUE`
* `--description-format VALUE` {endash} This template is used to generate the description. Input values can be used
using the syntax %{package}. You may also include the job category and template
name using %{job_category} and %{template_name}.
* `--file FILE` {endash} Path to a file that contains the template
* `--job-category VALUE` {endash} Job category
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-ids LIST` {endash} REPLACE locations with given ids
* `--location-title VALUE` {endash} Set the current location context for the request
* `--location-titles LIST`
* `--locations LIST`
* `--locked BOOLEAN` {endash} Whether or not the template is locked for editing
* `--name VALUE` {endash} Template name
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-ids LIST` {endash} REPLACE organizations with given ids.
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--organization-titles LIST`
* `--organizations LIST`
* `--overridable BOOLEAN` {endash} Whether it should be allowed to override the effective user from the invocation
form.
* `--provider-type ENUM` {endash} Provider type
Possible value(s): `SSH`, `script`, `Ansible`
* `--snippet BOOLEAN`
* `--value VALUE` {endash} What user should be used to run the script (using sudo-like mechanisms)
* `-h`, `--help` {endash} Print help
