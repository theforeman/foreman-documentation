[id="hammer-template-add-operatingsystem"]
= template add-operatingsystem

[role="_abstract"]
Associate an operating system

*Usage*

----
# hammer template add-operatingsystem [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Name to search by
* `--operatingsystem VALUE` {endash} Operating system title
* `--operatingsystem-id NUMBER`
* `-h`, `--help` {endash} Print help
