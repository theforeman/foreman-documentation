[id="hammer-organization-remove-user"]
= organization remove-user

[role="_abstract"]
Disassociate an user

*Usage*

----
# hammer organization remove-user [OPTIONS]
----

*Options*

* `--id VALUE` {endash} Organization ID
* `--name VALUE` {endash} Set the current organization context for the request
* `--title VALUE` {endash} Set the current organization context for the request
* `--user VALUE` {endash} User`s login to search by
* `--user-id NUMBER`
* `-h`, `--help` {endash} Print help
