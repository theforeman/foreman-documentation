= os add-architecture

[role="_abstract"]
Associate an architecture

*Usage*

----
# hammer os add-architecture [OPTIONS]
----

*Options*

* `--architecture VALUE` {endash} Architecture name
* `--architecture-id NUMBER`
* `--id VALUE`
* `--title VALUE` {endash} Operating system title
* `-h`, `--help` {endash} Print help
