[id="hammer-lifecycle-environment-update"]
= lifecycle-environment update

[role="_abstract"]
Update an environment

*Usage*

----
# hammer lifecycle-environment update [OPTIONS]
----

*Options*

* `--async BOOLEAN` {endash} Do not wait for the update action to finish. Default: true
* `--description VALUE` {endash} Description of the environment
* `--id NUMBER` {endash} ID of the environment
* `--name VALUE` {endash} Lifecycle environment name to search by
* `--new-name VALUE` {endash} New name to be given to the environment
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Name of the organization
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--registry-name-pattern VALUE` {endash} Pattern for container image names
* `--registry-unauthenticated-pull BOOLEAN` Allow unauthenticed pull of container images
* `-h`, `--help` {endash} Print help
