[id="hammer-config-group-create"]
= config-group create

[role="_abstract"]
Create a config group

*Usage*

----
# hammer config-group create [OPTIONS]
----

*Options*

* `--name VALUE`
* `--puppet-class-ids LIST`
* `--puppet-classes LIST`
* `-h`, `--help` {endash} Print help
