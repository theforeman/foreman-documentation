= location remove-compute-resource

[role="_abstract"]
Disassociate a compute resource

*Usage*

----
# hammer location remove-compute-resource [OPTIONS]
----

*Options*

* `--compute-resource VALUE` {endash} Compute resource name
* `--compute-resource-id NUMBER`
* `--id VALUE`
* `--name VALUE` {endash} Set the current location context for the request
* `--title VALUE` {endash} Set the current location context for the request
* `-h`, `--help` {endash} Print help
