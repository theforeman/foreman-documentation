[id="hammer-host-traces-resolve"]
= host traces resolve

[role="_abstract"]
Resolve traces

*Usage*

----
# hammer host traces resolve [OPTIONS]
----

*Options*

* `--async` {endash} Do not wait for the task
* `--host VALUE` {endash} Host name
* `--host-id NUMBER` {endash} ID of the host
* `--trace-ids LIST` {endash} Array of Trace IDs
* `-h`, `--help` {endash} Print help
