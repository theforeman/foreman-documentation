[id="hammer-product-info"]
= product info

[role="_abstract"]
Show a product

*Usage*

----
# hammer product <info|show> [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--id NUMBER` {endash} Product numeric identifier
* `--name VALUE` {endash} Product name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization ID
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS               | ALL | DEFAULT | THIN

| Id                   | x   | x       | x
| Name                 | x   | x       | x
| Label                | x   | x       |
| Description          | x   | x       |
| Sync state (all)     | x   | x       |
| Sync state (last)    | x   | x       |
| Sync plan id         | x   | x       |
| Gpg/gpg key id       | x   | x       |
| Gpg/gpg key          | x   | x       |
| Organization         | x   | x       |
| Readonly             | x   | x       |
| Deletable            | x   | x       |
| Content/repo name    | x   | x       |
| Content/url          | x   | x       |
| Content/content type | x   | x       |
|===
