[id="hammer-repository-set-available-repositories"]
= repository-set available-repositories

[role="_abstract"]
Get list of available repositories for the repository set

*Usage*

----
# hammer repository-set available-repositories [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--id NUMBER` {endash} ID of the repository set
* `--name VALUE` {endash} Repository set name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization identifier
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--product VALUE` {endash} Product name to search by
* `--product-id NUMBER` {endash} ID of a product to list repository sets from
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS        | ALL | DEFAULT | THIN

| Name          | x   | x       | x
| Arch          | x   | x       |
| Release       | x   | x       |
| Registry name | x   | x       |
| Enabled       | x   | x       |
|===
