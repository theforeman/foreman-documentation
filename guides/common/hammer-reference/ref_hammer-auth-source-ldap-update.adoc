= auth-source ldap update

[role="_abstract"]
Update an LDAP authentication source

*Usage*

----
# hammer auth-source ldap update [OPTIONS]
----

*Options*

* `--account VALUE`
* `--account-password VALUE` {endash} Required if onthefly_register is true
* `--attr-firstname VALUE` {endash} Required if onthefly_register is true
* `--attr-lastname VALUE` {endash} Required if onthefly_register is true
* `--attr-login VALUE` {endash} Required if onthefly_register is true
* `--attr-mail VALUE` {endash} Required if onthefly_register is true
* `--attr-photo VALUE`
* `--base-dn VALUE`
* `--groups-base VALUE` {endash} Groups base DN
* `--host VALUE` {endash} The hostname of the LDAP server
* `--id VALUE`
* `--ldap-filter VALUE` {endash} LDAP filter
* `--ldap-group-membership ENUM`  Type of group membership to use, applicable only when server_type is posix,
free_ipa or netiq. Option rfc4519 is only applicable  when server_type is posix.
Possible value(s): `posix`, `rfc4519`, `nis_netgroups`
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-ids LIST` {endash} REPLACE locations with given ids
* `--location-title VALUE` {endash} Set the current location context for the request
* `--location-titles LIST`
* `--locations LIST`
* `--name VALUE`
* `--new-name VALUE`
* `--onthefly-register BOOLEAN`
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-ids LIST` {endash} REPLACE organizations with given ids.
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--organization-titles LIST`
* `--organizations LIST`
* `--port NUMBER` {endash} Defaults to 389
* `--server-type ENUM` {endash} Type of the LDAP server
Possible value(s): `free_ipa`, `active_directory`, `posix`, `netiq`
* `--tls BOOLEAN`
* `--use-netgroups BOOLEAN` {endash} Use NIS netgroups instead of posix groups, applicable only when server_type is
posix or free_ipa. Deprecated in favor of ldap_group_membership = nis_netgroups
* `--usergroup-sync BOOLEAN` {endash} Sync external user groups on login
* `-h`, `--help` {endash} Print help
