[id="hammer-user-group-create"]
= user-group create

[role="_abstract"]
Create a user group

*Usage*

----
# hammer user-group create [OPTIONS]
----

*Options*

* `--admin BOOLEAN` {endash} Is an admin user group, can be modified by admins only
* `--name VALUE`
* `--role-ids LIST`
* `--roles LIST`
* `--user-group-ids LIST`
* `--user-groups LIST`
* `--user-ids LIST`
* `--users LIST`
* `-h`, `--help` {endash} Print help
