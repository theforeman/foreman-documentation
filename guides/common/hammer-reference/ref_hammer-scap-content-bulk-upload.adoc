= scap-content bulk-upload

[role="_abstract"]
Upload scap contents in bulk

*Usage*

----
# hammer scap-content bulk-upload [OPTIONS]
----

*Options*

* `--directory VALUE` {endash} Directory to upload when using “directory” upload type
* `--files LIST` {endash} File paths to upload when using “files” upload type
* `--location VALUE` {endash} Name to search by
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--organization VALUE` {endash} Name to search by
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--type ENUM` {endash} Type of the upload
Possible value(s): `files`, `directory`, `default`
* `-h`, `--help` {endash} Print help
