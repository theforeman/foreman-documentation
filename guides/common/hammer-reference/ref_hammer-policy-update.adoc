[id="hammer-policy-update"]
= policy update

[role="_abstract"]
Update a Policy

*Usage*

----
# hammer policy update [OPTIONS]
----

*Options*

* `--cron-line VALUE` {endash} Policy schedule cron line (only if period == “custom”)
* `--day-of-month NUMBER` {endash} Policy schedule day of month (only if period == “monthly”)
* `--deploy-by ENUM` {endash} How the policy should be deployed
Possible value(s): `puppet`, `ansible`, `manual`
* `--description VALUE` {endash} Policy description
* `--host-ids LIST` {endash} Apply policy to hosts
* `--hostgroup-ids LIST` {endash} Apply policy to host groups
* `--hostgroups LIST`
* `--hosts LIST`
* `--id VALUE`
* `--location VALUE` {endash} Name to search by
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-ids LIST` {endash} REPLACE locations with given ids
* `--locations LIST`
* `--name VALUE` {endash} Policy name
* `--new-name VALUE` {endash} Policy name
* `--organization VALUE` {endash} Name to search by
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-ids LIST` {endash} REPLACE organizations with given ids.
* `--organizations LIST`
* `--period VALUE` {endash} Policy schedule period (weekly, monthly, custom)
* `--scap-content VALUE` {endash} SCAP content title
* `--scap-content-id NUMBER` {endash} Policy SCAP content ID
* `--scap-content-profile VALUE` {endash} Name to search by
* `--scap-content-profile-id NUMBER` {endash} Policy SCAP content profile ID
* `--tailoring-file VALUE` {endash} Tailoring file name
* `--tailoring-file-id NUMBER` {endash} Tailoring file ID
* `--tailoring-file-profile-id NUMBER` Tailoring file profile ID
* `--weekday VALUE` {endash} Policy schedule weekday (only if period == “weekly”)
* `-h`, `--help` {endash} Print help
