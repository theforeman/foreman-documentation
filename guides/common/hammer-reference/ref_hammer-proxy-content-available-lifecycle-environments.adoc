[id="hammer-proxy-content-available-lifecycle-environments"]
= proxy content available-lifecycle-environments

[role="_abstract"]
List the lifecycle environments not attached to the Capsule

*Usage*

----
# hammer proxy content available-lifecycle-environments [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--id NUMBER` {endash} Id of the Capsule
* `--name VALUE` {endash} Name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Id of the organization to limit environments on
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS       | ALL | DEFAULT | THIN

| Id           | x   | x       | x
| Name         | x   | x       | x
| Organization | x   | x       |
|===
