[id="hammer-domain-delete-parameter"]
= domain delete-parameter

[role="_abstract"]
Delete parameter for a domain

*Usage*

----
# hammer domain delete-parameter [OPTIONS]
----

*Options*

* `--domain VALUE` {endash} Domain name
* `--domain-id NUMBER` {endash} Numerical ID or domain name
* `--name VALUE` {endash} Parameter name
* `-h`, `--help` {endash} Print help
