[id="hammer-host-package-list"]
= host package list

[role="_abstract"]
List packages installed on the host

*Usage*

----
# hammer host package <list|index> [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--full-result BOOLEAN` {endash} Whether or not to show all results
* `--host VALUE` {endash} Host name
* `--host-id NUMBER` {endash} ID of the host
* `--include-latest-upgradable BOOLEAN` Also include the latest upgradable package version for each host package
* `--order VALUE` {endash} Sort field and order, eg. `id DESC`
* `--page NUMBER` {endash} Page number, starting at 1
* `--per-page NUMBER` {endash} Number of results per page to return
* `--search VALUE` {endash} Search string
* `--status VALUE` {endash} Return only packages of a particular status (upgradable or up-to-date)
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS | ALL | DEFAULT

| Nvra   | x   | x
|===

*Search and order fields*

* `arch` {endash} string
* `epoch` {endash} string
* `id` {endash} integer
* `name` {endash} string
* `nvra` {endash} string
* `nvrea` {endash} string
* `release` {endash} string
* `vendor` {endash} string
* `version` {endash} string
