= compute-profile values add-volume

[role="_abstract"]
Add volume for Compute Profile

*Usage*

----
# hammer compute-profile values add-volume [OPTIONS]
----

*Options*

* `--compute-profile VALUE` {endash} Compute profile name
* `--compute-profile-id VALUE`
* `--compute-resource VALUE` {endash} Compute resource name
* `--compute-resource-id VALUE`
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-title VALUE` {endash} Set the current location context for the request
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--volume KEY_VALUE_LIST` {endash} Volume parameters, should be comma separated list of values
* `-h`, `--help` {endash} Print help

*Provider-specific options*

Bold attributes are required.

EC2:

* `--volume`:

Libvirt:

* `--volume`:
** `*pool_name*` {endash} One of available storage pools
** `*capacity*` {endash} String value, e.g. 10G
** `allocation` {endash} 
Initial allocation, e.g. 0G
** `format_type` {endash} Possible values: raw, qcow2

OpenStack:

* `--volume`:

VMware:

* `--volume`:
** `name` {endash} 
** `storage_pod` {endash} Storage Pod ID from VMware
** `datastore` {endash} Datastore ID from VMware
** `mode` {endash} persistent/independent_persistent/independent_nonpersistent
** `size_gb` {endash} Integer number, volume size in GB
** `thin` {endash} true/false
** `eager_zero` {endash} true/false
** `controller_key` {endash} Associated controller key

AzureRM:

* `--volume`:
** `disk_size_gb` {endash} Volume Size in GB (integer value)
** `data_disk_caching` {endash} Data Disk Caching (None, ReadOnly, ReadWrite)

GCE:

* `--volume`:
** `size_gb` {endash} Volume size in GB, integer value

KubeVirt:

* `--volume`:
** `capacity` {endash} Volume size in GB, integer value
** `storage_class` {endash} Name of the storage class
** `bootable` {endash} Boolean, only one volume can be bootable (overrides network interface boot)
