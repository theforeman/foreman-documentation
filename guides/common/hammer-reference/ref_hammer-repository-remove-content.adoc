[id="hammer-repository-remove-content"]
= repository remove-content

[role="_abstract"]
Remove content from a repository

*Usage*

----
# hammer repository remove-content [OPTIONS]
----

*Options*

* `--content-type VALUE` {endash} The type of content unit to remove (srpm, docker_manifest, etc.). View removable
types with "hammer repository types"
* `--id NUMBER` {endash} Repository ID
* `--ids LIST` {endash} Array of content ids to remove
* `--name VALUE` {endash} Repository name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id VALUE` {endash} Organization ID to search by
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title to search by
* `--product VALUE` {endash} Product name to search by
* `--product-id NUMBER` {endash} Product numeric identifier
* `--sync-capsule BOOLEAN` {endash} Whether or not to sync an external capsule after upload. Default: true
* `-h`, `--help` {endash} Print help
