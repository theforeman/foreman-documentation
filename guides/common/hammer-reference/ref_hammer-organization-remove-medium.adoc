[id="hammer-organization-remove-medium"]
= organization remove-medium

[role="_abstract"]
Disassociate a medium

*Usage*

----
# hammer organization remove-medium [OPTIONS]
----

*Options*

* `--id VALUE` {endash} Organization ID
* `--medium VALUE` {endash} Medium name
* `--medium-id NUMBER`
* `--name VALUE` {endash} Set the current organization context for the request
* `--title VALUE` {endash} Set the current organization context for the request
* `-h`, `--help` {endash} Print help
