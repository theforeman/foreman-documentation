[id="hammer-sync-plan-info"]
= sync-plan info

[role="_abstract"]
Show a sync plan

*Usage*

----
# hammer sync-plan <info|show> [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--id NUMBER` {endash} Sync plan numeric identifier
* `--name VALUE` {endash} Sync plan name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization ID
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS             | ALL | DEFAULT | THIN

| Id                 | x   | x       | x
| Name               | x   | x       | x
| Start date         | x   | x       |
| Interval           | x   | x       |
| Enabled            | x   | x       |
| Cron expression    | x   | x       |
| Recurring logic id | x   | x       |
| Description        | x   | x       |
| Created at         | x   | x       |
| Updated at         | x   | x       |
| Next sync          | x   | x       |
| Products/id        | x   | x       |
| Products/name      | x   | x       |
|===
