= host errata recalculate

[role="_abstract"]
Force regenerate applicability.

*Usage*

----
# hammer host errata recalculate [OPTIONS]
----

*Options*

* `--host VALUE` {endash} Host name
* `--host-id NUMBER` {endash} Host ID
* `-h`, `--help` {endash} Print help
