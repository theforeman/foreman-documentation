[id="hammer-user-remove-role"]
= user remove-role

[role="_abstract"]
Remove a user role

*Usage*

----
# hammer user remove-role [OPTIONS]
----

*Options*

* `--id VALUE`
* `--login VALUE` {endash} User`s login to search by
* `--role VALUE` {endash} User role name
* `--role-id NUMBER`
* `-h`, `--help` {endash} Print help
