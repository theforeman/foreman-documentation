= user access-token create

[role="_abstract"]
Create a Personal Access Token for a user

*Usage*

----
# hammer user access-token create [OPTIONS]
----

*Options*

* `--expires-at VALUE` {endash} Expiry Date
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-title VALUE` {endash} Set the current location context for the request
* `--name VALUE`
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--user VALUE` {endash} User`s login to search by
* `--user-id VALUE` {endash} ID of the user
* `-h`, `--help` {endash} Print help
