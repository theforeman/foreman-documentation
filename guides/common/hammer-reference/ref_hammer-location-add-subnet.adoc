[id="hammer-location-add-subnet"]
= location add-subnet

[role="_abstract"]
Associate a subnet

*Usage*

----
# hammer location add-subnet [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Set the current location context for the request
* `--subnet VALUE` {endash} Subnet name
* `--subnet-id NUMBER`
* `--title VALUE` {endash} Set the current location context for the request
* `-h`, `--help` {endash} Print help
