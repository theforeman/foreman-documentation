[id="hammer-location-add-organization"]
= location add-organization

[role="_abstract"]
Associate an organization

*Usage*

----
# hammer location add-organization [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Set the current location context for the request
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Organization ID
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--title VALUE` {endash} Set the current location context for the request
* `-h`, `--help` {endash} Print help
