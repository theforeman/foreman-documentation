= content-import repository

[role="_abstract"]
Imports a repository

*Usage*

----
# hammer content-import repository [OPTIONS]
----

*Options*

* `--async` {endash} Do not wait for the task
* `--metadata-file VALUE` {endash} Location of the metadata.json file. This is not required if the metadata.json
file is already in the archive directory.
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization identifier
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--path VALUE` {endash} Directory containing the exported Content View Version
* `-h`, `--help` {endash} Print help
