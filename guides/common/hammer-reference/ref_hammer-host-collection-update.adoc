[id="hammer-host-collection-update"]
= host-collection update

[role="_abstract"]
Update a host collection

*Usage*

----
# hammer host-collection update [OPTIONS]
----

*Options*

* `--description VALUE`
* `--host-ids LIST` {endash} List of host ids to replace the hosts in host collection
* `--hosts LIST`
* `--id NUMBER` {endash} Id of the host collection
* `--max-hosts NUMBER` {endash} Maximum number of hosts in the host collection
* `--name VALUE` {endash} Host Collection name
* `--new-name VALUE` {endash} Host Collection name
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization ID
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--unlimited-hosts` {endash} Set hosts max to unlimited
* `-h`, `--help` {endash} Print help
