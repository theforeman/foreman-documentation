[id="hammer-sc-param-add-matcher"]
= sc-param add-matcher

[role="_abstract"]
Create an override value for a specific smart class parameter

*Usage*

----
# hammer sc-param add-matcher [OPTIONS]
----

*Options*

* `--match VALUE` {endash} Override match
* `--omit BOOLEAN` {endash} Satellite will not send this parameter in classification output
* `--puppet-class VALUE` {endash} Puppet class name
* `--puppet-class-id NUMBER` {endash} ID of Puppet class
* `--smart-class-parameter VALUE` {endash} Smart class parameter name
* `--smart-class-parameter-id VALUE`
* `--value VALUE` {endash} Override value, required if omit is false
* `-h`, `--help` {endash} Print help
