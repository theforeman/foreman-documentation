:_mod-docs-content-type: REFERENCE

[id="hammer-content-credentials-delete"]
= content-credentials delete

[role="_abstract"]
Destroy a Content Credential

*Usage*

----
# hammer content-credentials <delete|destroy> [OPTIONS]
----

*Options*

* `--id NUMBER` {endash} Content Credential ID
* `--name VALUE` {endash} Name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER`
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `-h`, `--help` {endash} Print help
