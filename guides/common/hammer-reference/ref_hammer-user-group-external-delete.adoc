[id="hammer-user-group-external-delete"]
= user-group external delete

[role="_abstract"]
Delete an external user group

*Usage*

----
# hammer user-group external <delete|destroy> [OPTIONS]
----

*Options*

* `--id VALUE` {endash} ID or name external user group
* `--name VALUE` {endash} Name to search by
* `--user-group VALUE` {endash} Name to search by
* `--user-group-id VALUE` {endash} ID or name of user group
* `-h`, `--help` {endash} Print help
