= host subscription register

[role="_abstract"]
Register a host with subscription and information

*Usage*

----
# hammer host subscription register [OPTIONS]
----

*Options*

* `--content-view VALUE` {endash} Content view name to search by
* `--content-view-id NUMBER` {endash} Content View ID
* `--environment VALUE` {endash} Lifecycle environment name to search by (--environment is deprecated: Use
* `--lifecycle-environment` instead)
* `--environment-id NUMBER` {endash} (--environment-id is deprecated: Use `--lifecycle-environment-id` instead)
* `--hypervisor-guest-uuids LIST` {endash} UUIDs of the virtual guests from the host`s hypervisor
* `--lifecycle-environment VALUE` {endash} Lifecycle environment name to search by
* `--lifecycle-environment-id NUMBER` Lifecycle Environment ID
* `--name VALUE` {endash} Name of the host
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id VALUE` {endash} Organization ID to search by
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title to search by
* `--release-version VALUE` {endash} Release version of the content host
* `--service-level VALUE` {endash} A service level for auto-healing process, e.g. SELF-SUPPORT
* `--uuid VALUE` {endash} UUID to use for registered host, random uuid is generated if not provided
* `-h`, `--help` {endash} Print help
