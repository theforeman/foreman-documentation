[id="hammer-location-delete-parameter"]
= location delete-parameter

[role="_abstract"]
Delete parameter for a location

*Usage*

----
# hammer location delete-parameter [OPTIONS]
----

*Options*

* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER`
* `--location-title VALUE` {endash} Set the current location context for the request
* `--name VALUE` {endash} Parameter name
* `-h`, `--help` {endash} Print help
