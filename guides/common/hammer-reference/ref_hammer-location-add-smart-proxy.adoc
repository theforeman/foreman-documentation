= location add-smart-proxy

[role="_abstract"]
Associate a smart proxy

*Usage*

----
# hammer location add-smart-proxy [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Set the current location context for the request
* `--smart-proxy VALUE` {endash} Name to search by
* `--smart-proxy-id NUMBER`
* `--title VALUE` {endash} Set the current location context for the request
* `-h`, `--help` {endash} Print help
