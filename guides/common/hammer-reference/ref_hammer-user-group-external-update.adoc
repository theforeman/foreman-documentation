= user-group external update

[role="_abstract"]
Update external user group

*Usage*

----
# hammer user-group external update [OPTIONS]
----

*Options*

* `--auth-source VALUE` {endash} Name to search by
* `--auth-source-id NUMBER` {endash} ID of linked authentication source
* `--id VALUE` {endash} ID or name of external user group
* `--name VALUE` {endash} External user group name
* `--new-name VALUE` {endash} External user group name
* `--user-group VALUE` {endash} Name to search by
* `--user-group-id VALUE` {endash} ID or name of user group
* `-h`, `--help` {endash} Print help
