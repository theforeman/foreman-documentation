[id="hammer-user-add-role"]
= user add-role

[role="_abstract"]
Assign a user role

*Usage*

----
# hammer user add-role [OPTIONS]
----

*Options*

* `--id VALUE`
* `--login VALUE` {endash} User`s login to search by
* `--role VALUE` {endash} User role name
* `--role-id NUMBER`
* `-h`, `--help` {endash} Print help
