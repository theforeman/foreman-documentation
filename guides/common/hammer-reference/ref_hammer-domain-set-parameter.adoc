[id="hammer-domain-set-parameter"]
= domain set-parameter

[role="_abstract"]
Create or update parameter for a domain

*Usage*

----
# hammer domain set-parameter [OPTIONS]
----

*Options*

* `--domain VALUE` {endash} Domain name
* `--domain-id NUMBER` {endash} Numerical ID or domain name
* `--hidden-value BOOLEAN` {endash} Should the value be hidden
* `--name VALUE` {endash} Parameter name
* `--parameter-type ENUM` {endash} Type of the parameter
Possible value(s): `string`, `boolean`, `integer`, `real`, `array`, `hash`,
`yaml`, `json`
Default: "string"
* `--value VALUE` {endash} Parameter value
* `-h`, `--help` {endash} Print help
