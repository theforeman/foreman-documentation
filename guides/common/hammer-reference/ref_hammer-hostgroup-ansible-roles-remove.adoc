[id="hammer-hostgroup-ansible-roles-remove"]
= hostgroup ansible-roles remove

[role="_abstract"]
Disassociate an Ansible role

*Usage*

----
# hammer hostgroup ansible-roles remove [OPTIONS]
----

*Options*

* `--ansible-role VALUE` {endash} Name to search by
* `--ansible-role-id NUMBER`
* `--id VALUE`
* `--name VALUE` {endash} Hostgroup name
* `--title VALUE` {endash} Hostgroup title
* `-h`, `--help` {endash} Print help
