= activation-key product-content

[role="_abstract"]
List associated products

*Usage*

----
# hammer activation-key product-content [OPTIONS]
----

*Options*

* `--content-access-mode-all BOOLEAN` Get all content available, not just that provided by subscriptions
* `--content-access-mode-env BOOLEAN` Limit content to just that available in the activation key`s content view
version
* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--full-result BOOLEAN` {endash} Whether or not to show all results
* `--id VALUE` {endash} ID of the activation key
* `--name VALUE` {endash} Activation key name to search by
* `--order VALUE` {endash} Sort field and order, eg. `id DESC`
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER`
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--page NUMBER` {endash} Page number, starting at 1
* `--per-page NUMBER` {endash} Number of results per page to return
* `--search VALUE` {endash} Search string
* `-h`, `--help` {endash} Print help
