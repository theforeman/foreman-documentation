[id="hammer-content-view-update"]
= content-view update

[role="_abstract"]
Update a content view

*Usage*

----
# hammer content-view update [OPTIONS]
----

*Options*

* `--auto-publish BOOLEAN` {endash} Enable/Disable auto publish of composite view
* `--component-ids LIST` {endash} List of component content view version ids for composite views
* `--description VALUE` {endash} Description for the content view
* `--id NUMBER` {endash} Content view identifier
* `--import-only BOOLEAN` {endash} Designate this Content View for importing from upstream servers only. Defaults
to false
* `--name VALUE` {endash} New name for the content view
* `--new-name VALUE` {endash} New name for the content view
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id VALUE` {endash} Organization ID to search by
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title to search by
* `--repository-ids LIST` {endash} List of repository ids
* `--solve-dependencies BOOLEAN`  Solve RPM dependencies by default on Content View publish, defaults to false
* `-h`, `--help` {endash} Print help
