= flatpak-remote remote-repository info

[role="_abstract"]
Show a flatpak remote repository

*Usage*

----
# hammer flatpak-remote remote-repository <info|show> [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--flatpak-remote VALUE` {endash} Flatpak remote name to search by
* `--flatpak-remote-id NUMBER`
* `--id NUMBER` {endash} Flatpak remote repository numeric identifier
* `--manifests BOOLEAN` {endash} Include manifests
* `--name VALUE` {endash} Name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER`
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS                    | ALL | DEFAULT | THIN

| Id                        | x   | x       | x
| Name                      | x   | x       | x
| Label                     | x   | x       |
| Flatpak remote id         | x   | x       |
| Flatpak remote name       | x   | x       |
| Flatpak remote url        | x   | x       |
| Manifests/manifest name   | x   | x       |
| Manifests/manifest digest | x   | x       |
| Manifests/manifest tags   | x   | x       |
|===
