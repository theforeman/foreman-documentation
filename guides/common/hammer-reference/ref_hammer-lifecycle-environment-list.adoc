[id="hammer-lifecycle-environment-list"]
= lifecycle-environment list

[role="_abstract"]
List environments in an organization

*Usage*

----
# hammer lifecycle-environment <list|index> [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--full-result BOOLEAN` {endash} Whether or not to show all results
* `--label VALUE` {endash} Filter only environments containing this label
* `--library BOOLEAN` {endash} Set true if you want to see only library environments
* `--name VALUE` {endash} Filter only environments containing this name
* `--order VALUE` {endash} Sort field and order, eg. `id DESC`
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization identifier
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--page NUMBER` {endash} Page number, starting at 1
* `--per-page NUMBER` {endash} Number of results per page to return
* `--search VALUE` {endash} Search string
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS | ALL | DEFAULT | THIN

| Id     | x   | x       | x
| Name   | x   | x       | x
| Prior  | x   | x       |
|===

*Search and order fields*

* `id` {endash} integer
* `label` {endash} string
* `name` {endash} string
* `organization_id` {endash} integer
