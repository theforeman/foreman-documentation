[id="hammer-config-group-update"]
= config-group update

[role="_abstract"]
Update a config group

*Usage*

----
# hammer config-group update [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE`
* `--new-name VALUE`
* `--puppet-class-ids LIST`
* `--puppet-classes LIST`
* `-h`, `--help` {endash} Print help
