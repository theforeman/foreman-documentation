[id="hammer-puppet-class-info"]
= puppet-class info

[role="_abstract"]
Show a Puppet class

*Usage*

----
# hammer puppet-class <info|show> [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--host VALUE` {endash} Host name
* `--host-id VALUE` {endash} ID of host
* `--hostgroup VALUE` {endash} Hostgroup name
* `--hostgroup-id VALUE` {endash} ID of host group
* `--hostgroup-title VALUE` {endash} Hostgroup title
* `--id VALUE` {endash} ID of Puppet class
* `--name VALUE` {endash} Puppet class name
* `--puppet-environment VALUE` {endash} Puppet environment name
* `--puppet-environment-id VALUE` ID of environment
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS                  | ALL | DEFAULT | THIN

| Id                      | x   | x       | x
| Name                    | x   | x       | x
| Smart class parameters/ | x   | x       |
| Hostgroups/             | x   | x       |
| Environments/           | x   | x       |
| Parameters/             | x   | x       |
|===
