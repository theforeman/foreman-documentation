[id="hammer-host-delete-parameter"]
= host delete-parameter

[role="_abstract"]
Delete parameter for a host

*Usage*

----
# hammer host delete-parameter [OPTIONS]
----

*Options*

* `--host VALUE` {endash} Host name
* `--host-id NUMBER`
* `--name VALUE` {endash} Parameter name
* `-h`, `--help` {endash} Print help
