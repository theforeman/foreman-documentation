[id="hammer-template-input-update"]
= template-input update

[role="_abstract"]
Update a template input

*Usage*

----
# hammer template-input update [OPTIONS]
----

*Options*

* `--advanced BOOLEAN` {endash} Input is advanced
* `--default VALUE` {endash} Default value for user input
* `--description VALUE` {endash} Input description
* `--fact-name VALUE` {endash} Fact name, used when input type is Fact value
* `--hidden-value BOOLEAN` {endash} The value contains sensitive information and shouldn not be normally visible,
useful e.g. for passwords
* `--id VALUE`
* `--input-type ENUM` {endash} Input type
Possible value(s): `puppet_parameter`, `user`, `fact`, `variable`
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-title VALUE` {endash} Set the current location context for the request
* `--name VALUE` {endash} Input name
* `--new-name VALUE` {endash} Input name
* `--options LIST` {endash} Selectable values for user inputs
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--puppet-class-name VALUE` {endash} Puppet class name, used when input type is Puppet parameter
* `--puppet-parameter-name VALUE` Puppet parameter name, used when input type is Puppet parameter
* `--required BOOLEAN` {endash} Input is required
* `--resource-type ENUM` {endash} For values of type search, this is the resource the value searches in
Possible value(s): `AnsibleRole`, `AnsibleVariable`, `Architecture`, `Audit`,
`AuthSource`, `Bookmark`, `ComputeProfile`, `ComputeResource`, `ConfigReport`,
`DiscoveryRule`, `Domain`, `ExternalUsergroup`, `FactValue`, `Filter`,
`ForemanOpenscap::ArfReport`, `ForemanOpenscap::Policy`,
`ForemanOpenscap::ScapContent`, `ForemanOpenscap::TailoringFile`,
`ForemanPuppet::ConfigGroup`, `ForemanPuppet::Environment`,
`ForemanPuppet::HostClass`, `ForemanPuppet::Puppetclass`,
`ForemanPuppet::PuppetclassLookupKey`, `ForemanTasks::RecurringLogic`,
`ForemanTasks::Task`, `ForemanVirtWhoConfigure::Config`, `Host`, `Hostgroup`,
`HttpProxy`, `Image`, `InsightsHit`, `JobInvocation`, `JobTemplate`,
`Katello::ActivationKey`, `Katello::AlternateContentSource`,
`Katello::ContentCredential`, `Katello::ContentView`, `Katello::FlatpakRemote`,
`Katello::HostCollection`, `Katello::KTEnvironment`, `Katello::Product`,
`Katello::Subscription`, `Katello::SyncPlan`, `KeyPair`, `Location`,
`LookupValue`, `MailNotification`, `Medium`, `Model`, `Operatingsystem`,
`Organization`, `Parameter`, `PersonalAccessToken`, `ProvisioningTemplate`,
`Ptable`, `Realm`, `RemoteExecutionFeature`, `ReportTemplate`, `Role`,
`Setting`, `SmartProxy`, `SshKey`, `Subnet`, `Template`, `TemplateInvocation`,
`User`, `Usergroup`, `Webhook`, `WebhookTemplate`
* `--template-id VALUE`
* `--value-type ENUM` {endash} Value type, defaults to plain
Possible value(s): `plain`, `search`, `date`, `resource`
* `--variable-name VALUE` {endash} Variable name, used when input type is Variable
* `-h`, `--help` {endash} Print help
