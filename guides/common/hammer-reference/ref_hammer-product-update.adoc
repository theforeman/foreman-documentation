[id="hammer-product-update"]
= product update

[role="_abstract"]
Updates a product

*Usage*

----
# hammer product update [OPTIONS]
----

*Options*

* `--description VALUE` {endash} Product description
* `--gpg-key-id NUMBER` {endash} Identifier of the GPG key
* `--id NUMBER` {endash} Product numeric identifier
* `--name VALUE` {endash} Product name
* `--new-name VALUE` {endash} Product name
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER`
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--ssl-ca-cert-id NUMBER` {endash} Idenifier of the SSL CA Cert
* `--ssl-client-cert-id NUMBER` {endash} Identifier of the SSL Client Cert
* `--ssl-client-key-id NUMBER` {endash} Identifier of the SSL Client Key
* `--sync-plan VALUE` {endash} Sync plan name to search by
* `--sync-plan-id NUMBER` {endash} Plan numeric identifier
* `-h`, `--help` {endash} Print help
