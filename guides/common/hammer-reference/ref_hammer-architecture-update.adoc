= architecture update

[role="_abstract"]
Update an architecture

*Usage*

----
# hammer architecture update [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE`
* `--new-name VALUE`
* `--operatingsystem-ids LIST` {endash} Operating system IDs
* `--operatingsystems LIST`
* `-h`, `--help` {endash} Print help
