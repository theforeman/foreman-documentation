= job-invocation info

[role="_abstract"]
Show job invocation

*Usage*

----
# hammer job-invocation <info|show> [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--id VALUE`
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--show-host-status` {endash} Show job status for the hosts
* `--show-inputs` {endash} Show the complete input of the job
* `-h`, `--help` {endash} Print help

*Predefined field sets*

| Id                  | x   | x

| Pending             | x   | x

| Start               | x   | x
| Randomized ordering | x   | x
| Inputs              | x   | x
| Job category        | x   | x
| Mode                | x   | x
| Cron line           | x   | x
| Recurring logic id  | x   | x
| Time to pickup      | x   | x
| Hosts               | x   | x
|===
