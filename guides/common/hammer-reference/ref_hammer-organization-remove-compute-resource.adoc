= organization remove-compute-resource

[role="_abstract"]
Disassociate a compute resource

*Usage*

----
# hammer organization remove-compute-resource [OPTIONS]
----

*Options*

* `--compute-resource VALUE` {endash} Compute resource name
* `--compute-resource-id NUMBER`
* `--id VALUE` {endash} Organization ID
* `--name VALUE` {endash} Set the current organization context for the request
* `--title VALUE` {endash} Set the current organization context for the request
* `-h`, `--help` {endash} Print help
