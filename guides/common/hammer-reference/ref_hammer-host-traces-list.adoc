[id="hammer-host-traces-list"]
= host traces list

[role="_abstract"]
List services that need restarting on the host

*Usage*

----
# hammer host traces list [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--host VALUE` {endash} Host name
* `--host-id NUMBER` {endash} ID of the host
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS      | ALL | DEFAULT

| Trace id    | x   | x
| Application | x   | x
| Helper      | x   | x
| Type        | x   | x
|===
