[id="hammer-compute-resource-create"]
= compute-resource create

[role="_abstract"]
Create a compute resource

*Usage*

----
# hammer compute-resource create [OPTIONS]
----

*Options*

* `--app-ident VALUE` {endash} Client ID for AzureRm
* `--caching-enabled BOOLEAN` {endash} Enable caching, for VMware only
* `--cloud VALUE` {endash} Cloud
* `--datacenter VALUE` {endash} For VMware Datacenter
* `--description VALUE`
* `--display-type ENUM` {endash} For Libvirt only
Possible value(s): `VNC`, `SPICE`
* `--domain VALUE` {endash} For Red Hat OpenStack Platform (v3) only
* `--email VALUE` {endash} Deprecated, email is automatically loaded from the JSON file. For GCE only
* `--key-path VALUE` {endash} Certificate path, for GCE only
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-ids LIST` {endash} REPLACE locations with given ids
* `--location-title VALUE` {endash} Set the current location context for the request
* `--location-titles LIST`
* `--locations LIST`
* `--name VALUE`
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-ids LIST` {endash} REPLACE organizations with given ids.
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--organization-titles LIST`
* `--organizations LIST`
* `--password VALUE` {endash} Password for EC2, VMware, Red Hat OpenStack Platform. Secret key for EC2
* `--project VALUE` {endash} Deprecated, project is automatically loaded from the JSON file. For GCE only
* `--project-domain-id VALUE` {endash} For Red Hat OpenStack Platform (v3) only
* `--project-domain-name VALUE` {endash} For Red Hat OpenStack Platform (v3) only
* `--provider VALUE` {endash} Providers include Libvirt, EC2, Vmware, Openstack, AzureRm, GCE
* `--region VALUE` {endash} For AzureRm eg. `eastus` and for EC2 only. Use `us-gov-west-1` for EC2 GovCloud
region
* `--secret-key VALUE` {endash} Client Secret for AzureRm
* `--server VALUE` {endash} For VMware
* `--set-console-password BOOLEAN` For Libvirt and VMware only
* `--sub-id VALUE` {endash} Subscription ID for AzureRm
* `--tenant VALUE` {endash} For Red Hat OpenStack Platform and AzureRm only
* `--url VALUE` {endash} URL for Libvirt and Red Hat OpenStack Platform
* `--user VALUE` {endash} Username for EC2, VMware, Red Hat OpenStack Platform. Access Key for EC2.
* `--zone VALUE` {endash} Zone, for GCE only
* `-h`, `--help` {endash} Print help
