[id="hammer-content-view-component-remove"]
= content-view component remove

[role="_abstract"]
Remove components from the content view

*Usage*

----
# hammer content-view component remove [OPTIONS]
----

*Options*

* `--component-content-view-ids VALUE` Array of component content view identfiers to remove. Comma separated list of
values
* `--component-content-views VALUE` {endash} Array of component content view names to remove. Comma separated list of values
* `--component-ids LIST` {endash} Array of content view component IDs to remove. Identifier of the component
association
* `--composite-content-view VALUE` {endash} Name of the composite content view
* `--composite-content-view-id NUMBER` Composite content view identifier
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id VALUE` {endash} Organization ID to search by
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title to search by
* `-h`, `--help` {endash} Print help
