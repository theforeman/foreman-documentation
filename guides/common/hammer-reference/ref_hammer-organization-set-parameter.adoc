[id="hammer-organization-set-parameter"]
= organization set-parameter

[role="_abstract"]
Create or update parameter for an organization

*Usage*

----
# hammer organization set-parameter [OPTIONS]
----

*Options*

* `--hidden-value BOOLEAN` {endash} Should the value be hidden
* `--name VALUE` {endash} Parameter name
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Organization ID
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--parameter-type ENUM` {endash} Type of the parameter
Possible value(s): `string`, `boolean`, `integer`, `real`, `array`, `hash`,
`yaml`, `json`
Default: "string"
* `--value VALUE` {endash} Parameter value
* `-h`, `--help` {endash} Print help
