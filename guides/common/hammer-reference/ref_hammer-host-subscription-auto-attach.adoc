= host subscription auto-attach

[role="_abstract"]
Trigger an auto-attach of subscriptions

*Usage*

----
# hammer host subscription auto-attach [OPTIONS]
----

*Options*

* `--host VALUE` {endash} Host name
* `--host-id NUMBER` {endash} Id of the host
* `-h`, `--help` {endash} Print help
