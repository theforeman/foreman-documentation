= product verify-checksum

[role="_abstract"]
Verify checksum for one or more products

*Usage*

----
# hammer product verify-checksum [OPTIONS]
----

*Options*

* `--async` {endash} Do not wait for the task
* `--ids LIST` {endash} List of product ids
* `-h`, `--help` {endash} Print help
