[id="hammer-content-view-remove-repository"]
= content-view remove-repository

[role="_abstract"]
Disassociate a resource

*Usage*

----
# hammer content-view remove-repository [OPTIONS]
----

*Options*

* `--id VALUE` {endash} Content view numeric identifier
* `--name VALUE` {endash} Content view name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id VALUE` {endash} Organization ID to search by
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title to search by
* `--product VALUE` {endash} Product name to search by
* `--product-id NUMBER` {endash} Product numeric identifier
* `--repository VALUE` {endash} Repository name to search by
* `--repository-id NUMBER` {endash} Repository ID
* `-h`, `--help` {endash} Print help
