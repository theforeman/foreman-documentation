[id="hammer-activation-key-content-override"]
= activation-key content-override

[role="_abstract"]
Override product content defaults

*Usage*

----
# hammer activation-key content-override [OPTIONS]
----

*Options*

* `--content-label VALUE` {endash} Label of the content
* `--force` {endash} Force the override. Required for overrides other than `enabled`
* `--id NUMBER` {endash} ID of the activation key
* `--name VALUE` {endash} Activation key name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER`
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--override-name VALUE` {endash} Override parameter key or name.
To enable or disable a repo select `enabled`.
Default value: enabled
Default: "enabled"
* `--remove` {endash} Remove a content override
* `--value VALUE` {endash} Override value. Note for repo enablement you can use a boolean value
* `-h`, `--help` {endash} Print help
