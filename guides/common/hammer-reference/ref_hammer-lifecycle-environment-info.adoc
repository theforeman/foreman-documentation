= lifecycle-environment info

[role="_abstract"]
Show an environment

*Usage*

----
# hammer lifecycle-environment <info|show> [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--id NUMBER` {endash} ID of the environment
* `--name VALUE` {endash} Lifecycle environment name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} ID of the organization
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS                      | ALL | DEFAULT | THIN

| Id                          | x   | x       | x
| Name                        | x   | x       | x
| Label                       | x   | x       |
| Description                 | x   | x       |
| Organization                | x   | x       |
| Library                     | x   | x       |
| Prior lifecycle environment | x   | x       |
| Unauthenticated pull        | x   | x       |
| Registry name pattern       | x   | x       |
|===
