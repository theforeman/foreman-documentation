[id="hammer-job-invocation-cancel"]
= job-invocation cancel

[role="_abstract"]
Cancel the job

*Usage*

----
# hammer job-invocation cancel [OPTIONS]
----

*Options*

* `--force BOOLEAN`
* `--id VALUE`
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--search VALUE` {endash} Search query to cancel tasks only on matching hosts. If not provided, the whole
job invocation will be cancelled.
* `-h`, `--help` {endash} Print help
