= organization add-hostgroup

[role="_abstract"]
Associate a hostgroup

*Usage*

----
# hammer organization add-hostgroup [OPTIONS]
----

*Options*

* `--hostgroup VALUE` {endash} Hostgroup name
* `--hostgroup-id NUMBER`
* `--hostgroup-title VALUE` {endash} Hostgroup title
* `--id VALUE` {endash} Organization ID
* `--name VALUE` {endash} Set the current organization context for the request
* `--title VALUE` {endash} Set the current organization context for the request
* `-h`, `--help` {endash} Print help
