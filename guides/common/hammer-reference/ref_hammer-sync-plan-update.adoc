[id="hammer-sync-plan-update"]
= sync-plan update

[role="_abstract"]
Update a sync plan

*Usage*

----
# hammer sync-plan update [OPTIONS]
----

*Options*

* `--cron-expression VALUE` {endash} Add custom cron logic for sync plan
* `--description VALUE` {endash} Sync plan description
* `--enabled BOOLEAN` {endash} Enables or disables synchronization
* `--id NUMBER` {endash} Sync plan numeric identifier
* `--interval VALUE` {endash} How often synchronization should run
* `--name VALUE` {endash} Sync plan name
* `--new-name VALUE` {endash} Sync plan name
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization ID
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--sync-date DATETIME` {endash} Start date and time of the synchronization
* `-h`, `--help` {endash} Print help
