[id="hammer-activation-key-subscriptions"]
= activation-key subscriptions

[role="_abstract"]
List associated subscriptions

*Usage*

----
# hammer activation-key subscriptions [OPTIONS]
----

*Options*

* `--activation-key VALUE` {endash} Activation key name to search by
* `--activation-key-id VALUE` {endash} Activation key ID
* `--available-for VALUE` {endash} Object to show subscriptions available for, either `host` or `activation_key`
* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--full-result BOOLEAN` {endash} Whether or not to show all results
* `--host VALUE` {endash} Host name
* `--host-id VALUE` {endash} Id of a host
* `--id VALUE` {endash} ID of the activation key
* `--match-host BOOLEAN` {endash} Ignore subscriptions that are unavailable to the specified host
* `--match-installed BOOLEAN` {endash} Return subscriptions that match installed products of the specified host
* `--name VALUE` {endash} Activation key name to search by
* `--no-overlap BOOLEAN` {endash} Return subscriptions which do not overlap with a currently-attached subscription
* `--order VALUE` {endash} Sort field and order, eg. `id DESC`
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization ID
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--page NUMBER` {endash} Page number, starting at 1
* `--per-page NUMBER` {endash} Number of results per page to return
* `--search VALUE` {endash} Search string
* `-h`, `--help` {endash} Print help
