[id="hammer-sc-param-update"]
= sc-param update

[role="_abstract"]
Update a smart class parameter

*Usage*

----
# hammer sc-param update [OPTIONS]
----

*Options*

* `--avoid-duplicates BOOLEAN` {endash} Remove duplicate values (only array type)
* `--default-value VALUE` {endash} Value to use when there is no match
* `--description VALUE` {endash} Description of smart class
* `--hidden-value BOOLEAN` {endash} When enabled the parameter is hidden in the UI
* `--id VALUE`
* `--merge-default BOOLEAN` {endash} Include default value when merging all matching values
* `--merge-overrides BOOLEAN` {endash} Merge all matching values (only array/hash type)
* `--name VALUE` {endash} Smart class parameter name
* `--omit BOOLEAN` {endash} Satellite will not send this parameter in classification output.Puppet will use
the value defined in the Puppet manifest for this parameter
* `--override BOOLEAN` {endash} Override this parameter
* `--override-value-order LIST` {endash} The order in which values are resolved
* `--parameter-type ENUM` {endash} Type of the parameter
Possible value(s): `string`, `boolean`, `integer`, `real`, `array`, `hash`,
`yaml`, `json`
* `--path VALUE` {endash} The order in which values are resolved
* `--puppet-class VALUE` {endash} Puppet class name
* `--puppet-class-id NUMBER` {endash} ID of Puppet class
* `--required BOOLEAN` {endash} This parameter is required
* `--validator-rule VALUE` {endash} Used to enforce certain values for the parameter values
* `--validator-type ENUM` {endash} Type of the validator
Possible value(s): `regexp`, `list`, ``
* `-h`, `--help` {endash} Print help
