[id="hammer-job-invocation-create"]
= job-invocation create

[role="_abstract"]
Create a job invocation

*Usage*

----
# hammer job-invocation create [OPTIONS]
----

*Options*

* `--async` {endash} Do not wait for the task
* `--bookmark VALUE` {endash} Name to search by
* `--bookmark-id NUMBER`
* `--concurrency-level NUMBER` {endash} Run at most N tasks at a time
* `--cron-line VALUE` {endash} Create a recurring execution
Cron line format `a b c d e`, where:
a. is minute (range: 0-59)
b. is hour (range: 0-23)
c. is day of month (range: 1-31)
d. is month (range: 1-12)
e. is day of week (range: 0-6)
* `--description-format VALUE` {endash} Override the description format from the template for this invocation only
* `--dynamic` {endash} Dynamic search queries are evaluated at run time
* `--effective-user VALUE` {endash} What user should be used to run the script (using sudo-like mechanisms).
Defaults to a template parameter or global setting.
* `--effective-user-password VALUE` {endash} Set password for effective user (using sudo-like mechanisms)
* `--end-time DATETIME` {endash} Perform no more executions after this time, used with `--cron-line`
* `--execution-timeout-interval NUMBER` Override the timeout interval from the template for this invocation only
* `--feature VALUE` {endash} Remote execution feature label that should be triggered, job template assigned
to this feature will be used
* `--input-files KEY_VALUE_LIST` {endash} Read input values from files
Comma-separated list of key=file, where file is a path to a text file to be read
* `--inputs KEY_VALUE_LIST` {endash} Specify inputs from command line
* `--job-template VALUE` {endash} Name to search by
* `--job-template-id VALUE` {endash} The job template to use, parameter is required unless feature was specified
* `--key-passphrase VALUE` {endash} Set SSH key passphrase
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-title VALUE` {endash} Set the current location context for the request
* `--max-iteration NUMBER` {endash} Repeat a maximum of N times
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--password VALUE` {endash} Set SSH password
* `--purpose VALUE` {endash} Designation of a special purpose
* `--randomized-ordering BOOLEAN` {endash} Execute the jobs on hosts in randomized order
* `--search-query VALUE`
* `--ssh-user VALUE` {endash} Set SSH user
* `--start-at DATETIME` {endash} Schedule the execution for a later time
* `--start-before DATETIME` {endash} Execution should be cancelled if it cannot be started before `--start-at`
* `--tags VALUE` {endash} A comma separated list of tags to use for Ansible run
* `--tags-flag ENUM` {endash} IncludeExclude tags for Ansible run
Possible value(s): `include`, `exclude`
* `--time-to-pickup NUMBER` {endash} Override the global time to pickup interval for this invocation only
* `-h`, `--help` {endash} Print help
