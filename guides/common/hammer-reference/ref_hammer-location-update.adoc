= location update

[role="_abstract"]
Update a location

*Usage*

----
# hammer location update [OPTIONS]
----

*Options*

* `--compute-resource-ids LIST` {endash} Compute resource IDs
* `--compute-resources LIST`
* `--description VALUE`
* `--domain-ids LIST` {endash} Domain IDs
* `--domains LIST`
* `--environment-ids LIST` {endash} Environment IDs
* `--environments LIST`
* `--hostgroup-ids LIST` {endash} Host group IDs
* `--hostgroup-titles LIST`
* `--hostgroups LIST`
* `--id VALUE` {endash} Location numeric id to search by
* `--ignore-types LIST` {endash} List of resources types that will be automatically associated
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-title VALUE` {endash} Set the current location context for the request
* `--media LIST`
* `--medium-ids LIST` {endash} Medium IDs
* `--name VALUE`
* `--new-name VALUE`
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-ids LIST` {endash} Associated organization IDs
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--organization-titles LIST`
* `--organizations LIST`
* `--parent-id NUMBER` {endash} Parent ID
* `--partition-table-ids LIST` {endash} Partition template IDs
* `--partition-tables LIST`
* `--provisioning-template-ids LIST` Provisioning template IDs
* `--provisioning-templates LIST`
* `--puppet-environment-ids LIST` {endash} IDs of associated Puppet environment
* `--puppet-environments VALUE` {endash} Names of associated Puppet environment
* `--realm-ids LIST` {endash} Realm IDs
* `--realms LIST`
* `--smart-proxies LIST`
* `--smart-proxy-ids LIST` {endash} Capsule IDs
* `--subnet-ids LIST` {endash} Subnet IDs
* `--subnets LIST`
* `--title VALUE` {endash} Set the current location context for the request
* `--user-ids LIST` {endash} User IDs
* `--users LIST`
* `-h`, `--help` {endash} Print help
