[id="hammer-os-add-ptable"]
= os add-ptable

[role="_abstract"]
Associate a partition table

*Usage*

----
# hammer os add-ptable [OPTIONS]
----

*Options*

* `--id VALUE`
* `--partition-table VALUE` {endash} Partition table name
* `--partition-table-id NUMBER`
* `--title VALUE` {endash} Operating system title
* `-h`, `--help` {endash} Print help
