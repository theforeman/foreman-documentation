[id="hammer-puppet-environment-info"]
= puppet-environment info

[role="_abstract"]
Show an environment

*Usage*

----
# hammer puppet-environment <info|show> [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--id VALUE`
* `--name VALUE` {endash} Puppet environment name
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS         | ALL | DEFAULT | THIN

| Id             | x   | x       | x
| Name           | x   | x       | x
| Puppetclasses/ | x   | x       |
| Locations/     | x   | x       |
| Organizations/ | x   | x       |
| Created at     | x   | x       |
| Updated at     | x   | x       |
|===
