= location add-medium

[role="_abstract"]
Associate a medium

*Usage*

----
# hammer location add-medium [OPTIONS]
----

*Options*

* `--id VALUE`
* `--medium VALUE` {endash} Medium name
* `--medium-id NUMBER`
* `--name VALUE` {endash} Set the current location context for the request
* `--title VALUE` {endash} Set the current location context for the request
* `-h`, `--help` {endash} Print help
