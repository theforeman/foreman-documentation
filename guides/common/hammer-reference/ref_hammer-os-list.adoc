[id="hammer-os-list"]
= os list

[role="_abstract"]
List all operating systems

*Usage*

----
# hammer os <list|index> [OPTIONS]
----

*Options*

* `--architecture VALUE` {endash} Architecture name
* `--architecture-id VALUE` {endash} ID of architecture
* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-title VALUE` {endash} Set the current location context for the request
* `--medium VALUE` {endash} Medium name
* `--medium-id VALUE` {endash} ID of medium
* `--order VALUE` {endash} Sort and order by a searchable field, e.g. `<field> DESC`
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--os-parameters-attributes SCHEMA` Array of parameters
* `--page NUMBER` {endash} Page number, starting at 1
* `--partition-table VALUE` {endash} Partition table name
* `--partition-table-id VALUE` {endash} ID of partition table
* `--per-page VALUE` {endash} Number of results per page to return, `all` to return all results
* `--provisioning-template VALUE` {endash} Name to search by
* `--provisioning-template-id VALUE`  ID of template
* `--search VALUE` {endash} Filter results
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS       | ALL | DEFAULT | THIN

| Id           | x   | x       | x
| Title        | x   | x       | x
| Release name | x   | x       |
| Family       | x   | x       |
|===

Following parameters accept format defined by its schema (bold are required; <> contains acceptable type; [] contains acceptable value):

* `--os-parameters-attributes`  "*name*=<string>,*value*=<string>, ... "

*Search and order fields*

* `architecture` {endash} string
* `description` {endash} string
* `family` {endash} string
* `id` {endash} integer
* `major` {endash} string
* `medium` {endash} string
* `minor` {endash} string
* `name` {endash} string
* `params` {endash} string
* `template` {endash} string
* `title` {endash} string
