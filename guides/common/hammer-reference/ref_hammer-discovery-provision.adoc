[id="hammer-discovery-provision"]
= discovery provision

[role="_abstract"]
Provision a discovered host

*Usage*

----
# hammer discovery provision [OPTIONS]
----

*Options*

* `--architecture VALUE` {endash} Architecture name
* `--architecture-id NUMBER` {endash} Required if host is managed and value is not inherited from host group
* `--ask-root-password BOOLEAN`
* `--build BOOLEAN`
* `--capabilities VALUE`
* `--config-group-ids LIST` {endash} IDs of associated config groups
* `--config-groups LIST`
* `--domain VALUE` {endash} Domain name
* `--domain-id NUMBER` {endash} Required if host is managed and value is not inherited from host group
* `--enabled BOOLEAN`
* `--environment VALUE` {endash} Environment name (--environment is deprecated: Use `--puppet-environment[-id]`
instead)
* `--environment-id NUMBER` {endash} (--environment-id is deprecated: Use `--puppet-environment[-id]` instead)
* `--hostgroup VALUE` {endash} Hostgroup name
* `--hostgroup-id NUMBER`
* `--hostgroup-title VALUE` {endash} Hostgroup title
* `--id VALUE`
* `--image VALUE` {endash} Name to search by
* `--image-id NUMBER`
* `--interface KEY_VALUE_LIST` {endash} Interface parameters
Can be specified multiple times.
* `--ip VALUE` {endash} Not required if using a subnet with DHCP Capsule
* `--ip6 VALUE` {endash} Not required if using an IPv6 subnet with DHCP Capsule
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-title VALUE` {endash} Set the current location context for the request
* `--mac VALUE` {endash} Not required if it`s a virtual machine
* `--managed BOOLEAN`
* `--medium VALUE` {endash} Medium name
* `--medium-id VALUE` {endash} Required if not imaged based provisioning and host is managed and value is not
inherited from host group
* `--model VALUE` {endash} Model name
* `--model-id NUMBER`
* `--name VALUE`
* `--new-name VALUE`
* `--operatingsystem VALUE` {endash} Operating system title
* `--operatingsystem-id NUMBER` {endash} Required if host is managed and value is not inherited from host group
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--overwrite BOOLEAN`
* `--owner-id NUMBER`
* `--owner-type ENUM` {endash} Host`s owner type
Possible value(s): `User`, `Usergroup`
* `--parameters KEY_VALUE_LIST` {endash} Host parameters
* `--partition-table VALUE` {endash} Partition table name
* `--partition-table-id NUMBER`
* `--progress-report-id VALUE` {endash} UUID to track orchestration tasks status, GET /api/orchestration/:UUID/tasks
* `--provision-method ENUM` {endash} Possible value(s): `build`, `image`
* `--puppet-ca-proxy VALUE` {endash} Name of Puppet CA proxy
* `--puppet-ca-proxy-id NUMBER`
* `--puppet-class-ids LIST` {endash} List of Puppet class ids
* `--puppet-classes LIST`
* `--puppet-environment VALUE` {endash} Name of associated Puppet environment
* `--puppet-environment-id NUMBER` ID of associated Puppet environment
* `--puppet-proxy VALUE` {endash} Name of Puppet proxy
* `--puppet-proxy-id NUMBER`
* `--pxe-loader ENUM` {endash} DHCP filename option (Grub2 or PXELinux by default)
Possible value(s): `None`, `PXELinux BIOS`, `PXELinux UEFI`, `Grub UEFI`, `Grub2
BIOS`, `Grub2 ELF`, `Grub2 UEFI`, `Grub2 UEFI SecureBoot`, `Grub2 UEFI HTTP`,
`Grub2 UEFI HTTPS`, `Grub2 UEFI HTTPS SecureBoot`, `iPXE Embedded`, `iPXE UEFI
HTTP`, `iPXE Chain BIOS`, `iPXE Chain UEFI`
* `--root-password VALUE`
* `--sp-subnet-id NUMBER`
* `--subnet VALUE` {endash} Subnet name
* `--subnet-id NUMBER` {endash} Required if host is managed and value is not inherited from host group
* `--subnet6-id NUMBER` {endash} Required if host is managed, does not have IPv4 IP / Subnet, or the value is not
inherited from the host group
* `-h`, `--help` {endash} Print help
