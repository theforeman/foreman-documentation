[id="hammer-architecture-info"]
= architecture info

[role="_abstract"]
Show an architecture

*Usage*

----
# hammer architecture <info|show> [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--id VALUE`
* `--name VALUE` {endash} Architecture name
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS             | ALL | DEFAULT | THIN

| Id                 | x   | x       | x
| Name               | x   | x       | x
| Operating systems/ | x   | x       |
| Locations/         | x   | x       |
| Organizations/     | x   | x       |
| Created at         | x   | x       |
| Updated at         | x   | x       |
|===
