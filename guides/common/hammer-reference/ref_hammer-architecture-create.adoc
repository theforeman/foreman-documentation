= architecture create

[role="_abstract"]
Create an architecture

*Usage*

----
# hammer architecture create [OPTIONS]
----

*Options*

* `--name VALUE`
* `--operatingsystem-ids LIST` {endash} Operating system IDs
* `--operatingsystems LIST`
* `-h`, `--help` {endash} Print help
