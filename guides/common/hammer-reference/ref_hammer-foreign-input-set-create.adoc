[id="hammer-foreign-input-set-create"]
= foreign-input-set create

[role="_abstract"]
Create a foreign input set

*Usage*

----
# hammer foreign-input-set create [OPTIONS]
----

*Options*

* `--description VALUE` {endash} Input set description
* `--exclude VALUE` {endash} A comma separated list of input names to be included from the foreign template.
* `--include VALUE` {endash} A comma separated list of input names to be included from the foreign template.
* `--include-all BOOLEAN` {endash} Include all inputs from the foreign template
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-title VALUE` {endash} Set the current location context for the request
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--target-template-id VALUE` {endash} Target template ID
* `--template-id VALUE`
* `-h`, `--help` {endash} Print help
