[id="hammer-subnet-create"]
= subnet create

[role="_abstract"]
Create a subnet

*Usage*

----
# hammer subnet create [OPTIONS]
----

*Options*

* `--bmc VALUE` {endash} BMC Proxy to use within this subnet
* `--bmc-id NUMBER` {endash} BMC Capsule ID to use within this subnet
* `--boot-mode ENUM` {endash} Default boot mode for interfaces assigned to this subnet.
Possible value(s): `Static`, `DHCP`
* `--description VALUE` {endash} Subnet description
* `--dhcp VALUE` {endash} DHCP Proxy to use within this subnet
* `--dhcp-id NUMBER` {endash} DHCP Capsule ID to use within this subnet
* `--discovery-id NUMBER` {endash} ID of Discovery Capsule to use within this subnet for managing connection to
discovered hosts
* `--dns VALUE` {endash} DNS Proxy to use within this subnet
* `--dns-id NUMBER` {endash} DNS Capsule ID to use within this subnet
* `--dns-primary VALUE` {endash} Primary DNS for this subnet
* `--dns-secondary VALUE` {endash} Secondary DNS for this subnet
* `--domain-ids LIST` {endash} Domains in which this subnet is part
* `--domains LIST`
* `--externalipam-group VALUE` {endash} External IPAM group - only relevant when IPAM is set to external
* `--externalipam-id NUMBER` {endash} External IPAM Capsule ID to use within this subnet
* `--from VALUE` {endash} Starting IP Address for IP auto suggestion
* `--gateway VALUE` {endash} Subnet gateway
* `--httpboot-id NUMBER` {endash} HTTPBoot Capsule ID to use within this subnet
* `--ipam ENUM` {endash} IP Address auto suggestion mode for this subnet.
Possible value(s): `DHCP`, `Internal DB`, `Random DB`, `EUI-64`, `External
IPAM`, `None`
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-ids LIST` {endash} REPLACE locations with given ids
* `--location-title VALUE` {endash} Set the current location context for the request
* `--location-titles LIST`
* `--locations LIST`
* `--mask VALUE` {endash} Netmask for this subnet
* `--mtu NUMBER` {endash} MTU for this subnet
* `--name VALUE` {endash} Subnet name
* `--network VALUE` {endash} Subnet network
* `--network-type ENUM` {endash} Type or protocol, IPv4 or IPv6, defaults to IPv4
Possible value(s): `IPv4`, `IPv6`
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-ids LIST` {endash} REPLACE organizations with given ids.
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--organization-titles LIST`
* `--organizations LIST`
* `--prefix VALUE` {endash} Network prefix in CIDR notation (e.g. 64) for this subnet
* `--remote-execution-proxy-ids LIST` List of Capsule IDs to be used for remote execution
* `--template-id NUMBER` {endash} Template HTTP proxy ID to use within this subnet
* `--tftp VALUE` {endash} TFTP Proxy to use within this subnet
* `--tftp-id NUMBER` {endash} TFTP Capsule ID to use within this subnet
* `--to VALUE` {endash} Ending IP Address for IP auto suggestion
* `--vlanid VALUE` {endash} VLAN ID for this subnet
* `-h`, `--help` {endash} Print help
