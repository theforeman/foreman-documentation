[id="hammer-repository-types"]
= repository types

[role="_abstract"]
Show the available repository types

*Usage*

----
# hammer repository types [OPTIONS]
----

*Options*

* `--creatable BOOLEAN` {endash} When set to `True` repository types that are creatable will be returned
* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS                    | ALL | DEFAULT | THIN

| Name                      | x   | x       | x
| Content types/type        | x   | x       |
| Content types/generic?    | x   | x       |
| Content types/removable?  | x   | x       |
| Content types/uploadable? | x   | x       |
| Content types/indexed?    | x   | x       |
|===
