[id="hammer-repository-set-enable"]
= repository-set enable

[role="_abstract"]
Enable a repository from the set

*Usage*

----
# hammer repository-set enable [OPTIONS]
----

*Options*

* `--basearch VALUE` {endash} Basearch to enable
* `--id NUMBER` {endash} ID of the repository set to enable
* `--name VALUE` {endash} Repository set name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization identifier
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--product VALUE` {endash} Product name to search by
* `--product-id NUMBER` {endash} ID of the product containing the repository set
* `--releasever VALUE` {endash} Releasever to enable
* `-h`, `--help` {endash} Print help
