[id="hammer-capsule-content-add-lifecycle-environment"]
= capsule content add-lifecycle-environment

[role="_abstract"]
Add lifecycle environments to the Capsule

*Usage*

----
# hammer capsule content add-lifecycle-environment [OPTIONS]
----

*Options*

* `--environment VALUE` {endash} Lifecycle environment name to search by (--environment is deprecated: Use
* `--lifecycle-environment` instead)
* `--environment-id NUMBER` {endash} (--environment-id is deprecated: Use `--lifecycle-environment-id` instead)
* `--id NUMBER` {endash} Id of the Capsule
* `--lifecycle-environment VALUE` {endash} Lifecycle environment name to search by
* `--lifecycle-environment-id NUMBER` Id of the lifecycle environment
* `--name VALUE` {endash} Name to search by
* `--organization VALUE` {endash} Organization name
* `--organization-id VALUE` {endash} Organization ID
* `-h`, `--help` {endash} Print help
