= user-group remove-role

[role="_abstract"]
Remove a user role

*Usage*

----
# hammer user-group remove-role [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Name to search by
* `--role VALUE` {endash} User role name
* `--role-id NUMBER`
* `-h`, `--help` {endash} Print help
