[id="hammer-alternate-content-source-update"]
= alternate-content-source update

[role="_abstract"]
Update an alternate content source.

*Usage*

----
# hammer alternate-content-source update [OPTIONS]
----

*Options*

* `--base-url VALUE` {endash} Base URL for finding alternate content
* `--description VALUE` {endash} Description for the alternate content source
* `--id NUMBER` {endash} Alternate content source ID
* `--name VALUE` {endash} Name of the alternate content source
* `--new-name VALUE` {endash} Name of the alternate content source
* `--product-ids LIST` {endash} IDs of products to copy repository information from into a Simplified Alternate
Content Source. Products must include at least one repository of the chosen
content type.
* `--products LIST`
* `--smart-proxies LIST`
* `--smart-proxy-ids LIST` {endash} Ids of Capsules to associate
* `--smart-proxy-names LIST` {endash} Names of Capsules to associate
* `--ssl-ca-cert-id NUMBER` {endash} Identifier of the content credential containing the SSL CA Cert
* `--ssl-client-cert-id NUMBER` {endash} Identifier of the content credential containing the SSL Client Cert
* `--ssl-client-key-id NUMBER` {endash} Identifier of the content credential containing the SSL Client Key
* `--subpaths LIST` {endash} Path suffixes for finding alternate content
* `--upstream-password VALUE` {endash} Basic authentication password
* `--upstream-username VALUE` {endash} Basic authentication username
* `--use-http-proxies BOOLEAN` {endash} If the Capsules` assigned HTTP proxies should be used
* `--verify-ssl BOOLEAN` {endash} If SSL should be verified for the upstream URL
* `-h`, `--help` {endash} Print help
