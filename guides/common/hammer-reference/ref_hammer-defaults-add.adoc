[id="hammer-defaults-add"]
= defaults add

[role="_abstract"]
Add a default parameter to config

*Usage*

----
# hammer defaults add [OPTIONS]
----

*Options*

* `--param-name VALUE` {endash} The name of the default option (e.g. organization_id)
* `--param-value VALUE` {endash} The value for the default option
* `--provider VALUE` {endash} The name of the provider providing the value. For list available providers see
`hammer defaults providers`
* `-h`, `--help` {endash} Print help
