[id="hammer-flatpak-remote-create"]
= flatpak-remote create

[role="_abstract"]
Create a flatpak remote

*Usage*

----
# hammer flatpak-remote create [OPTIONS]
----

*Options*

* `--description VALUE` {endash} Description of the flatpak remote
* `--name VALUE` {endash} Name
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization identifier
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--token VALUE` {endash} Token/password for the flatpak remote
* `--url VALUE` {endash} Base URL of the flatpak registry index, ex: flatpaks.redhat.io/rhel/ ,
registry.fedoraproject.org/.
* `--username VALUE` {endash} Username for the flatpak remote
* `-h`, `--help` {endash} Print help
