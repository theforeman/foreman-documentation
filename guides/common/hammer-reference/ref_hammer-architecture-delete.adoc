= architecture delete

[role="_abstract"]
Delete an architecture

*Usage*

----
# hammer architecture <delete|destroy> [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Architecture name
* `-h`, `--help` {endash} Print help
