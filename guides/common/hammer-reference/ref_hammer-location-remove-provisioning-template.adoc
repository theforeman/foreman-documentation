[id="hammer-location-remove-provisioning-template"]
= location remove-provisioning-template

[role="_abstract"]
Disassociate provisioning templates

*Usage*

----
# hammer location remove-provisioning-template [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Set the current location context for the request
* `--provisioning-template VALUE` {endash} Name to search by
* `--provisioning-template-id NUMBER`
* `--provisioning-template-ids LIST` {endash} List of provisioning template ids
* `--provisioning-template-search VALUE` Provisioning template name regex to search, all matching templates will be
disassociated
* `--provisioning-templates LIST` {endash} List of provisioning template names
* `--title VALUE` {endash} Set the current location context for the request
* `-h`, `--help` {endash} Print help
