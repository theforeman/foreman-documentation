[id="hammer-repository-synchronize"]
= repository synchronize

[role="_abstract"]
Sync a repository

*Usage*

----
# hammer repository synchronize [OPTIONS]
----

*Options*

* `--async` {endash} Do not wait for the task
* `--id NUMBER` {endash} Repository ID
* `--incremental BOOLEAN` {endash} Perform an incremental import
* `--name VALUE` {endash} Repository name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization ID
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--product VALUE` {endash} Product name to search by
* `--product-id NUMBER` {endash} Product numeric identifier
* `--skip-metadata-check BOOLEAN` Force sync even if no upstream changes are detected. Only used with yum or deb
repositories.
* `--validate-contents BOOLEAN` {endash} Force a sync and validate the checksums of all content. Only used with yum
repositories.
* `-h`, `--help` {endash} Print help
