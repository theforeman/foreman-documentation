[id="hammer-os-delete-parameter"]
= os delete-parameter

[role="_abstract"]
Delete parameter for an operating system

*Usage*

----
# hammer os delete-parameter [OPTIONS]
----

*Options*

* `--name VALUE` {endash} Parameter name
* `--operatingsystem VALUE` {endash} Operating system title
* `--operatingsystem-id NUMBER`
* `-h`, `--help` {endash} Print help
