[id="hammer-repository-list"]
= repository list

[role="_abstract"]
List of enabled repositories

*Usage*

----
# hammer repository <list|index> [OPTIONS]
----

*Options*

* `--ansible-collection VALUE` {endash} Name to search by
* `--ansible-collection-id VALUE` {endash} Id of an ansible collection to find repositories that contain the ansible
collection
* `--archived BOOLEAN` {endash} Show archived repositories
* `--available-for VALUE` {endash} Interpret specified object to return only Repositories that can be associated
with specified object.  Only `content_view` & `content_view_version` are
supported.
* `--content-type VALUE` {endash} Limit the repository type to return. View available types with "hammer
repository types"
* `--content-view VALUE` {endash} Content view name to search by
* `--content-view-id NUMBER` {endash} ID of a content view to show repositories in
* `--content-view-version VALUE` {endash} Content view version number
* `--content-view-version-id NUMBER`  ID of a content view version to show repositories in
* `--deb VALUE` {endash} Name to search by
* `--deb-id VALUE` {endash} Id of a deb package to find repositories that contain the deb
* `--description VALUE` {endash} Description of the repository
* `--download-policy ENUM` {endash} Limit to only repositories with this download policy
Possible value(s): `immediate`, `on_demand`
* `--environment VALUE` {endash} Lifecycle environment name to search by (--environment is deprecated: Use
* `--lifecycle-environment` instead)
* `--environment-id NUMBER` {endash} (--environment-id is deprecated: Use `--lifecycle-environment-id` instead)
* `--erratum-id VALUE` {endash} Id of an erratum to find repositories that contain the erratum
* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--file-id VALUE` {endash} Id of a file to find repositories that contain the file
* `--full-result BOOLEAN` {endash} Whether or not to show all results
* `--label VALUE` {endash} Label of the repository
* `--library BOOLEAN` {endash} Show repositories in Library and the default content view
* `--lifecycle-environment VALUE` {endash} Lifecycle environment name to search by
* `--lifecycle-environment-id NUMBER` ID of an environment to show repositories in
* `--name VALUE` {endash} Name of the repository
* `--order VALUE` {endash} Sort field and order, eg. `id DESC`
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} ID of an organization to show repositories in
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--page NUMBER` {endash} Page number, starting at 1
* `--per-page NUMBER` {endash} Number of results per page to return
* `--product VALUE` {endash} Product name to search by
* `--product-id NUMBER` {endash} ID of a product to show repositories of
* `--rpm-id VALUE` {endash} Id of a rpm package to find repositories that contain the rpm
* `--search VALUE` {endash} Search string
* `--username VALUE` {endash} Only show the repositories readable by this user with this username
* `--with-content VALUE` {endash} Limit the repository type to return. View available types with "hammer
repository types"
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS                   | ALL | DEFAULT | THIN

| Id                       | x   | x       | x
| Name                     | x   | x       | x
| Product                  | x   | x       |
| Content type             | x   | x       |
| Content label            | x   | x       |
| Url                      | x   | x       |
| Upstream repository name | x   |         |
|===

*Search and order fields*

* `container_repository_name` {endash} string
* `content_label` {endash} string
* `content_type` {endash} string
* `content_view_id` {endash} integer
* `description` {endash} text
* `distribution_arch` {endash} string
* `distribution_bootable` {endash} boolean
* `distribution_family` {endash} string
* `distribution_variant` {endash} string
* `distribution_version` {endash} string
* `download_policy` {endash} string
* `label` {endash} string
* `name` {endash} string
* `product` {endash} string
* `product_id` {endash} integer
* `product_name` {endash} string
* `redhat` {endash} Values: true, false
