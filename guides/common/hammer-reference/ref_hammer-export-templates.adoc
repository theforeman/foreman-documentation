[id="hammer-export-templates"]
= export-templates

[role="_abstract"]
Export templates to a git repo or a directory on the server

*Usage*

----
# hammer export-templates [OPTIONS]
----

*Options*

* `--branch VALUE` {endash} Branch in Git repo.
* `--commit-msg VALUE` {endash} Custom commit message for templates export
* `--dirname VALUE` {endash} Directory within Git repo containing the templates.
* `--filter VALUE` {endash} Export templates with names matching this regex (case-insensitive; snippets are
not filtered).
* `--http-proxy VALUE` {endash} Name to search by
* `--http-proxy-id NUMBER` {endash} ID of an HTTP proxy to use for template sync. Use this parameter together with
``http_proxy_policy`:`selected``
* `--http-proxy-policy ENUM` {endash} HTTP proxy policy for template sync. Use only when synchronizing templates
through the HTTP or the HTTPS protocol. If you choose `selected`, provide the
`http_proxy_id` parameter.
Possible value(s): `global`, `none`, `selected`
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--location-ids LIST` {endash} REPLACE locations with given ids
* `--location-title VALUE` {endash} Set the current location context for the request
* `--location-titles LIST`
* `--locations LIST`
* `--metadata-export-mode ENUM` {endash} Specify how to handle metadata
Possible value(s): `refresh`, `keep`, `remove`
* `--negate BOOLEAN` {endash} Negate the prefix (for purging).
* `--organization VALUE` {endash} Set the current organization context for the request
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `--organization-ids LIST` {endash} REPLACE organizations with given ids.
* `--organization-title VALUE` {endash} Set the current organization context for the request
* `--organization-titles LIST`
* `--organizations LIST`
* `--repo VALUE` {endash} Override the default repo from settings.
* `--verbose BOOLEAN` {endash} Be verbose
* `-h`, `--help` {endash} Print help
