[id="hammer-settings-info"]
= settings info

[role="_abstract"]
Show a setting

*Usage*

----
# hammer settings <info|show> [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--id VALUE`
* `--name VALUE` {endash} Setting name
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS        | ALL | DEFAULT | THIN

| Id            | x   | x       | x
| Name          | x   | x       | x
| Description   | x   | x       |
| Category      | x   | x       |
| Settings type | x   | x       |
| Value         | x   | x       |
|===
