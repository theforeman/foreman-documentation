[id="hammer-config-group-delete"]
= config-group delete

[role="_abstract"]
Delete a config group

*Usage*

----
# hammer config-group <delete|destroy> [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Name to search by
* `-h`, `--help` {endash} Print help
