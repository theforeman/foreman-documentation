[id="hammer-user-group-update"]
= user-group update

[role="_abstract"]
Update a user group

*Usage*

----
# hammer user-group update [OPTIONS]
----

*Options*

* `--admin BOOLEAN` {endash} Is an admin user group, can be modified by admins only
* `--id VALUE`
* `--name VALUE`
* `--new-name VALUE`
* `--role-ids LIST`
* `--roles LIST`
* `--user-group-ids LIST`
* `--user-groups LIST`
* `--user-ids LIST`
* `--users LIST`
* `-h`, `--help` {endash} Print help
