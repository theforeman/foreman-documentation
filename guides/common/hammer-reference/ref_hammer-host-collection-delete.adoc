[id="hammer-host-collection-delete"]
= host-collection delete

[role="_abstract"]
Destroy a host collection

*Usage*

----
# hammer host-collection <delete|destroy> [OPTIONS]
----

*Options*

* `--id NUMBER` {endash} Id of the host collection
* `--name VALUE` {endash} Host collection name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization ID
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `-h`, `--help` {endash} Print help
