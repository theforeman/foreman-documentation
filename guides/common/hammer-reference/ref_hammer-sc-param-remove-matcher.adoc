[id="hammer-sc-param-remove-matcher"]
= sc-param remove-matcher

[role="_abstract"]
Delete an override value for a specific smart class parameter

*Usage*

----
# hammer sc-param remove-matcher [OPTIONS]
----

*Options*

* `--id VALUE`
* `--puppet-class VALUE` {endash} Puppet class name
* `--puppet-class-id NUMBER` {endash} ID of Puppet class
* `--smart-class-parameter VALUE` {endash} Smart class parameter name
* `--smart-class-parameter-id VALUE`
* `-h`, `--help` {endash} Print help
