[id="hammer-user-group-add-user"]
= user-group add-user

[role="_abstract"]
Associate an user

*Usage*

----
# hammer user-group add-user [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Name to search by
* `--user VALUE` {endash} User`s login to search by
* `--user-id NUMBER`
* `-h`, `--help` {endash} Print help
