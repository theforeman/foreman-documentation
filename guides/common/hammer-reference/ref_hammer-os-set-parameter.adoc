[id="hammer-os-set-parameter"]
= os set-parameter

[role="_abstract"]
Create or update parameter for an operating system

*Usage*

----
# hammer os set-parameter [OPTIONS]
----

*Options*

* `--hidden-value BOOLEAN` {endash} Should the value be hidden
* `--name VALUE` {endash} Parameter name
* `--operatingsystem VALUE` {endash} Operating system title
* `--operatingsystem-id NUMBER`
* `--parameter-type ENUM` {endash} Type of the parameter
Possible value(s): `string`, `boolean`, `integer`, `real`, `array`, `hash`,
`yaml`, `json`
Default: "string"
* `--value VALUE` {endash} Parameter value
* `-h`, `--help` {endash} Print help
