[id="hammer-proxy-content-update-counts"]
= proxy content update-counts

[role="_abstract"]
Update content counts for the Capsule

*Usage*

----
# hammer proxy content update-counts [OPTIONS]
----

*Options*

* `--async` {endash} Do not wait for the task
* `--content-view VALUE` {endash} Content view name to search by
* `--content-view-id NUMBER` {endash} Id of the content view to limit the content counting on
* `--id NUMBER` {endash} Id of the Capsule
* `--lifecycle-environment VALUE` {endash} Lifecycle environment name to search by
* `--lifecycle-environment-id NUMBER` Id of the environment to limit the content counting on
* `--name VALUE` {endash} Name to search by
* `--organization VALUE` {endash} Organization name
* `--organization-id VALUE` {endash} Organization ID
* `--repository VALUE` {endash} Repository name to search by
* `--repository-id NUMBER` {endash} Id of the repository to limit the content counting on
* `-h`, `--help` {endash} Print help
