[id="hammer-repository-delete"]
= repository delete

[role="_abstract"]
Destroy a custom repository

*Usage*

----
# hammer repository <delete|destroy> [OPTIONS]
----

*Options*

* `--delete-empty-repo-filters BOOLEAN` {endash} Delete content view filters that have this repository as the last associated
repository. Defaults to true. If false, such filters will now apply to all
repositories in the content view.
* `--id NUMBER`
* `--name VALUE` {endash} Repository name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id VALUE` {endash} Organization ID to search by
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title to search by
* `--product VALUE` {endash} Product name to search by
* `--product-id NUMBER` {endash} Product numeric identifier
* `--remove-from-content-view-versions BOOLEAN` Force delete the repository by removing it from all content view versions
* `-h`, `--help` {endash} Print help
