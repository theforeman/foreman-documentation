[id="hammer-host-subscription-remove"]
= host subscription remove

[role="_abstract"]
Remove subscriptions from a host

*Usage*

----
# hammer host subscription remove [OPTIONS]
----

*Options*

* `--host VALUE` {endash} Host name
* `--host-id NUMBER` {endash} Id of the host
* `--quantity VALUE` {endash} Remove the first instance of a subscription with matching id and quantity
* `--subscription-id VALUE` {endash} ID of subscription
* `-h`, `--help` {endash} Print help
