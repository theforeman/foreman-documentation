[id="hammer-product-synchronize"]
= product synchronize

[role="_abstract"]
Sync all repositories for a product

*Usage*

----
# hammer product synchronize [OPTIONS]
----

*Options*

* `--async` {endash} Do not wait for the task
* `--id NUMBER` {endash} Product ID
* `--name VALUE` {endash} Product name to search by
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER`
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `-h`, `--help` {endash} Print help
