[id="hammer-package-list"]
= package list

[role="_abstract"]
List packages

*Usage*

----
# hammer package <list|index> [OPTIONS]
----

*Options*

* `--available-for VALUE` {endash} Return packages that can be added to the specified object.  Only the value
`content_view_version` is supported.
* `--content-view VALUE` {endash} Content view name to search by
* `--content-view-filter VALUE` {endash} Name to search by
* `--content-view-filter-id NUMBER` {endash} Content View Filter identifier. Use to filter by ID
* `--content-view-id NUMBER` {endash} Content view numeric identifier
* `--content-view-version VALUE` {endash} Content view version number
* `--content-view-version-id NUMBER` {endash} Content View Version identifier
* `--environment VALUE` {endash} Lifecycle environment name to search by (--environment is deprecated: Use
* `--lifecycle-environment` instead)
* `--environment-id NUMBER` {endash} (--environment-id is deprecated: Use `--lifecycle-environment-id` instead)
* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--full-result BOOLEAN` {endash} Whether or not to show all results
* `--host VALUE` {endash} Host name
* `--host-id NUMBER` {endash} Host id to list applicable packages for
* `--ids LIST` {endash} Package identifiers to filter content by
* `--lifecycle-environment VALUE` {endash} Lifecycle environment name to search by
* `--lifecycle-environment-id NUMBER` {endash} Environment identifier
* `--order VALUE` {endash} Sort field and order, eg. `id DESC`
* `--organization VALUE` {endash} Organization name to search by
* `--organization-id NUMBER` {endash} Organization identifier
* `--organization-label VALUE` {endash} Organization label to search by
* `--organization-title VALUE` {endash} Organization title
* `--packages-restrict-applicable BOOLEAN` Return packages that are applicable to one or more hosts (defaults to true if
host_id is specified)
* `--packages-restrict-latest BOOLEAN` {endash} Return only the latest version of each package
* `--packages-restrict-upgradable BOOLEAN` Return packages that are upgradable on one or more hosts
* `--page NUMBER` {endash} Page number, starting at 1
* `--per-page NUMBER` {endash} Number of results per page to return
* `--product VALUE` {endash} Product name to search by
* `--product-id NUMBER` {endash} Product numeric identifier
* `--repository VALUE` {endash} Repository name to search by
* `--repository-id NUMBER` {endash} Repository identifier
* `--search VALUE` {endash} Search string
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS     | ALL | DEFAULT

| Id         | x   | x
| Filename   | x   | x
| Source rpm | x   | x
|===
