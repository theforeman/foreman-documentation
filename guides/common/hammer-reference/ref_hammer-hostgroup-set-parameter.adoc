[id="hammer-hostgroup-set-parameter"]
= hostgroup set-parameter

[role="_abstract"]
Create or update parameter for a hostgroup

*Usage*

----
# hammer hostgroup set-parameter [OPTIONS]
----

*Options*

* `--hidden-value BOOLEAN` {endash} Should the value be hidden
* `--hostgroup VALUE` {endash} Hostgroup name
* `--hostgroup-id NUMBER`
* `--hostgroup-title VALUE` {endash} Hostgroup title
* `--name VALUE` {endash} Parameter name
* `--parameter-type ENUM` {endash} Type of the parameter
Possible value(s): `string`, `boolean`, `integer`, `real`, `array`, `hash`,
`yaml`, `json`
Default: "string"
* `--value VALUE` {endash} Parameter value
* `-h`, `--help` {endash} Print help
