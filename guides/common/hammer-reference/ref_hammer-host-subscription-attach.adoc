[id="hammer-host-subscription-attach"]
= host subscription attach

[role="_abstract"]
Add a subscription to a host

*Usage*

----
# hammer host subscription attach [OPTIONS]
----

*Options*

* `--host VALUE` {endash} Host name
* `--host-id NUMBER` {endash} Id of the host
* `--quantity VALUE` {endash} Quantity of this subscriptions to add. Defaults to 1
* `--subscription-id VALUE` {endash} ID of subscription
* `-h`, `--help` {endash} Print help
