= location add-hostgroup

[role="_abstract"]
Associate a hostgroup

*Usage*

----
# hammer location add-hostgroup [OPTIONS]
----

*Options*

* `--hostgroup VALUE` {endash} Hostgroup name
* `--hostgroup-id NUMBER`
* `--hostgroup-title VALUE` {endash} Hostgroup title
* `--id VALUE`
* `--name VALUE` {endash} Set the current location context for the request
* `--title VALUE` {endash} Set the current location context for the request
* `-h`, `--help` {endash} Print help
