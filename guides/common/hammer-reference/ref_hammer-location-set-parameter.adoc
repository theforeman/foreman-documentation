[id="hammer-location-set-parameter"]
= location set-parameter

[role="_abstract"]
Create or update parameter for a location

*Usage*

----
# hammer location set-parameter [OPTIONS]
----

*Options*

* `--hidden-value BOOLEAN` {endash} Should the value be hidden
* `--location VALUE` {endash} Set the current location context for the request
* `--location-id NUMBER`
* `--location-title VALUE` {endash} Set the current location context for the request
* `--name VALUE` {endash} Parameter name
* `--parameter-type ENUM` {endash} Type of the parameter
Possible value(s): `string`, `boolean`, `integer`, `real`, `array`, `hash`,
`yaml`, `json`
Default: "string"
* `--value VALUE` {endash} Parameter value
* `-h`, `--help` {endash} Print help
