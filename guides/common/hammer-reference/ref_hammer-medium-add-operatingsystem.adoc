= medium add-operatingsystem

[role="_abstract"]
Associate an operating system

*Usage*

----
# hammer medium add-operatingsystem [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Medium name
* `--operatingsystem VALUE` {endash} Operating system title
* `--operatingsystem-id NUMBER`
* `-h`, `--help` {endash} Print help
