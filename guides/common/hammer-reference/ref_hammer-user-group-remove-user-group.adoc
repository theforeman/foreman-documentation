= user-group remove-user-group

[role="_abstract"]
Disassociate an user group

*Usage*

----
# hammer user-group remove-user-group [OPTIONS]
----

*Options*

* `--id VALUE`
* `--name VALUE` {endash} Name to search by
* `--user-group VALUE` {endash} Name to search by
* `--user-group-id NUMBER`
* `-h`, `--help` {endash} Print help
