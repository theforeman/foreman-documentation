[id="hammer-tailoring-file-info"]
= tailoring-file info

[role="_abstract"]
Show a Tailoring file

*Usage*

----
# hammer tailoring-file <info|show> [OPTIONS]
----

*Options*

* `--fields LIST` {endash} Show specified fields or predefined field sets only. (See below)
* `--id VALUE`
* `--location VALUE` {endash} Name to search by
* `--location-id NUMBER` {endash} Set the current location context for the request
* `--name VALUE` {endash} Tailoring file name
* `--organization VALUE` {endash} Name to search by
* `--organization-id NUMBER` {endash} Set the current organization context for the request
* `-h`, `--help` {endash} Print help

*Predefined field sets*

|===
| FIELDS                             | ALL | DEFAULT | THIN

| Id                                 | x   | x       | x
| Name                               | x   | x       | x
| Created at                         | x   | x       |
| Original filename                  | x   | x       |
| Tailoring file profiles/id         | x   | x       |
| Tailoring file profiles/profile id | x   | x       |
| Tailoring file profiles/title      | x   | x       |
| Locations/                         | x   | x       |
| Organizations/                     | x   | x       |
|===
