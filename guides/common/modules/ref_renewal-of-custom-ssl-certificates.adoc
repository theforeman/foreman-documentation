= Renewal of {customssl} certificates

[role="_abstract"]
If you configured {ProjectName} with {customssl} certificates, you are responsible for renewing those certificates when they expire.

When you renew {customssl} certificates, you must replace the certificate files by using the same installation options that you used during the original installation.

ifndef::containerized[]
[NOTE]
====
To renew the certificate authority (CA) certificate, follow a separate process to avoid disrupting connections to existing hosts.
For more information, see xref:planning-for-self-signed-ca-certificate-renewal[].
====
endif::[]
