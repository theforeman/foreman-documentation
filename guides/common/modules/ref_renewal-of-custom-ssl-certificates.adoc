= Renewal of {customssl} certificates

[role="_abstract"]
If you configured {ProjectName} with {customssl} certificates, you are responsible for renewing those certificates when they expire.

When you renew {customssl} certificates, you must replace them by using the same installation options that you used during the original installation.
For example, if you used the `--certs-server-cert`, `--certs-server-key`, and `--certs-server-ca-cert` options during installation, renew the {customssl} certificate files by using the same options.

[IMPORTANT]
====
Renewing certificates is different from renewing the certificate authority (CA).
This section covers certificate renewal only.
For information about renewing the CA, see xref:planning-for-self-signed-ca-certificate-renewal[].
====
