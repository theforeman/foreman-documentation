= Renewal of {customssl} certificates

[role="_abstract"]
When you use {customssl} certificates with {ProjectName}, you must renew them before they expire and redeploy them using the same installer options as during the original installation.

ifndef::containerized[]
[NOTE]
====
To renew the certificate authority (CA) certificate, follow a separate process to avoid disrupting connections to existing hosts.
For more information, see xref:planning-for-self-signed-ca-certificate-renewal[].
====
endif::[]
