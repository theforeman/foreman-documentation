= Supported OpenVox and Puppet versions

[role="_abstract"]
Before you begin with the Puppet integration, review the supported OpenVox and Puppet versions.

The following versions of the OpenVox server are supported on {ProjectServer} and {SmartProxyServers}:

* OpenVox 8

[IMPORTANT]
====
Ensure that the Puppet modules you are using are compatible with the supported version of OpenVox server.
====

The following versions of OpenVox agent and Puppet agent are supported on {Project} hosts:

* OpenVox agent 8
* Puppet agent 7
