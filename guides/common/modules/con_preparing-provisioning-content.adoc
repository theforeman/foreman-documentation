= Preparing provisioning content

[role="_abstract"]
You have to configure provisioning content that will be used for installation of the operating system on the provisioned host.

ifdef::katello,orcharhino[]
[IMPORTANT]
====
During host provisioning, you must set a *Content Source*, a *Lifecycle Environment*, and a *Content View* to ensure that your host has access to {project-client-name} repositories.
Note that you cannot solely set *Content Source* because provisioning templates expect either no *Content Source* or values for all three fields.
====
endif::[]
