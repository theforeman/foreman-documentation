= Upgrade Puppet agent 7 to OpenVox agent 8

[role="_abstract"]
If you use Puppet to configure hosts, you have the option to upgrade `puppet agent 7 to OpenVox agent 8.
Puppet agent 7 remains compatible with OpenVox in {Project} as a legacy alternative, but {Team} recommends to upgrade to OpenVox agent 8.
For more information, see {ManagingConfigurationsPuppetDocURL}openvox-versions-supported-for-integration-with-satellite[OpenVox versions supported for integration with {Project}] in _{ManagingConfigurationsPuppetDocTitle}_.

.Prerequisites
* *{project-client-openvox}* repository is enabled on the host.

.Procedure
* On your hosts, upgrade the `puppet-agent` agent package to `openvox-agent`:
+
[options="nowrap" subs="+quotes,verbatim,attributes"]
----
# {project-package-update} puppet-agent
----
