= Install the Ansible Director plugin on {SmartProxies}

[role="_abstract"]
Install the Ansible Director plugin on your {SmartProxies} to configure hosts with platform-independent Ansible and Python versions through {SmartProxyServers}.

.Procedure
. On your {SmartProxyServer}, install the Ansible Director plugin:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --enable-foreman-proxy-plugin-ansible-director
----
. Optional: Install the Hammer CLI plugin for Ansible Director:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --enable-foreman-cli-ansible-director
----

.Verification
. In the {ProjectWebUI}, navigate to *Infrastructure* > *{SmartProxies}*.
. Select your {SmartProxy}.
. On the *Overview* tab, verify that Ansible Director is listed under *Active features*.
