= Creating a conventional backup

[role="_abstract"]
You can use conventional backup methods to back up your {ProjectServer} or {SmartProxyServer}.
Unlike {Project} backup methods, conventional backup methods require you to stop {Project} services.

.Procedure
. When creating a snapshot or conventional backup, stop all {Project} services:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-maintain} service stop
----
. Create the snapshot or conventional backup.
ifndef::orcharhino,foreman-deb[]
For more information, see {RHELDocsBaseURL}9/html-single/configuring_basic_system_settings/index#assembly_recovering-and-restoring-a-system_configuring-basic-system-settings[Recovering and restoring a system in _{RHEL}{nbsp}9 Configuring basic system settings_].
endif::[]
. Start {Project} services after creating a snapshot or conventional backup:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-maintain} service start
----
