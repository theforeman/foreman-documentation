= Using the {Project} MCP server

The MCP (Model Context Protocol) technology is designed for easier integration of AI applications.
It is an open protocol that standardizes how AI applications integrate with data sources.
{Project} provides an MCP server to enable integration with your MCP-compatible AI applications.
