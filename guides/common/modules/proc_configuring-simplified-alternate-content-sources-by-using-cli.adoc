= Configuring simplified alternate content sources by using Hammer CLI

[role="_abstract"]
You can configure simplified alternate content sources (ACS) to reuse the upstream source to download content during synchronization on {SmartProxyServers}.

.Procedure
. Create a simplified ACS:
+
[options="nowrap" subs="+quotes,attributes"]
----
$ hammer alternate-content-source create \
--alternate-content-source-type simplified \
--name _My_ACS_Name_ \
--product-ids _My_Product_ID_1_,_My_Product_ID_2_ \
--smart-proxy-ids __My_{smart-proxy-context-titlecase}_ID_1__,__My_{smart-proxy-context-titlecase}_ID_2__
----
. Check if the newly created ACS is listed:
+
[options="nowrap" subs="+quotes,attributes"]
----
$ hammer alternate-content-source list --search "_My_ACS_Name_"
----
. Refresh the ACS:
+
[options="nowrap" subs="+quotes,attributes"]
----
$ hammer alternate-content-source refresh --id _My_ACS_ID_
----

.Verification
* Verify that the task ran succesfully:
+
[options="nowrap" subs="+quotes,attributes"]
----
$ hammer task list --search "Refresh Alternate Content Source"
----
