= Assigning content view environments to a host by using Subscription Manager

[role="_abstract"]
You can assign content view environments to a host by running Subscription Manager on the host.
A host must have at least one content view environment assigned at all times.

.Prerequisites
* If you want to assign multiple content view environments, the *Allow multiple content views* setting must be enabled in {Project}.
For more information, see xref:content-view-environments-overview[].

.Procedure
. List available content view environments:
+
[options="nowrap" subs="+quotes"]
----
# subscription-manager environments --list
----
+
Note the names of the environments you want to assign to the host.
. Assign the content view environments to the host:
+
[options="nowrap" subs="+quotes"]
----
# subscription-manager environments --set _My_Environment_1_,_My_Environment_2_
----

.Verification
* List the assigned content view environments:
+
[options="nowrap" subs="+quotes"]
----
# subscription-manager environments --list-enabled
----
