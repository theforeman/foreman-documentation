= Adding VMware images to {ProjectServer} by using Hammer CLI

[role="_abstract"]
You must add VMware template details to {ProjectServer} for image-based provisioning.

.Procedure
* Create an image.
Use the `--uuid` field to store the relative template path on the vSphere environment:
+
[options="nowrap" subs="+quotes"]
----
$ hammer compute-resource image create \
--architecture "_My_Architecture_" \
--compute-resource "_My_VMware_"
--name "_My_Image_" \
--operatingsystem "_My_Operating_System_" \
--username root \
--uuid "_My_UUID_"
----
