= Setting up an installation medium for Oracle Linux 9

[role="_abstract"]
Set up an installation medium for Oracle Linux 9 to provision hosts in disconnected environments.

.Procedure
. In the {ProjectWebUI}, navigate to *Hosts* > *Provisioning Setup* > *Installation Media*.
. Select your installation medium for Oracle Linux 9.
. Enter the *Path* to the `BaseOS` directory within the extracted installation medium.
. Click *Submit* to save your change.
