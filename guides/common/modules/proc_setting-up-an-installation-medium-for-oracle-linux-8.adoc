= Setting up an installation medium for Oracle Linux 8

[role="_abstract"]
Set up an installation medium for Oracle Linux 8 to provision hosts in disconnected environments.

.Procedure
. In the {ProjectWebUI}, navigate to *Hosts* > *Provisioning Setup* > *Installation Media*.
. Select your installation medium for Oracle Linux 8.
. Enter the *Path* to the `BaseOS` directory within the extracted installation medium.
. Click *Submit* to save your change.
. Navigate to *Hosts* > *Provisioning Setup* > *Operating Systems*.
. Select your operating system entry for Oracle linux 8.
. On the *Parameters* tab, enter the path to the `AppStream` repository within the extracted installation medium.
Set the variable `additional_media` as type `string` with the value `[{"url": "http://{foreman-example-com}/pub/installation_media/ol/8/AppStream/", "name": "AppStream"}]`.
. On your {ProjectServer}, create a symbolic link in the `/BaseOS` path for this version:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# cd /var/www/html/pub/installation_media/ol/8/BaseOS/
# ln --symbolic ../images
----
+
This is required for {Project} to find the kernel and initrd files which are used for network based installations of hosts running Oracle Linux 8.
