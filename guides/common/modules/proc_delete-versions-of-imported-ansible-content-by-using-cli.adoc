= Delete versions of imported Ansible content by using Hammer CLI

[role="_abstract"]
You can delete versions of imported Ansible content from {Project} by using Hammer CLI.

.Prerequisites
* Your {Project} account has a role that grants the `destroy_ansible_content`, `view_ansible_content`, and `view_organizations` permissions.
* You have installed the Hammer CLI plugin for Ansible Director.
For more information, see xref:install-the-hammer-cli-plugin-for-ansible-director[].

.Procedure
. Optional: List imported Ansible content:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ hammer
----
. Optional: List all versions of imported Ansible content:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ hammer
----
. Delete a version of imported Ansible content:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ hammer
----

.Verification
* List all versions of imported Ansible content:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ hammer
----
+
Verify that the version that you have deleted is not part of the output.
