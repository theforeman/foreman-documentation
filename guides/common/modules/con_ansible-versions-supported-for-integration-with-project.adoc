= Ansible versions supported for integration with {Project}

[role="_abstract"]
ifdef::foreman-deb[]
Supported Ansible versions follow the Ansible versions provided by the base operating system of {ProjectServer} and {SmartProxies}.
endif::[]
ifndef::foreman-deb[]
endif::[]

ifdef::foreman-deb[]
{Project} uses Ansible as provided by the base operating system of {ProjectServer} or any {SmartProxies} for remote execution.
Therefore, the supported version of Ansible depends on your base operating system configuration.
endif::[]
ifndef::foreman-deb[]
{Project} repositories include `ansible-core` as a packaged dependency for Ansible integration with {ProjectServer} and {SmartProxyServers}.
The supported version of Ansible is `ansible-core` 2.16.
endif::[]
