= Repository definitions for CentOS Stream

[role="_abstract"]
Use these example repository names and upstream URLs when you synchronize CentOS Stream BaseOS, AppStream, extras, and the optional {project-client-name} content.

[id="centos-stream-10"]
.CentOS Stream 10
[width="100%",cols="30%,70%",options="header"]
|===
| Repository name | Upstream URL
| `CentOS Stream 10 BaseOS` | `https://mirror.stream.centos.org/10-stream/BaseOS/x86_64/os/`
| `CentOS Stream 10 AppStream` | `https://mirror.stream.centos.org/10-stream/AppStream/x86_64/os/`
| `CentOS Stream 10 extras` | `https://mirror.stream.centos.org/SIGs/10-stream/extras/x86_64/extras-common/`
ifdef::katello[]
| `{project-client-name}` | `{project-client-url}/el10/x86_64/`
endif::[]
ifdef::orcharhino[]
| `{project-client-name}` | see {atix-kb-clients} in the _ATIX Service Portal_
endif::[]
|===

GPG key: https://www.centos.org/keys/RPM-GPG-KEY-CentOS-Official
