= Delete an Ansible lifecycle environment path by using Hammer CLI

[role="_abstract"]
You can delete an Ansible lifecycle environment path in {Project} by using {ProjectWebUI}.

.Prerequisites
* Your {Project} account has a role that grants the `destroy_ansible_lifecycle_environment_paths`, `view_ansible_content`, `view_ansible_lifecycle_environment_paths`, `view_ansible_lifecycle_environments`, and `view_organizations` permissions.
* You have installed the Hammer CLI plugin for Ansible Director.
For more information, see xref:install-the-hammer-cli-plugin-for-ansible-director[].

.Procedure
* Delete your Ansible lifecycle environment path:
+
----
$ hammer
----

.Verification
* Verify that {Project} does not contain your previously deleted lifecycle environment path:
+
----
$ hammer
----
