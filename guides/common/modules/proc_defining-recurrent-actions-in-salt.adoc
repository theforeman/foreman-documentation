= Defining recurrent actions in Salt

[role="_abstract"]
You can define recurrent actions using the native Salt way.

.Procedure
. Define a recurrent action.
For example, you can schedule hourly `state.highstate` runs on individual Salt Minions by extending `/etc/salt/minion`:
[options="nowrap" subs="attributes"]
----
schedule:
  highstate:
    function: state.highstate
    minutes: 60
----
