= Subscriptions tracking tools

[role="_abstract"]
You can track subscriptions usage by combining the following tools.

System Purpose::
Defines non-technical system attributes for hosts to aid accurate inventory reporting:

* Service Level Agreement (SLA): Self-Support, Standard, Premium
* Usage type: Production, Development/Test
* Role: Server, Workstation

+
System purpose is configured by using the Subscription Manager tool.

Insights client::
Collects data from hosts and sends it to {Project}.
{Project} then forwards the data and host inventory to the {RHCloud} for analysis.
The Insights client is a tool installed on hosts and {ProjectServer}.

Subscriptions service::
Provides capacity data, calculates utilization, and reports subscription usage in connected scenarios.
ifdef::satellite[]
This service is hosted on the https://console.redhat.com[{RHCloud}] and is not available in disconnected environments.
endif::[]
ifndef::satellite[]
This service is hosted on the https://console.redhat.com[{RHCloud}].
endif::[]

{ProjectName}::
Provides reports on products installed on hosts.
You can download these reports in various formats, such as YAML, JSON, HTML, and CSV.
You can use these reports to calculate utilization locally.
