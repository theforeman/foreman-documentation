= Changing the download policy for a repository by using Hammer CLI

[role="_abstract"]
Update the download policy for a specific repository when you need immediate full downloads or on-demand lazy synchronization for that repository.

.Procedure
. List the repositories for an organization:
+
[subs="+quotes"]
----
$ hammer repository list \
--organization-label _My_Organization_Label_
----
. Change the download policy for a repository to `immediate` or `on_demand`:
+
[subs="+quotes"]
----
$ hammer repository update \
--download-policy _immediate_ \
--name "_My_Repository_" \
--organization-label _My_Organization_Label_ \
--product "_My_Product_"
----
