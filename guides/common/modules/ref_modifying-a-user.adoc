= Modifying a user

[role="_abstract"]
You can modify {Project} user accounts using PUT requests by specifying the user ID and the fields you want to update, such as login name or email address.

This example changes the given name and login for the `test_user` account.

Example API request::
+
[options="nowrap", subs="+quotes,attributes"]
----
$ curl \
--header "Accept:application/json" \
--header "Content-Type:application/json" \
--request PUT \
--user _My_User_Name_:__My_Password__ \
--data "{\"firstname\":\"_New Test Name_\",\"mail\":\"_test@example.com_\",\"login\":\"_new_test_user_\",\"password\":\"_password123_\",\"auth_source_id\":__1__}" \
https://_{foreman-example-com}_/api/users/_8_ \
| python3 -m json.tool
----
