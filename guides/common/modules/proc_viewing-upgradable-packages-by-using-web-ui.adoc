= Viewing upgradable packages by using {ProjectWebUI}

[role="_abstract"]
You can use {ProjectWebUI} to view upgradable packages.
This can help you identify and prioritize updating hosts.

.Procedure
. In the {ProjectWebUI}, navigate to *Hosts* > *All Hosts*.
. Select your host.
. On the *Content* tab, select the *Packages* tab.
. Select *Upgradable* from the *Status* list.
+
By default, {Project} shows all installed packages.

.Next steps
* You can use remote execution to upgrade packages on your host.
For more information, see xref:executing-a-remote-job-by-using-web-ui[].
