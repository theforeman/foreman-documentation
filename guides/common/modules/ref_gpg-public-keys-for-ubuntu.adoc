= GPG public keys for Ubuntu

[role="_abstract"]
You can download the GPG public keys from https://wiki.ubuntu.com/SecurityTeam/FAQ#GPG_Keys_used_by_Ubuntu[ubuntu.com].
Alternatively, download and extract the `ubuntu-keyring` package from https://packages.ubuntu.com/resolute/all/ubuntu-keyring[ubuntu.com].

[id="gpg-ubuntu-resolute"]
.Ubuntu 26.04 (Resolute)
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ gpg --keyserver keyserver.ubuntu.com --recv-key F6ECB3762474EDA9D21B7022871920D1991BC93C
$ gpg --armor --export F6ECB3762474EDA9D21B7022871920D1991BC93C > ubuntu_2604_gpg_keys.txt
----
