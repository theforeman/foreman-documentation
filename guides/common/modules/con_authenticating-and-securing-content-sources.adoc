= Authenticating and securing content sources

[role="_abstract"]
Remote content sources often use HTTPS and signed metadata or packages.
Import SSL certificates when upstream servers use custom or private certificate authorities so {Project} can connect securely.
Import GPG keys so {Project} and managed hosts can verify signatures on RPM or Deb content.
