= Logging in to {Project} configured with {keycloak-wildfly} as an authentication source

With {keycloak-wildfly} configured as an external authentication source for {Project}, users defined in a {keycloak-wildfly} realm can log in to {ProjectServer}.
The particular login methods available to users depend on how you configured integration between {keycloak-wildfly} and {Project}.
