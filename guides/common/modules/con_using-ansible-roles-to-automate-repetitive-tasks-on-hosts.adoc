= Using Ansible roles to automate repetitive tasks on hosts

[role="_abstract"]
Ansible roles provide a way to standardize and automate configuration and maintenance tasks on {Project} hosts.
After importing and assigning roles to hosts or host groups, you can execute them through the {ProjectWebUI} to apply consistent configurations and perform routine maintenance operations on your hosts.
