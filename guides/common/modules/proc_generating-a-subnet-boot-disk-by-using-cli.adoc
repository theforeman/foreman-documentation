= Generating a subnet boot disk by using Hammer CLI

[role="_abstract"]
Generate a boot ISO not associated with a specific host that chain loads the installer from TFTP {SmartProxy} for a subnet.

.Prerequisites
* Ensure that the `{ipxe-package}` package is installed on your {SmartProxy}.

.Procedure
* Generate the *Subnet image* boot disk:
+
[options="nowrap" subs="+quotes"]
----
$ hammer bootdisk subnet --subnet _My_Subnet_Name_
----
