= Install the Hammer CLI plugin for Ansible Director

[role="_abstract"]
Install the Hammer CLI plugin for Ansible Director plugin to control Ansible Director on {Project} by using Hammer CLI.

.Prerequisites
* Ensure that your host can access content for {SmartProxies}.

.Procedure
* Install the Hammer CLI plugin for Ansible Director:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --enable-foreman-cli-ansible-director
----

.Verification
. View the help output for Ansible Director:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
$ hammer
----
