= Enabling the Ansible plugin

[role="_abstract"]
Before you can use Ansible for configuration management in {Project}, enable Ansible integration in {Project} and on all {SmartProxyServers} that you want to use to run Ansible roles on hosts.

.Procedure
ifndef::satellite[]
. Enable the Ansible plugin on your {ProjectServer}:
+
[options="nowrap" subs="+quotes,attributes"]
----
# {foreman-installer} \
--enable-foreman-plugin-ansible \
--enable-foreman-proxy-plugin-ansible
----
. On all {SmartProxyServers} that you want to use to run Ansible roles on hosts, enable the Ansible plugin:
endif::[]
ifdef::satellite[]
* On all {SmartProxyServers} that you want to use to run Ansible roles on hosts, enable the Ansible plugin:
endif::[]
+
[options="nowrap" subs="+quotes,attributes"]
----
# {foreman-installer} --enable-foreman-proxy-plugin-ansible
----
