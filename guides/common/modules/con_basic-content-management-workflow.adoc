= Basic content management workflow

[role="_abstract"]
Synchronize and manage {EL} content to get started with content management with {Project} and Katello.
Content management enables version control and lifecycle management for content available to your hosts.

To manage {EL} content, start with the following steps:

* Create a {customrepo} to mirror content from CentOS Stream.
* Synchronize the content.
* Create an activation key.
* Register hosts to consume this content.
