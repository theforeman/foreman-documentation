= {Project} use cases

[role="_abstract"]
{ProjectName} is a centralized tool for provisioning,
ifdef::katello,orcharhino,satellite[]
content delivery,
endif::[]
remote management, and monitoring of multiple {client-os} deployments.
With {Project}, you can deploy, configure, and maintain your systems across physical, virtual, and cloud environments.

ifdef::satellite[]
{ProjectName} helps you provision hosts, manage patches and content, strengthen security, and understand infrastructure health across physical, virtual, and cloud environments.
endif::[]
ifdef::katello,orcharhino[]
ifndef::satellite[]
{ProjectName} helps you provision hosts, manage patches and content, and strengthen security across physical, virtual, and cloud environments.
endif::[]
endif::[]
ifdef::foreman-el,foreman-deb[]
ifndef::katello[]
{ProjectName} helps you provision hosts and strengthen security across physical, virtual, and cloud environments.
endif::[]
endif::[]
{ProjectName} focuses on the following use cases:

Provisioning::
Deploy consistently configured hosts on bare metal, virtual machines, and cloud instances from a unified interface.

ifdef::foreman-el,foreman-deb[]
Patch management::
Apply updates to hosts in a consistent, scalable, and repeatable manner.
endif::[]

ifdef::katello,orcharhino,satellite[]
Patch management and content distribution::
Import, organize, and distribute content to hosts.
Apply updates to hosts in a consistent, scalable, and repeatable manner.
endif::[]

Security management::
Reduce the effort required to secure host environments at scale.

Knowledge and analytics::
Identify potential issues and gain a comprehensive view of your infrastructure.

ifndef::satellite[]
Configuration management::
Apply configurations to hosts in a consistent and repeatable manner.
endif::[]

ifdef::satellite[]
In addition to these core capabilities, you can also use {Project} for subscription expenses management and to apply configurations to hosts in a consistent and repeatable manner.
endif::[]
