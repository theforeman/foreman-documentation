= Editing a virt-who configuration by using Hammer CLI

[role="_abstract"]
You can edit a virt-who configuration by using Hammer CLI.

.Procedure
. Update the virt-who configuration:
+
[options="nowrap" subs="+quotes,verbatim"]
----
$ hammer virt-who-config update \
--name _My_virt-who_Configuration_ \
--new-name _My_new_virt-who_Configuration_ \
--interval 1440
----
. Deploy the modified virt-who configuration.
