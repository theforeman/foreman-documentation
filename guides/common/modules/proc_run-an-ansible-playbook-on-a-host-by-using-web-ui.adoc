= Run an Ansible Playbook on a host by using {ProjectWebUI}

[role="_abstract"]
You can run a custom Ansible Playbook to configure your host through {Project} by using {ProjectWebUI}.

.Prerequisites
* Your {Project} account has a role that grants the `create_job_invocations`, `create_template_invocations`, `view_hosts`, `view_job_invocations`, `view_job_templates`, `view_remote_execution_features`, and `view_smart_proxies` permissions.

.Procedure
. In the {ProjectWebUI}, navigate to *Monitor* > *Jobs*.
. Click *Run Job*.
. From the *Job category* menu, select `Ansible Playbook`.
. From the *Job template* menu, select `Apply Ansible configuration`.
. Click *Next*.
. Select the hosts which you want to configure.
. In the *playbook* field, enter your custom Ansible Playbook.
. Follow the wizard to complete setting the remote job.
. Click *Submit* to run an Ansible Playbook to your selected hosts.

.Verification
. In the {ProjectWebUI}, navigate to *Monitor* > *Jobs*.
. In the search bar, enter `Apply Ansible configuration`.
. From the list of jobs, select the job that you previously started based on the search query and date.
. Verify that {Project} configured your selected hosts successfully.
