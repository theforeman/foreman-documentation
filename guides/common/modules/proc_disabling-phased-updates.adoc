= Disabling phased updates

[role="_abstract"]
Disable phased updates on hosts running Ubuntu to ensure {Project} controls the installation of packages and security updates.

.Procedure
* On your host running Ubuntu, ensure that `apt` is configured to always include all updates.
** You can disable phased updates during provisioning by adding the following section to your provisioning template:
+
[source, erb]
----
<% if @host.operatingsystem.name == 'Ubuntu' -%>
<%# Disable phased updates on Ubuntu -%>
cat << EOF >/etc/apt/apt.conf.d/98phased-updates
APT::Get::Always-Include-Phased-Updates "true";
EOF
<% end -%>
----
ifdef::orcharhino[]
+
This is already part of the *or_deb_client* snippet which is included in all Preseed and Preseed Autoinstall templates provided by {Team}.
endif::[]
** You can disable phased updates on hosts by using the *Disable Phased Updates - SSH Default* template.
