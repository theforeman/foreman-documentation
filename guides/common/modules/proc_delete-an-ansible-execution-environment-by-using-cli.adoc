= Delete an Ansible Execution Environment by using Hammer CLI

[role="_abstract"]
You can delete an Ansible Execution Environment from {Project} by using Hammer CLI.

.Prerequisites
* Your {Project} account has a role that grants the `destroy_ansible_execution_environments`, `view_ansible_content`, `view_ansible_execution_environments`, and `view_organizations` permissions.
* You have installed the Hammer CLI plugin for Ansible Director.
For more information, see xref:install-the-hammer-cli-plugin-for-ansible-director[].

.Procedure
* Delete your Ansible Execution Environment from {Project}:
+
----
$ hammer
----

.Verification
* Verify that {Project} deleted the Ansible Execution Environment successfully:
+
----
$ hammer
----
