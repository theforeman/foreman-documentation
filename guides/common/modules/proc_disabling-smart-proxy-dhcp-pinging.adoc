= Disabling {SmartProxy} DHCP pinging

[role="_abstract"]
A DHCP {SmartProxy} performs ICMP ping and TCP echo connection attempts to hosts in subnets by default.
The {SmartProxy} uses DHCP IPAM set to find out if an IP address considered for use is free.

.Procedure
* To prevent the DHCP {SmartProxy} from pinging hosts to check for available IP addresses, disable DHCP IP address pinging: 
+ 
[options="nowrap", subs="+quotes,attributes"] 
---- 
# {foreman-installer} --foreman-proxy-dhcp-ping-free-ip false 
----
