= Disabling {SmartProxy} DHCP pinging

[role="_abstract"]
You can disable DHCP IP address validation on {SmartProxy} to reduce network traffic or avoid triggering network security alerts.

By default, A DHCP {SmartProxy} performs ICMP ping and TCP echo connection attempts to validate whether an IP address is free.
This happens for hosts in subnets that have the IPAM configured as DHCP.

.Procedure
* On your {SmartProxy}, disable DHCP IP address pinging:
+
[options="nowrap", subs="+quotes,attributes"]
----
# {foreman-installer} --foreman-proxy-dhcp-ping-free-ip false
----
