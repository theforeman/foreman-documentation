= Disabling {SmartProxy} DHCP pinging

[role="_abstract"]
A DHCP {SmartProxy} performs ICMP ping and TCP echo connection attempts to hosts in subnets by default.
{SmartProxies} use IPAM configured for DHCP, to find out if an IP address is free.

.Procedure
* On your {SmartProxy} disable DHCP IP address pinging: 
+ 
[options="nowrap", subs="+quotes,attributes"] 
---- 
# {foreman-installer} --foreman-proxy-dhcp-ping-free-ip false 
----
