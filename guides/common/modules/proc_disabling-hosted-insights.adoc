= Disabling hosted {Insights}

[role="_abstract"]
If you do not want to use the hosted {Insights} services in the {RHCloud}, you can disable the integration between {Project} and the {RHCloud}.

.Prerequisites
* The Red{nbsp}Hat Cloud plugin is installed on {ProjectServer}.

.Procedure
. Uninstall the Red{nbsp}Hat Cloud plugin on {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --no-enable-foreman-plugin-rh-cloud
----
. Remove the Red{nbsp}Hat Cloud package and its dependencies from {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {project-package-remove} rubygem-foreman_rh_cloud
----
. Restart the {Project} services:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-maintain} service restart
----

. Verification
. In the {ProjectWebUI}, navigate to *Administer* > *About*.
. On the *Plugins* tab, verify that it does not list the `foreman_rh_cloud` plugin.
