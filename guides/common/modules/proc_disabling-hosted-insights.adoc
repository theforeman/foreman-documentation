= Disabling hosted {Insights}

[role="_abstract"]
If you do not want to or cannot use the hosted {Insights} services in the {RHCloud}, you can disable the integration between {Project} and the {RHCloud}.

.Prerequisites
* The RH Cloud plugin is installed on {ProjectServer}.

.Procedure
. Disable {Insights} on {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --disable-something
----
ifndef::satellite[]
. Optional: Uninstall the RH Cloud plugin on {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --no-enable-foreman-plugin-rh-cloud
----
