= Performing a backup without Pulp content

[role="_abstract"]
Run an offline backup that excludes the contents of the Pulp directory.
A backup without Pulp content is useful for debugging purposes and is only intended to provide access to configuration files without backing up the Pulp database.

[WARNING]
====
Do not use a backup without Pulp content to restore your {ProjectServer} or {SmartProxyServer} for production use cases.
====

.Prerequisites
* Your backup location must have sufficient available disk space to store the backup.
For more information, see xref:Estimating_the_Size_of_a_Backup_{context}[].

.Procedure
* Back up your {ProjectServer} without Pulp content:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-maintain} backup offline --skip-pulp-content _/var/backup_directory_
----
