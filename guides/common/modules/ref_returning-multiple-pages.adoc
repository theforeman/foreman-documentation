= Returning multiple pages

[role="_abstract"]
You can iterate through multiple pages of API results to retrieve complete datasets when the total number of items exceeds the per-page limit.

Bash script::
+
[source, bash, options="nowrap", subs="+quotes,attributes"]
----
$ for i in `seq 1 3`; do \
  curl \
  --request GET \
  --user _My_User_Name_:__My_Password__ \
  https://{foreman-example-com}/katello/api/content_views?per_page=5&amp;page=$i; \
done
----
