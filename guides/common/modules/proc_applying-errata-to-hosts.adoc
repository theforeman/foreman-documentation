= Applying errata to hosts

You can use {Project} apply errata to hosts.
ifndef::rest-api[]

.Procedure
. In the {ProjectWebUI}, navigate to *Hosts* > *All Hosts*.
. Select a host.
. On the *Content* tab, select the *Errata* tab.
. Select errata and click *Apply*.
endif::[]
