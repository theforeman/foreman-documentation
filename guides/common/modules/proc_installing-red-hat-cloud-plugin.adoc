= Installing Red Hat Cloud plugin

[role="_abstract"]
Install the Red{nbsp}Hat Cloud plugin on your {ProjectServer} to upload inventory reports to the {RHCloud}.
This step is required before you configure the cloud connector.

.Procedure
* Install the Red{nbsp}Hat Cloud plugin on your {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --enable-foreman-plugin-rh-cloud
----

.Verification
. In the {ProjectWebUI}, navigate to *Administer* > *About*.
. On the *Plugins* tab, verify that it lists the `foreman_rh_cloud` plugin.
