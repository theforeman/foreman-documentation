= {insights-iop} overview

[role="_abstract"]
When you use {insights-iop}, the Insights services run locally on your {ProjectServer} without connecting to the {RHCloud}.

If you use {insights-iop}:

* The {Insights} services run locally on your {ProjectServer}.
* {Project} only communicates with the locally running services of {insights-iop}.
* {Project} does not send any reports to the {RHCloud}.
* {Project} does not forward any data through the Insights client to the {RHCloud}.
* You cannot use any of the hosted services on the {RHCloud}.
