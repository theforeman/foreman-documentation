= Removing lifecycle environments from {ProjectServer} by using {ProjectWebUI}

[role="_abstract"]
Remove a lifecycle environment from {ProjectServer} when it is empty and no longer needed so your content paths stay accurate and easier to manage.

.Procedure
. In the {ProjectWebUI}, navigate to *Content* > *Lifecycle Environments*.
. Click the name of the lifecycle environment that you want to remove, and then click *Remove Environment*.
. Click *Remove* to remove the environment.
