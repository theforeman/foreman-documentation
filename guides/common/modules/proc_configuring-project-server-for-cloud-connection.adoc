= Configuring {ProjectServer} for cloud connection

[role="_abstract"]
You can configure {ProjectServer} to connect to the {RHCloud} by using remote execution.

.Prerequisites
* You have added the SSH key to authorized keys on your {ProjectServer}.
For more information, see the xref:adding-{project-context}-server-ssh-key-to-authorized-keys[].
* Your {Project} account has a role that grants the `execute_jobs_on_infrastructure_hosts` permission.
+
Note that this permission is not included in the default *Remote Execution User* role.

.Procedure
. In the {ProjectWebUI}, navigate to *{Insights}* > *Inventory Upload*.
. Click *Configure cloud connector*.
. Click *Confirm*.

.Verification
. In the {ProjectWebUI}, navigate to *Monitor* > *{Project} Tasks* > *Tasks*.
. Find the job with the name `Remote action: Configure cloud connector`.
. Verify that the job completed successfully.
