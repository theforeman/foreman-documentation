= Creating a location by using Hammer CLI

[role="_abstract"]
Use this procedure to create a location so that you can manage your hosts and resources by location by using Hammer CLI.

.Procedure
* Create a location:
+
[subs="+quotes"]
----
$ hammer location create \
--description "_My_Location_Description_" \
--name "_My_Location_" \
--parent-id "_My_Location_Parent_ID_"
----
