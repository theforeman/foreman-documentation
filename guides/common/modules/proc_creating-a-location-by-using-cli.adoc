= Creating a location by using Hammer CLI

[role="_abstract"]
Create a location by using Hammer CLI.
You can use the location to assign hosts and resources to the location.

.Procedure
* Create a location:
+
[subs="+quotes"]
----
$ hammer location create \
--description "_My_Location_Description_" \
--name "_My_Location_" \
--parent-id "_My_Location_Parent_ID_"
----

.Creating multiple locations
====
The following example Bash script creates three locations {endash} London, Munich, Boston {endash} and assigns them to the Example Organization.

[source,terminal,subs="+quotes"]
----
ORG="_Example Organization_"
LOCATIONS="_London Munich Boston_"

for LOC in ${LOCATIONS}
do
  hammer location create --name "${LOC}"
  hammer location add-organization --name "${LOC}" --organization "${ORG}"
done
----
