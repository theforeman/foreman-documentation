= Apply Ansible-based configuration to a host by using {ProjectWebUI}

[role="_abstract"]
You can apply Ansible-based configuration to your host by using {ProjectWebUI}.

.Prerequisites
* Your {Project} account has a role that grants the `create_ansible_lifecycle_environment_paths`, `edit_ansible_lifecycle_environment_paths`, `view_ansible_content`, `view_ansible_lifecycle_environment_paths`, `view_ansible_lifecycle_environments`, and `view_organizations` permissions.
* Your host has Ansible roles assigned to it.

.Procedure
. In the {ProjectWebUI}, navigate to *Monitor* > *Jobs*.
. Click *Run Job*.
. From the *Job category* menu, select `Ansible Playbook`.
. From the *Job template* menu, select `Apply Ansible configuration`.
. Click *Next*.
. Select the hosts which you want to configure.
. Follow the wizard to complete setting the remote job.
. Click *Submit* to apply Ansible-based configuration to your selected hosts.

.Verification
. In the {ProjectWebUI}, navigate to *Monitor* > *Jobs*.
. In the search bar, enter `Configure the host using Ansible content assigned to it`.
. From the list of jobs, select the job that you previously started based on the search query and date.
. Verify that {Project} configured your selected hosts successfully.
