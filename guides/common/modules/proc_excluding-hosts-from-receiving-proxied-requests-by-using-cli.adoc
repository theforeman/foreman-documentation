= Excluding hosts from receiving proxied requests by using Hammer CLI

[role="_abstract"]
If you use an HTTP Proxy for all {Project} HTTP or HTTPS requests, you can prevent certain hosts from communicating through the HTTP proxy.

.Procedure
* Enter the following command:
+
[options="nowrap" subs="+quotes"]
----
$ hammer settings set --name=http_proxy_except_list --value=[_hostname1.hostname2..._]
----
