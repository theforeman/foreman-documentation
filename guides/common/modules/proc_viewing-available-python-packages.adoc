= Viewing available Python packages

[role="_abstract"]
Review Python packages in the Library lifecycle environment of {Project} to verify that the packages in your organization are available and up to date.

.Prerequisites
* Python packages are synchronized or uploaded to your organization in {Project}.

.Procedure
. In the {ProjectWebUI}, navigate to *Content* > *Content Types* > *Other Content Types*.
. In the *Type* menu, select *Python Packages*.
