== Activation keys

[cols="3a,7a",options="header",]
|====
|Subcommand |Description and tasks
|`*activation-key*`
xref:common/modules/ref_general-information.adoc#general-information[_organization-specific_]
|Create an activation key:
[subs="+quotes"]
----
hammer activation-key create \
--name _ak_name_ \
--content-view _cv_n_ \
--lifecycle-environment _lc_name_
----
Add a subscription to the activation key:
[subs="+quotes"]
----
hammer activation-key add-subscription \
--id _ak_ID_ \
--subscription-id _sub_ID_
----
|====
