= Listing hosts

This example returns a list of registered hosts.

[id="api-listing-hosts"]
.API request
[options="nowrap", subs="+quotes,attributes"]
----
$ curl \
--request GET \
--user _My_User_Name_:__My_Password__ \
https://_{foreman-example-com}_/api/v2/hosts \
| python3 -m json.tool
----
