= Accessing Grafana remotely on Linux or macOS

By default, Grafana runs on `http://localhost:3000` on the {ProjectServer} and is only accessible locally.
To access Grafana from a remote workstation, create an SSH tunnel that forwards your local port 3000 to the {ProjectServer} port 3000.

. On your local workstation, create an SSH tunnel:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
# ssh -L 3000:localhost:3000 root@_{foreman-example-com}_ -N
----
+
The `-L` flag creates the port forwarding tunnel, and `-N` prevents executing any remote commands.
Keep this terminal session open while accessing Grafana.
+
To run the tunnel in the background, add the `-f` flag: `ssh -f -L 3000:localhost:3000 root@_{foreman-example-com}_ -N`.
To close a background tunnel, find its PID with `ps aux | grep "3000:localhost[:]3000"` and then run `kill _PID_`.

. Open your web browser and navigate to: http://localhost:3000
