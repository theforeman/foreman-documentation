= Creating a role by using {ProjectWebUI}

[role="_abstract"]
Create a role by using the {ProjectWebUI}.
You can add permissions to the role to control which actions users can perform.

.Procedure
. In the {ProjectWebUI}, navigate to *Administer* > *Roles*.
. Click *Create Role*.
. Provide a *Name* for the role.
. Click *Submit* to save your new role.

.Next steps
* Add permissions to the role.
To serve its purpose, a role must contain permissions.
