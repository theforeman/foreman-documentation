= Synchronizing repositories by using Hammer CLI

[role="_abstract"]
Run a on-demand sync for a product or individual repository to download the latest upstream content into {Project}.

.Procedure
* Synchronize an entire product:
+
[options="nowrap" subs="+quotes"]
----
$ hammer product synchronize \
--name "_My_Product_" \
--organization "_My_Organization_"
----
* Synchronize an individual repository:
+
[options="nowrap" subs="+quotes"]
----
$ hammer repository synchronize \
--name "_My_Repository_" \
--organization "_My_Organization_" \
--product "_My_Product_"
----
