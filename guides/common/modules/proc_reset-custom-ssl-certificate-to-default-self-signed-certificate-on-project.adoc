= Reset {customssl} certificate to default self-signed certificate on {ProjectServer}

.Procedure
* Reset the {customssl} certificate to default self-signed certificate:
+
[options="nowrap" subs="+quotes,attributes"]
----
# {foreman-installer} --certs-reset
----

.Verification
Ensure the following parameters in `/etc/foreman-installer/scenarios.d/satellite-answers.yaml` have no values:

* server_cert:
* server_key:
* server_cert_req:
* server_ca_cert:
