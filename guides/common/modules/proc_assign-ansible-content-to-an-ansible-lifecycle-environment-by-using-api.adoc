= Assign Ansible content to an Ansible lifecycle environment by using {Project} API

[role="_abstract"]
You can assign Ansible content to an Ansible lifecycle environment by using {Project} API.
Ansible content consists of imported Ansible collections and roles in specific versions.

.Prerequisites
* Your {Project} account has a role that grants the `create_ansible_lifecycle_environment_paths`, `edit_ansible_lifecycle_environment_paths`, `view_ansible_content`, `view_ansible_lifecycle_environment_paths`, `view_ansible_lifecycle_environments`, and `view_organizations` permissions.
* Your {Project} contains an Ansible lifecycle environment path.
* Your {Project} contains Ansible content.

.Procedure
. Optional: Get a list of all Ansible lifecycle environments within your Ansible lifecycle environment path:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ curl \
--header "Accept: application/json" \
--user "_My_User_Name_:__My_Password__" \
https://{foreman-example-com}/api/v2/ansible_director/lifecycle_environments/_My_Ansible_Lifecycle_Environment_Path_ID_/ \
| jq
----
. Create a JSON file, for example, `data.json` to define Ansible content for your Ansible lifecycle environment:
+
[source, json, options="nowrap", subs="verbatim,quotes,attributes"]
----
{
  "content_assignments": [
    {
      "id": "_My_Ansible_Content_ID_",
      "version": "_My_Ansible_Content_Version_"
    }
  ],
  "organization_id": "_My_Organization_ID_"
}
----
. Assign Ansible content to your Ansible lifecycle environment:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ curl \
--data @data.json \
--header "Accept: application/json" \
--header "Content-Type: application/json" \
--request PATCH \
--user "_My_User_Name_:__My_Password__" \
https://{foreman-example-com}/api/v2/ansible_director/lifecycle_environments/_My_Ansible_Lifecycle_Environment_ID_/ \
| jq
----

.Verification
* Query the {Project} API to verify that {Project} assigned the Ansible content to your Ansible lifecycle environment successfully:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ curl \
--header "Accept: application/json" \
--user "_My_User_Name_:__My_Password__" \
https://{foreman-example-com}/api/v2/ansible_director/lifecycle_environments/_My_Ansible_Lifecycle_Environment_ID_/ \
| jq ".content"
----
