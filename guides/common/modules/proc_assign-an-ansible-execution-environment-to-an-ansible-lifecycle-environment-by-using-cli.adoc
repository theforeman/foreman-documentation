= Assign an Ansible Execution Environment to an Ansible lifecycle environment by using Hammer CLI

[role="_abstract"]
You can assign an Ansible Execution Environment to an Ansible lifecycle environment by using Hammer CLI.
Ansible Execution Environments provide the environment to run Ansible to configure hosts.

.Prerequisites
* Your {Project} account has a role that grants the `create_ansible_lifecycle_environment_paths`, `edit_ansible_lifecycle_environment_paths`, `view_ansible_content`, `view_ansible_lifecycle_environment_paths`, `view_ansible_lifecycle_environments`, and `view_organizations` permissions.
* Your {Project} contains an Ansible lifecycle environment path.
* Your {Project} contains an Ansible Execution Environment.
* You have installed the Hammer CLI plugin for Ansible Director.
For more information, see xref:install-the-hammer-cli-plugin-for-ansible-director[].

.Procedure
. Optional: List all Ansible Execution Environments in {Project}:
+
----
$ hammer
----
. Assign an Ansible Execution Environment to your Ansible lifecycle environment:
+
----
$ hammer
----

.Verification
* Verify that {Project} assigned the Ansible Execution Environment to your Ansible lifecycle environment successfully:
+
----
$ hammer
----
