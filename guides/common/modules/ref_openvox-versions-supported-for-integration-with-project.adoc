= OpenVox versions supported for integration with {Project}

[role="_abstract"]
Before you begin with the Puppet integration, review which versions of the OpenVox server and OpenVox agent are supported for integration with {Project}.

The following version of the OpenVox server is supported on {ProjectServer} and {SmartProxyServers}:

* OpenVox 8

[IMPORTANT]
====
The Puppet modules you are using must be compatible with the supported version of OpenVox server.
====

The following version of OpenVox agent is supported on {Project} hosts:

* OpenVox agent 8

Additionally, Puppet agent 7 remains compatible with OpenVox in {Project} as a legacy alternative.
If your {Project} hosts run Puppet agent 7, they will integrate with the OpenVox implementation in {Project}.
However, Puppet agent 7 is end of life and it is recommended to migrate {Project} hosts to OpenVox agent 8.#Okay, but how?#
