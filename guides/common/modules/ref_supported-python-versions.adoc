= Supported Python versions

[role="_abstract"]
The Python version used to run Ansible to configure hosts is determined by the base container image.
It must be compatible with the `ansible-core` releases on pypi.org.

.Ansible version per Python version
[cols="50%,50%",options="header"]
|====
| Ansible version | Python versions
| 2.20 | 3.12, 3.13, 3.14
| 2.19 | 3.11, 3.12, 3.13
| 2.18 | 3.11, 3.12, 3.13
| 2.17 | 3.10, 3.11, 3.12
| 2.16 | 3.10, 3.11, 3.12
|====

.Container Image and default Python version
[cols="50%,50%",options="header"]
|====
| Operating system | Default Python version
| AlmaLinux 10.1 | 3.12
| Debian 13 | 3.13
| Fedora 44 | 3.14
| Ubuntu 26.04 | 3.14
|====
