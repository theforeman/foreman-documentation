= Synchronizing repositories by using {ProjectWebUI}

[role="_abstract"]
Synchronize repositories using {ProjectWebUI} to update content and make the latest packages available to your hosts.
On-demand synchronization enables immediate content updates outside of scheduled sync plans.

.Procedure
. In the {ProjectWebUI}, navigate to *Content* > *Products*.
. Select the product that contains the repositories that you want to synchronize.
. Select the repositories that you want to synchronize and click *Sync Now*.

.Verification
. In the {ProjectWebUI}, navigate to *Content* > *Sync Status*.
. Expand the corresponding product or repository tree.
. Verify that the synchronization is complete.
