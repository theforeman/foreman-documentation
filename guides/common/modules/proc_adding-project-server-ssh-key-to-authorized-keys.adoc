= Adding {ProjectServer} SSH key to authorized keys

[role="_abstract"]
If you want to run remote execution jobs on {ProjectServer} itself, add the SSH key to authorized keys on your {ProjectServer}.

.Prerequisites
* You can access your {ProjectServer} using SSH as root.

.Procedure
* Add the SSH key of the `foreman-proxy` user to the `authorized_keys` file:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# cat /var/lib/foreman-proxy/ssh/id_rsa_foreman_proxy.pub >>/root/.ssh/authorized_keys
----
