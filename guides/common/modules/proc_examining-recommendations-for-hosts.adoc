= Examining recommendations for hosts

[role="_abstract"]
You can view the recommendations for RHEL hosts in the {ProjectWebUI}.

.Procedure
. In the {ProjectWebUI}, navigate to *{Insights}* > *Recommendations*.
. From the menu next to the *Remediate* button, select *View in {Insights}* to navigate to the general recommendations page in {RHCloud}.
