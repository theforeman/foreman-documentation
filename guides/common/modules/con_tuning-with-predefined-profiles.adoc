= Tuning {ProjectServer} with predefined profiles

[role="_abstract"]
If your {Project} deployment includes a large number of hosts, you can use predefined tuning profiles to improve performance of {Project}.
Choosing the right tuning profile helps you set up {ProjectServer} with the optimal number of CPU cores and memory allocation for your deployment.
