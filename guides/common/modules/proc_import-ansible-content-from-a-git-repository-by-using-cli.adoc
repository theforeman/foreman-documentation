= Import Ansible content from a Git repository by using Hammer CLI

[role="_abstract"]
You can import Ansible content from a Git repository into {Project} by using Hammer CLI.
Select *Git repository* if you want to import Ansible content from a remote Git repository.

.Prerequisites
* Your {Project} account has a role that grants the `create_ansible_content`, `view_ansible_content`, and `view_organizations` permissions.
* You have installed the Hammer CLI plugin for Ansible Director.
For more information, see xref:install-the-hammer-cli-plugin-for-ansible-director[].

.Procedure
* Import Ansible content from a Git repository:
+
----
$ hammer
----

.Verification
* Verify that {Project} imported the Ansible content successfully:
+
----
$ hammer
----
