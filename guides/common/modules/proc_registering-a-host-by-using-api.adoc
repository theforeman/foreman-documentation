= Registering a host by using {Project} API

[role="_abstract"]
You can register a host by using {Project} API with registration templates.
This also enables you to set up various integration features and host tools during the registration process.

.Prerequisites
include::snip_registering-a-host-prerequisites.adoc[]

.Procedure
* Use the POST `/api/registration_commands` resource.
