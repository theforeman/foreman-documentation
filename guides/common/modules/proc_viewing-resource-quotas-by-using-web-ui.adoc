= Viewing resource quotas by using {ProjectWebUI}

[role="_abstract"]
You can use {ProjectWebUI} to view the resource quota of hosts.

.Procedure
. In the {ProjectWebUI}, navigate to *Hosts* > *All Hosts*.
. In the search bar, filter your hosts.
For example, `resource_quota = Unassigned`.
+
For more information, see xref:Searching_and_Bookmarking_{context}[].
