= Repository definitions for Ubuntu under Expanded Security Maintenance

[role="_abstract"]
Ubuntu Expanded Security Maintenance (ESM) provides extended security updates for selected releases through authenticated Deb repositories.
Use the values from your subscribed Ubuntu hosts when creating repositories in {Project}; see xref:Synchronizing-Ubuntu-Expanded-Security-Maintenance-content_{context}[].

[IMPORTANT]
====
Providing https://ubuntu.com/security/esm[Ubuntu ESM] repositories requires a subscription from Canonical.
Ensure that you have valid licenses for all Canonical products you mirror or distribute.
====

Repository type:: Deb

Authentication:: Required.
Set *Upstream Username* and *Upstream Password* from the credentials that Ubuntu Advantage writes to `/etc/apt/auth.conf.d/90ubuntu-advantage` on an attached host.

Obtaining definitions:: On a host running the target Ubuntu release, with `ubuntu-advantage-tools` installed and the host attached to your subscription (`ua attach`), read *Upstream URL*, *Releases*, *Components*, and *Architectures* from `/etc/apt/sources.list.d/ubuntu-esm-infra.list`.

.Example ESM Infra definitions (Ubuntu 14.04 LTS and 16.04 LTS)
[width="100%",cols="20%,34%,22%,12%,12%",options="header"]
|===
| Repository name | Upstream URL | Releases/Distributions | Components | Architectures
| `Ubuntu 14.04 ESM Infra` | `https://esm.ubuntu.com/infra/ubuntu` | `trusty-infra-security` | `main` | `amd64`
| `Ubuntu 16.04 ESM Infra` | `https://esm.ubuntu.com/infra/ubuntu` | `xenial-infra-security` | `main` | `amd64`
|===

Your subscription can expose additional ESM components or suites; use the values from your host's `*.list` files if they differ.

GPG public key:: Import the key material supplied for ESM (for example the `ubuntu-advantage-esm-infra-trusty.gpg` package on Ubuntu 14.04).
One signing key is identified by fingerprint `4067E40313CB4B13`; export it in ASCII-armored form for a content credential, for example:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# gpg --import ubuntu-advantage-esm-infra-trusty.gpg
# gpg --export --armor 4067E40313CB4B13
----
