= Viewing upgradable packages by using Hammer CLI

[role="_abstract"]
You can use Hammer CLI to view upgradable packages.
This can help you identify and prioritize updating hosts.

.Procedure
. Optional: Get the ID of your host:
+
[options="nowrap",subs="+quotes,verbatim,attributes"]
----
$ hammer host list --fields Id,Name
----
. Get a list of upgradable packages of your host:
+
[options="nowrap",subs="+quotes,verbatim,attributes"]
----
$ hammer host package list --status upgradable --host-id _My_Host_ID_
----

.Next steps
* You can use Hammer CLI to upgrade packages on your host:
+
[options="nowrap",subs="+quotes,verbatim,attributes"]
----
$ hammer job-invocation create --feature katello_package_update --search-query "id = _My_Host_ID_" --inputs "package=_My_Package_Name_"
----
+
For more information, see {HammerDocURL}[{HammerDocTitle}].
