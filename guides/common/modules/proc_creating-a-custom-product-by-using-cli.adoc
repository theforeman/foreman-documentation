= Creating a {customproduct} by using Hammer CLI

[role="_abstract"]
Create a {customproduct} with Hammer CLI to group related repositories before you attach mirrors and publish them to hosts.

.Procedure
* Create a product:
+
[options="nowrap" subs="+quotes"]
----
$ hammer product create \
--name "_My_Name_" \
--organization-id _My_Organization_ID_
----
