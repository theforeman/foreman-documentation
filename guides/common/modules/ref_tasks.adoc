[id='tasks']
== Tasks

[role="_abstract"]
Use Hammer CLI commands to list tasks and monitor their progress in {ProjectName}.
Track running operations and troubleshoot issues by using these commands.

[cols="3a,7a",options="header",]
|====
|Subcommand |Description and tasks
|`*task*` |List all tasks:
[subs="+quotes"]
----
hammer task list
----
Monitor progress of a running task:
[subs="+quotes"]
----
hammer task progress \
--id _task_ID_
----
|====
