= Enabling and disabling repositories on hosts

[role="_abstract"]
You can enable or disable repositories on a host from the {ProjectWebUI} to control which package sources the host can use.
This affects which packages are available for installation or upgrade on the host.

.Procedure
. In the {ProjectWebUI}, navigate to *Hosts* > *All Hosts*,
. Select a host.
. On the *Content* tab, click the *Repository sets* tab.
. Open the options menu and select *Override to disabled* or *Override to enabled* to disable or enable repositories on hosts.
