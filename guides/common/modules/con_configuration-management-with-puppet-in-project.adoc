= Configuration management with Puppet in {Project}

[role="_abstract"]
Puppet is a configuration management tool to automate host configurations across your infrastructure.
You can use Puppet to consistently manage configurations of hosts registered to {Project}.
{Project} uses OpenVox server as the Puppet-compatible central server for the integration.
