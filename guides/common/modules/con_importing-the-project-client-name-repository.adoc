= Importing the {project-client-name} repository

[role="_abstract"]
The {project-client-name} repository provides essential client integration tools, such as `katello-host-tools`, that enable communication between hosts and {Project}.
To use these tools, you must enable and synchronize the repository on {ProjectServer}, then enable it on your hosts. 
These tools are required for features like remote command execution, package management, and host reporting.
