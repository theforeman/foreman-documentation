= Enabling the installer-managed TFTP service

[role="_abstract"]
Instead of using a TFTP server, you can enable the installer-managed TFTP service to perform unattended installations with low maintenance.
With installer-managed TFTP, {Project} fully manages the service and its files.

.Procedure
* Configure your {ProjectServer} or {SmartProxyServer} as the TFTP server:
+
[options="nowrap",subs="+quotes,attributes"]
----
# {foreman-installer} \
--foreman-proxy-tftp true \
--foreman-proxy-tftp-managed true
----
