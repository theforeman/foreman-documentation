= Creating a role by using Hammer CLI

[role="_abstract"]
Create a role by using Hammer CLI.
You can add permissions to the role to control which actions users can perform.

.Procedure
* Create a role:
+
[options="nowrap", subs="+quotes,attributes"]
----
$ hammer role create --name _My_Role_Name_
----

.Next steps
* Add permissions to the role.
To serve its purpose, a role must contain permissions.
