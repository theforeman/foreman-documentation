= Repository definitions for Ubuntu under ESM

[role="_abstract"]
Ubuntu Expanded Security Maintenance (ESM) exposes authenticated Deb repositories that extend security support for selected releases.
You reuse URL, distribution, component, and key values from an attached ESM host when you create matching repositories in {Project}.

[IMPORTANT]
====
Providing Ubuntu ESM repositories requires a subscription from Canonical.
Ensure that you have valid licenses for all Canonical products you mirror or distribute.
Using insufficient, invalid, or otherwise inadequate licenses might violate your terms with Canonical.
====

On a dedicated host running the target Ubuntu ESM release, attach your subscription by running `ua attach _My_Token_from_Canonical_` and install the `ubuntu-advantage-tools` package.

GPG public key::
Use the key material supplied for ESM, for example from the `ubuntu-advantage-esm-infra-trusty.gpg` file on Ubuntu 14.04.
Convert the key to ASCII-armored form for a content credential, for example:
+
[options="nowrap" subs="+quotes,verbatim,attributes"]
----
$ gpg --import ubuntu-advantage-esm-infra-trusty.gpg
$ gpg --export --armor 4067E40313CB4B13
----
+
Import the GPG key to {Project}.
For more information, see xref:importing-a-custom-gpg-key-by-using-web-ui[] or xref:importing-a-custom-gpg-key-by-using-cli[].

Authentication::
Extract user name and password from the credentials that Ubuntu Advantage writes to `/etc/apt/auth.conf.d/90ubuntu-advantage` on your dedicated Ubuntu host.
Use the values for *Upstream Username* and *Upstream Password* when adding the ESM repository.

Repository definition::
Extract *Upstream URL*, *Releases*, *Components*, and *Architectures* from `/etc/apt/sources.list.d/ubuntu-esm-infra.list`.
+
.Example ESM Infra definitions (Ubuntu 14.04 LTS and 16.04 LTS)
[width="100%",cols="20%,34%,22%,12%,12%",options="header"]
|===
| Repository name | Upstream URL | Releases/Distributions | Components | Architectures
| `Ubuntu 14.04 ESM Infra` | `https://esm.ubuntu.com/infra/ubuntu` | `trusty-infra-security` | `main` | `amd64`
| `Ubuntu 16.04 ESM Infra` | `https://esm.ubuntu.com/infra/ubuntu` | `xenial-infra-security` | `main` | `amd64`
|===
+
Use the values when adding the ESM repository.
