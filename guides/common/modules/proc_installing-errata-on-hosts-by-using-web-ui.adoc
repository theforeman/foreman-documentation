= Installing errata on hosts by using {ProjectWebUI}

[role="_abstract"]
You can use the {ProjectWebUI} to review and install installable errata on hosts so that security and bugfix updates are installed on selected hosts.

.Prerequisites
* Requirements for errata installation on hosts are met.
For more information, see xref:prerequisites-for-installing-errata-on-hosts[].

.Procedure
. In the {ProjectWebUI}, navigate to *Content* > *Content Types* > *Errata*.
. Click the name of an erratum you want to install.
. On the to *Content Hosts* tab, select all hosts that you want to install errata on.
. Click *Apply to Hosts* and click *Confirm*.

.Verification
. In the {ProjectWebUI}, navigate to *Monitor* > *Jobs*.
. Select the most recent `Install errata` job.
. Verify that the remote execution to install errata on all selected hosts succeeded.
