= Exporting templates by using {Project} API

[role="_abstract"]
You can export templates to an existing repository by using {Project} API.

.Procedure
. Send a `POST` request to `api/v2/templates/export`:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ curl \
--header "Accept:application/json" \
--header "Content-Type:application/json" \
--request POST \
--user "__My_User_Name__:__My_Password__" \
https://_{foreman-example-com}_/api/v2/templates/export
----
+
If the export is successful, you receive `{"message":"Success"}`.
+
[NOTE]
====
You can override default API settings by specifying them in the request with the `-d` parameter.
The following example exports templates to the `git.example.com/templates` repository:

[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ curl \
--data "{\"repo\":\"git.example.com/templates\"}" \
--header "Accept:application/json" \
--header "Content-Type:application/json" \
--request POST \
--user "__My_User_Name__:__My_Password__" \
https://_{foreman-example-com}_/api/v2/templates/export
----
====
