= Delete an Ansible lifecycle environment path by using {Project} API

[role="_abstract"]
You can delete an Ansible lifecycle environment path in {Project} by using {ProjectWebUI}.

.Prerequisites
* Your {Project} account has a role that grants the `destroy_ansible_lifecycle_environment_paths`, `view_ansible_content`, `view_ansible_lifecycle_environment_paths`, `view_ansible_lifecycle_environments`, and `view_organizations` permissions.

.Procedure
. Optional: List all Ansible lifecycle environment paths in {Project}:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ curl \
--header "Accept: application/json" \
--user "_My_User_Name_:__My_Password__" \
https://{foreman-example-com}/api/v2/ansible_director/lifecycle_environments/paths/ \
| jq ".results | .[]"
----
. Delete your Ansible lifecycle environment path from {Project}:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ curl \
--header "Accept: application/json" \
--request DELETE \
--user "_My_User_Name_:__My_Password__" \
https://{foreman-example-com}/api/v2/ansible_director/lifecycle_environments/paths/_My_Ansible_Lifecycle_Environment_Path_ID_/ \
| jq
----

.Verification
* Verify that {Project} does not contain your previously deleted Ansible lifecycle environment path:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ curl \
--header "Accept: application/json" \
--user "_My_User_Name_:__My_Password__" \
https://{foreman-example-com}/api/v2/ansible_director/lifecycle_environments/paths/ \
| jq ".results | .[]"
----
