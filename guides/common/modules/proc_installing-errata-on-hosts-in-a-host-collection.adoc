= Installing errata on hosts in a host collection

[role="_abstract"]
You can install errata on all hosts in a host collection in one action using remote execution.

.Prerequisites
* Requirements for errata installation on hosts are met.
For more information, see xref:prerequisites-for-installing-errata-on-hosts[].

.Procedure
. In the {ProjectWebUI}, navigate to *Hosts* > *Host Collections*.
. Select the host collection where the errata should be installed.
. On the *Collection Actions* tab, click *Errata Installation*.
. Select the errata you want to install on the hosts in the collection and click *Install Selected* to use the default method.
Alternatively, select the drop-down icon to the right of the button to select a method to use.
Selecting the *via remote execution {endash} customize first* menu entry takes you to the *Job invocation* page where you can customize the action.
