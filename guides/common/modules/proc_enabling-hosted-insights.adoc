[role="_abstract"]
You can enable hosted {Insights}.

.Prerequisites

.Procedure
ifndef::satellite[]
. Enable the RH Cloud plugin on {ProjectServer}:
+

----
# {foreman-installer} --enable-foreman-plugin-rh-cloud
----
endif::[]
. Add the SSH key of your {ProjectServer} to authorized keys
. Configure {ProjectServer} for cloud connection
