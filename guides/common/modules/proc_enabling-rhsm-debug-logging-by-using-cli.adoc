= Enabling RHSM debug logging by using Hammer CLI

[role="_abstract"]
You can enable Red{nbsp}Hat Subscription Manager (RHSM) debug logging for the `/var/log/rhsm/rhsm.log` file by using Hammer CLI.

After you resolve the problem, you must disable debug logging and redeploy the virt-who configuration.

.Procedure
. Set the logging level to `debug`:
+
[options="nowrap" subs="+quotes"]
----
$ hammer virt-who-config update \
--debug true
----
. Redeploy the virt-who configuration on the target host.
