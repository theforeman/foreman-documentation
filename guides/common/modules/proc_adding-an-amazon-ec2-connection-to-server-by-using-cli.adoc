= Adding an Amazon EC2 connection to {ProjectServer} by using Hammer CLI

[role="_abstract"]
Use this procedure to add the Amazon EC2 connection to {ProjectServer}.

.Prerequisites
* An AWS EC2 user performing this procedure needs the `AmazonEC2FullAccess` permissions.
You can attach these permissions from AWS.
* Amazon Web Services uses time settings as part of the authentication process.
Ensure that {ProjectServer} has an NTP service configured, for example, `ntpd` or `chronyd`.

.Procedure
* Connect your {ProjectServer} to Amazon EC2:
+
[options="nowrap" subs="+quotes"]
----
$ hammer compute-resource create \
--description "Amazon EC2 Public Cloud` \
--locations "_My_Location_" \
--name "_My_EC2_Compute_Resource_" \
--organizations "_My_Organization_" \
--password "_My_Secret_Key_" \
--provider "EC2" \
--region "_My_Region_" \
--user "_My_User_Name_"
----
