= Adding a KVM connection to {ProjectServer} by using Hammer CLI

[role="_abstract"]
To provision and manage virtual machines on KVM (libvirt), add KVM as a compute resource to {Project}.

.Procedure
* Create a compute resource:
+
[options="nowrap" subs="+quotes"]
----
$ hammer compute-resource create --name "_My_KVM_Server_" \
--provider "Libvirt" --description "KVM server at _kvm.example.com_" \
--url "qemu+ssh://root@_kvm.example.com/system_" --locations "New York" \
--organizations "_My_Organization_"
----
