= Disabling {insights-iop}

[role="_abstract"]
If you do not want to use {insights-iop}, you can disable it.

.Prerequisites

.Procedure
* On {ProjectServer}, disable {insights-iop}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --iop-ensure absent
----
+
This command removes the container images from {ProjectServer}.
