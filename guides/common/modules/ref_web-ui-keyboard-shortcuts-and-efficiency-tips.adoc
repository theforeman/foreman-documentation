= Keyboard shortcuts and efficiency tips

[role="_abstract"]
Use these keyboard shortcuts and techniques to navigate and search more efficiently in the {ProjectWebUI}.

*Keyboard shortcuts*

* To focus the vertical navigation search bar, press `Ctrl + Shift + F`.
* To focus the page search bar, press `/`.

*Bookmarking searches*

Save frequently used searches as bookmarks:

. Enter your search query in the search field.
. Click the arrow next to the *Search* button and select *Bookmark this search*.
. Choose whether to make the bookmark public (visible to all users) or private.

Bookmarks are page-specific and appear in the search dropdown on the page where you created them.
To manage all bookmarks, navigate to *Administer* > *Bookmarks*.

*Search autocomplete*

Click the search field to see available search parameters for the current page.
As you type, autocomplete suggests field names, operators, and valid values.

*Combining searches*

Use `AND`, `OR`, and `NOT` operators to build complex queries:

[options="nowrap", subs="+quotes,attributes"]
----
environment = production AND hostgroup ~ webservers
errata_status = errata_needed OR errata_status = security_needed
status.enabled = true NOT status.failed
----

Alternative operator syntax: `&`, `|`, and `!` work the same as their word equivalents.
A space between terms is interpreted as `AND`.
