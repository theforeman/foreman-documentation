= Configuration management with Ansible in {Project}

[role="_abstract"]
By using Ansible as a configuration management tool in {Project}, you can define the desired state of your hosts and maintain it over time.
You import Ansible roles, assign them to hosts or host groups, and run them from {Project} to manage host configurations consistently.
