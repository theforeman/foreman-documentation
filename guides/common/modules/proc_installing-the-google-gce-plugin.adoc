= Installing the Google GCE plugin

[role="_abstract"]
Install the Google Compute Engine (GCE) plugin to attach a GCE compute resource provider to {Project}.
This allows you to manage and deploy hosts to GCE.

.Procedure
* Install the Google GCE plugin on your {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} \
--enable-foreman-cli-google \
--enable-foreman-plugin-google
----

.Verification
. In the {ProjectWebUI}, navigate to *Administer* > *About*.
. Select the *Compute Resources* tab to verify the installation of the Google GCE plugin.
