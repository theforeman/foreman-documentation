= Configure the Ansible Director plugin by using {ProjectWebUI}

[role="_abstract"]
Configure the Ansible Director plugin to adjust to your {Project} environment.

.Procedure
. In the {ProjectWebUI}, navigate to *Administer* > *Settings*.
. Select the *Ansible Director* tab.
. Adjust the plugin to your environment.
For example, you can configure a different default version of `ansible-core`.

.Verification
. In the {ProjectWebUI}, navigate to *Administer* > *Settings*.
. Reload the *Ansible Director* tab and verify that {Project} updated the plugin settings.
