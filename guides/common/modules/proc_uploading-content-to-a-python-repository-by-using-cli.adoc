= Uploading content to a Python repository by using Hammer CLI

[role="_abstract"]
You can upload files from your local machine to a Python repository in {Project} when you need to distribute Python content that you cannot synchronize.

.Prerequisites
* You have a `.whl` package or `.tar.gz` archive that contains Python packages.

.Procedure
* Upload files to your Python repository:
+
[options="nowrap" subs="+quotes"]
----
$ hammer repository upload-content \
--id _My_Repository_ID_ \
--organization-id _My_Organization_ID_ \
--path _Path_To_My_File_
----
+
The `--path` option can indicate a file, a directory of files, or a glob expression of files.
Globs must be escaped by single or double quotes.

.Verification
* Visit the *Published At* URL to see the file.
