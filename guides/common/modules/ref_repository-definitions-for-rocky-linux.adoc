= Repository definitions for Rocky Linux

[role="_abstract"]
Use these example repository names and upstream URLs when you synchronize Rocky Linux BaseOS, AppStream, extras, and the optional {project-client-name} content.

[id="rocky-linux-10"]
.Rocky Linux 10
[width="100%",cols="30%,70%",options="header"]
|===
| Repository name | Upstream URL
| `Rocky Linux 10 BaseOS` | `https://dl.rockylinux.org/pub/rocky/10/BaseOS/x86_64/os/`
| `Rocky Linux 10 AppStream` | `https://dl.rockylinux.org/pub/rocky/10/AppStream/x86_64/os/`
| `Rocky Linux 10 extras` | `https://dl.rockylinux.org/pub/rocky/10/extras/x86_64/os/`
ifdef::katello[]
| `{project-client-name}` | `{project-client-url}/el10/x86_64/`
endif::[]
ifdef::orcharhino[]
| `{project-client-name}` | see {atix-kb-clients} in the _ATIX Service Portal_
endif::[]
|===

GPG key: https://dl.rockylinux.org/pub/rocky/RPM-GPG-KEY-rockyofficial
