= Importing Ansible roles and variables

[role="_abstract"]
You can import Ansible roles and variables from the Ansible paths on {ProjectServer} or {SmartProxyServers} that have Ansible enabled.

.Prerequisites
* Ensure that the roles and variables that you import are located in the appropriate Ansible paths on all {SmartProxies} from where you want to use the roles.
For more information, see xref:ansible-role-and-collection-paths-in-{project-context}[].

.Procedure
. In the {ProjectWebUI}, navigate to *Configure* > *Ansible* > *Roles*.
. Click *Import* to select the {SmartProxy} from which you want to import.
. Select the roles that you want to import.
. Click *Submit*.
