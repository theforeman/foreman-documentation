= Removing a virt-who configuration

You can remove a `virt-who` configuration by deleting the configuration in the {ProjectWebUI} and then by deleting the configuration file from hypervisor or the virtualization manager.

.Procedure

. In the {ProjectWebUI}, navigate to *Infrastructure* > *Virt-who Configurations*.
. Select *Delete* from the *Actions* column of the .
. On the host that you want to remove the virt-who configuration from, remove the configuration file:
+
[options="nowrap" subs="+quotes"]
+
----
# rm /etc/virt-who.d/_conf_name_.conf
----
