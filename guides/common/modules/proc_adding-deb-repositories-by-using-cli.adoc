= Adding Deb repositories by using Hammer CLI

[role="_abstract"]
You can add Deb repositories to {Project} by using Hammer CLI.

.Prerequisites
* You have extracted and imported the GPG key that you want to use to verify signatures of the `Release` files.
For more information, see xref:Extracting_GPG_Public_Key_Fingerprints_from_Release_Files_{context}[].

.Procedure
* Create a Deb repository:
+
[options="nowrap" subs="+quotes"]
----
$ hammer repository create \
--content-type "deb" \
--deb-architectures "_My_Deb_Architectures_" \
--deb-components "_My_Deb_Components_" \
--deb-releases "_My_Deb_Releases_" \
--gpg-key-id "_My_GPG_Key_ID_" \
--name "_My_Repository_" \
--organization "_My_Organization_" \
--product "_My_Product_" \
--publish-via-http true \
--url _My_Upstream_URL_
----
