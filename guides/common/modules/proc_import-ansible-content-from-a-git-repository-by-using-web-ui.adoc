= Import Ansible content from a Git repository by using {ProjectWebUI}

[role="_abstract"]
You can import Ansible content from a Git repository into {Project} by using {ProjectWebUI}.
Select *Git repository* if you want to import Ansible content from a remote Git repository.

.Prerequisites
* Your {Project} account has a role that grants the `create_ansible_content`, `view_ansible_content`, and `view_organizations` permissions.

.Procedure
. In the {ProjectWebUI}, navigate to *Ansible* > *Content*.
. Click *Import Ansible content*.
. Select *Git repository* as source.

.Verification
. In the {ProjectWebUI}, navigate to *Ansible* > *Content*.
. For your imported Ansible content, click the *arrow* symbol to view the list of imported versions.
. Verify that {Project} contains all selected Ansible content and versions.
