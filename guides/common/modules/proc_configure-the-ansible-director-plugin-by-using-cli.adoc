= Configure the Ansible Director plugin by using Hammer CLI

[role="_abstract"]
Configure the Ansible Director plugin to adjust to your {Project} environment.

.Procedure
* Configure the Ansible director plugin to your environment.
For example, you can change the default version of `ansible-core`:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
$ hammer setting set \
--name ad_default_ansible_core_version \
--value "{ansible-director-ansible-core-version}"
----

.Verification
* Verify that {Project} updated the setting:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
$ hammer setting info --name ad_default_ansible_core_version
----
