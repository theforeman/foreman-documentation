= Publishing the content view

[role="_abstract"]
Publish content views to create new versions with updated repository snapshots for distribution to hosts.
Publishing enables version control and ensures hosts consume consistent, tested content.

.Procedure
. In the {ProjectWebUI}, navigate to *Content* > *Lifecycle* > *Content Views*.
. Select the content view.
. Click *Publish New Version* on the top right of the page.
. Click *Save*.
