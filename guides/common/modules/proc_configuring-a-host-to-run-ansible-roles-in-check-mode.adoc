= Configuring a host to run Ansible roles in check mode

[role="_abstract"]
Run Ansible roles in check mode through the {ProjectWebUI} to preview changes before applying them.
This helps you reduce risk when validating role behavior on a host.

.Procedure
. In the {ProjectWebUI}, navigate to *Hosts* > *All Hosts*.
. Click *Edit* for the host you want to enable check mode for.
. In the *Parameters* tab, ensure that the host has a parameter named `ansible_roles_check_mode` with type `boolean` set to `true`.
. Click *Submit*.
