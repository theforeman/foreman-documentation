= Adding errata to an incremental content view by using MCP

[role="_abstract"]
You can use your AI application to add errata to an incremental content view.

[WARNING]
====
Always review content generated by AI before using it in production.
====

.Prerequisites
* An AI application connected to the {Project} MCP server.
For more information, see {ManagingHostsDocURL}connecting-ai-applications-to-the-mcp-server-for-satellite[Deploying the MCP server for {Project}] in _{ManagingHostsDocTitle}_.

.Procedure
. Connect your AI application to the {Project} MCP server.
For more information, see connecting-ai-applications-to-the-mcp-server-for-satellite
. Enable content view actions:
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
$ podman run --interactive --tty --publish 127.0.0.1:8080:8080 \
  --volume _Path_to_My_CA_Bundle_:/app/ca.pem:ro,Z \
  {mcp-server-image} \
  --foreman-url https://_{foreman-example-com}_ \
  --allowed-cv-actions "incremental_update"
----
+
. Type a prompt in your AI application to list applicable errata.
[options="nowrap", subs="+quotes,attributes"]
+
----
Show a list of applicable errata on my {Project}.
----
. The MCP server will return a list of applicable errata to the AI application.
. Review the information and if it is correct, use the prompt to apply errata to the hosts.
The {Project} MCP server provides a prompt for this purpose:
+
[options="nowrap", subs="+quotes,attributes"]
----
Add Errata to Content Views
----
Review all steps that the AI application performs.
