= Uninstalling the Microsoft Azure plugin

[role="_abstract"]
If you previously installed the Microsoft Azure plugin but do not use it anymore to manage and deploy hosts to Azure, you can uninstall it from your {ProjectServer}.

.Procedure
* Uninstall the Azure plugin from your {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {project-package-remove} rubygem-foreman_azure_rm rubygem-ms_rest_azure && {foreman-installer} --no-enable-foreman-plugin-azure
----

.Verification
* In the {ProjectWebUI}, navigate to *Administer* > *About*.
. Select the *Available Providers* tab to verify the removal of the Microsoft Azure plugin.
