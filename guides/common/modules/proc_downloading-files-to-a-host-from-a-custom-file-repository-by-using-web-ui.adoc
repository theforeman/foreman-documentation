= Downloading files to a host from a {customfiletype} repository by using {ProjectWebUI}

[role="_abstract"]
You can download files to a host over HTTPS and optionally over HTTP if the *Unprotected* option for the repository is selected.
Use {ProjectWebUI} to find the download URL of the {customfiletype} repository.

.Prerequisites
* You have a {customfiletype} repository.
For more information, see {ContentManagementDocURL}creating-a-custom-file-type-repository-by-using-web-ui[Creating a {customfiletype} repository by using {ProjectWebUI}] in _{ContentManagementDocTitle}_.
* You know the name of the file you want to download.
* To download the files over HTTPS, your host has the following certificates:
** The `katello-server-ca.crt` certificate.
For more information, see {ConfiguringUserAuthenticationDocURL}importing-the-katello-root-ca-certificate-using-cli[Importing the Katello root CA certificate by using CLI] in _{ConfiguringUserAuthenticationDocTitle}_.
** An Organization Debug Certificate.
ifndef::satellite[]
For more information, see {ManagingOrganizationsLocationsDocURL}Creating_an_Organization_Debug_Certificate_managing-organizations-locations[Creating an Organization Debug Certificate] in _{ManagingOrganizationsLocationsDocTitle}_.
endif::[]
ifdef::satellite[]
For more information, see {AdministeringDocURL}Creating_an_Organization_Debug_Certificate_admin[Creating an Organization Debug Certificate] in _{AdministeringDocTitle}_.
endif::[]

.Procedure
. In the {ProjectWebUI}, navigate to *Content* > *Products*.
. Select a {customproduct} by name.
. Select a file type repository by name.
. Ensure to select the *Unprotected* checkbox to access the repository published through HTTP.
. Copy the *Published At* URL.
