= Disabling SSH host certificate verification on {SmartProxy}

[role="_abstract"]
If you disable SSH host certificate verification on {SmartProxy}, you will revert to the Trust On First Use (TOFU) model.

.Procedure
* On {SmartProxy}, disable SSH host certificate verification:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} \
--reset-foreman-proxy-plugin-remote-execution-script-ssh-host-ca-public-keys-file
----
