= Consuming content from an Ansible Collection repository

[role="_abstract"]
Your hosts can consume content from an Ansible Collection repository on {ProjectServer} or {SmartProxyServer}.

.Prerequisites
* You have synchronized an Ansible Collection to a {Project} repository.
* The `ansible-core` package is installed on the host.
ifdef::client-content-dnf[]
This package is available in AppStream repositories.
endif::[]

.Procedure
* On the host, install an Ansible Collection from the repository.
Use the `/pulp_ansible/galaxy/` path instead of the `/pulp/content/` path:
+
[options="nowrap", subs="+quotes,attributes,verbatim"]
----
# ansible-galaxy collection install _my_namespace.my_collection_ \
--server https://_{server-example-com}_/pulp_ansible/galaxy/_My_Organization_Label_/Library/custom/_My_Product_Label_/_My_Repository_Label_/api/
----
+
Replace `{server-example-com}` with the fully qualified domain name (FQDN) of your {ProjectServer} or {SmartProxyServer}.
