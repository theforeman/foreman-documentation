= Enabling {the-Cockpit} on {Project}

[role="_abstract"]
You can enable {the-Cockpit} on {ProjectServer} so that you can access {Cockpit} features for your hosts from within the {ProjectWebUI}.

.Prerequisites
* You have root or sudo access on {ProjectServer}.

.Procedure
* Enable {the-Cockpit} on your {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --enable-foreman-plugin-remote-execution-cockpit --reset-foreman-plugin-remote-execution-cockpit-ensure
----
