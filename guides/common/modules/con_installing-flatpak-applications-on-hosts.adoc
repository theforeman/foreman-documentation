= Installing Flatpak applications on hosts

[role="_abstract"]
After you mirror Flatpak repositories in {Project}, you can install Flatpak applications on hosts from {ProjectServer} or {SmartProxyServer}.
Hosts use the command line to add a Flatpak remote, authenticate to the container registry, and install applications.
