= Provisioning virtual machines in Proxmox

[role="_abstract"]
You can create and manage virtual machines on Proxmox Virtual Environment by using {Project}.
Proxmox integrates the KVM hypervisor and Linux Containers (LXC).

{Project} supports Proxmox 8 and 9.
