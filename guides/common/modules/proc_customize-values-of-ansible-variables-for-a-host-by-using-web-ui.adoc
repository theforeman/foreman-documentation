= Customize values of Ansible variables for a host by using {ProjectWebUI}

[role="_abstract"]
You can set custom values of Ansible variables for a host by using {ProjectWebUI}.
Setting a value for a specific host overrides all other levels and is helpful to test and debug host configurations.

.Prerequisites
* Your {Project} account has a role that grants the `create_ansible_variable_overrides`, `destroy_ansible_variable_overrides`, `destroy_ansible_variables`, `edit_ansible_variable_overrides`, `edit_ansible_variables`, `view_ansible_content`, `view_ansible_variable_overrides`, `view_ansible_variables`, and `view_organizations` permissions.
* Your host has an Ansible environment assigned to it.

.Procedure
. In the {ProjectWebUI}, navigate to *Hosts* > *All Hosts*.
. Select your host.
. On the *Ansible Director* tab, select the *Variables* tab.
. From the list of customizable variables, click the *Edit* icon for the variable that you want to override.
. In the *Edit override* window, set the custom value for your host.
. Click *Confirm* to submit your change to {Project}.

.Verification
. In the {ProjectWebUI}, navigate to *Hosts* > *All Hosts*.
. Select your host.
. On the *Ansible Director* tab, select the *Variables* tab.
. From the list of customizable variables, click the *Edit* icon for the variable that you want to override.

.Next steps
* Apply the configuration to your host.
