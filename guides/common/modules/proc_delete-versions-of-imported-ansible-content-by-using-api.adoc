= Delete versions of imported Ansible content by using {Project} API

[role="_abstract"]
You can delete versions of imported Ansible content from {Project} by using {Project} API.
This procedure deletes specific versions of your imported Ansible content from {Project}.

.Prerequisites
* Your {Project} account has a role that grants the `destroy_ansible_content`, `view_ansible_content`, and `view_organizations` permissions.

.Procedure
. Optional: List imported Ansible content:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ curl \
--header "Accept: application/json" \
--user "_My_User_Name_:__My_Password__" \
https://{foreman-example-com}/api/v2/ansible_director/ansible_content/ \
| jq ".results | .[]"
----
. Create a JSON file, for example, `data.json` to define the versions of your Ansible content that you want to delete:
+
[source, json, options="nowrap", subs="verbatim,quotes,attributes"]
----
{
  "units": [
    {
      "unit_id": "_My_Ansible_Content_ID_",
      "unit_version_ids": [
        "_My_First_Version_ID_",
        "_My_Second_Version_ID_"
      ]
    }
  ]
}
----
. Delete versions of imported Ansible content from {Project}:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ curl \
--data @data.json \
--header "Accept: application/json" \
--header "Content-Type: application/json" \
--request DELETE \
--user "_My_User_Name_:__My_Password__" \
https://{foreman-example-com}/api/v2/ansible_director/ansible_content/ \
| jq
----

.Verification
* List all versions of imported Ansible content:
+
[options="nowrap", subs="verbatim,quotes,attributes"]
----
$ curl \
--header "Accept: application/json" \
--user "_My_User_Name_:__My_Password__" \
https://{foreman-example-com}/api/v2/ansible_director/ansible_content/ \
| jq ".results | .[]"
----
+
Verify that the versions that you have deleted are not part of the output.
