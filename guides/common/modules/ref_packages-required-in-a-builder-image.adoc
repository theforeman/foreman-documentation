= Packages required in a {LoraxCompose} image

For the use in {Project}, the {LoraxCompose} image must contain the following minimum of packages:

* `NetworkManager`
* `authselect`
* `dnf`
* `dracut`
* `efibootmgr`
* `firewalld`
* `grub2`
* `grub2-efi`
* `grub2-tools-extra`
* `insights-client`
* `iproute`
* `iputils`
* `kernel`
* `kernel-core`
* `lvm2`
* `subscription-manager`
* `wget`
