= Creating {customproduct}s

ifdef::satellite[]
[role="_abstract"]
Create a {customproduct} in {Project} whenever you curate non-Red Hat Yum repositories so you can sync, publish, and entitle third-party packages separately from Red{nbsp}Hat products.
endif::[]
ifndef::satellite[]
[role="_abstract"]
Create a {customproduct} in {Project} whenever you curate custom Yum, Deb, or vendor repositories so you can sync and publish them under their own subscription-backed product.

[NOTE]
====
For Red{nbsp}Hat content, {Project} automatically creates a product when you enable a repository.
====
endif::[]
