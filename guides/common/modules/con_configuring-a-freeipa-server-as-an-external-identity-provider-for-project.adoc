= Configuring a {FreeIPA} server as an external identity provider for {Project}

{FreeIPA} is an open-source identity management solution that provides centralized authentication, authorization, and account management services.
With {Project}, you can integrate {ProjectServer} with your existing {IPA} server to enable {IPA} users to authenticate to {Project}.

With {aIPA} server configured as an external identity provider, users defined in {IPA} can log in to {Project} with their {IPA} credentials.
{IPA} users can log in to {Project} by providing username and password or by using Kerberos single sign-on.
