= Deleting a resource quota by using {ProjectWebUI}

[role="_abstract"]
You can delete a resource quota from {Project} by using {ProjectWebUI}.

.Procedure
. In the {ProjectWebUI}, navigate to *Configure* > *Resource Quotas*.
. In the *Actions* column, click *Delete* for your resource quota.
