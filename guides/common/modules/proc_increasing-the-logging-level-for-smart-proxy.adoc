= Increasing the logging level for {SmartProxy}

[role="_abstract"]
Increase {SmartProxy} logging to `debug` to capture detailed {SmartProxy} activity when troubleshooting communication between {Project} and hosts.
Reset the level when debugging is complete to avoid excessive log volume.

By default, {SmartProxy} writes logs to `/var/log/foreman-proxy/proxy.log`.
For more information, see xref:configuring-logging-type-and-layout[].

.Procedure
. Set the logging level to `debug`:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --foreman-proxy-log-level DEBUG
----
. After you complete debugging, reset the logging level to the default value:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --reset-foreman-proxy-log-level
----
