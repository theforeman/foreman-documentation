= Install the Ansible Director plugin on {ProjectServer}

[role="_abstract"]
Install the Ansible Director plugin on your {ProjectServer} to configure hosts with platform-independent Ansible and Python versions.

.Procedure
. On your {ProjectServer}, install the Ansible Director plugin:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} \
--enable-foreman-plugin-ansible-director \
--enable-foreman-proxy-plugin-ansible-director
----
. Optional: Install the Hammer CLI plugin for Ansible Director:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --enable-foreman-cli-ansible-director
----

.Verification
. In the {ProjectWebUI}, navigate to *Administer* > *About*.
. On the *Plugins* tab, verify the installation of the Ansible Director plugin.
