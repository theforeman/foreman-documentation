= Installing packages on a host

[role="_abstract"]
You can review and install packages on a host from the {ProjectWebUI} so that the host has the software it needs.
Available packages depend on the content view and lifecycle environment assigned to the host.

.Prerequisites
* The host is assigned to a content view and lifecycle environment.

.Procedure
. In the {ProjectWebUI}, navigate to *Hosts* > *All Hosts*.
. Select a host.
. On the *Content* tab, click the *Packages* tab.
. On the vertical ellipsis icon next to the upgrade button, click *Install packages*.
. In the *Install packages* window, select the package or packages that you want to install on the host.
. Click *Install*.
The {ProjectWebUI} shows a notification for the remote execution job.
