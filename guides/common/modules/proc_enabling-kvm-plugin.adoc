= Enabling KVM plugin

Install the KVM plugin to attach a KVM/libvirt compute resource provider to {Project}.
This allows you to manage and deploy hosts to KVM/libvirt.

.Procedure
. Install the KVM plugin on your {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --enable-foreman-compute-libvirt
----
. Optional: In the {ProjectWebUI}, navigate to *Administer* > *About* and select the *Available Providers* tab to verify that the KVM plugin is installed.

.Next steps
* Continue with connecting your {Project} to your KVM compute resource.
For more information, see xref:configuring-server-for-kvm-connections_kvm-provisioning[].
