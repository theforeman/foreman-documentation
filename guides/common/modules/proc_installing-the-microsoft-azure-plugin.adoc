= Installing the Microsoft Azure plugin

[role="_abstract"]
Install the Microsoft Azure plugin to attach an Azure compute resource provider to {Project}.
This allows you to manage and deploy hosts to Azure.

.Procedure
* Install the Microsoft Azure plugin on your {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-installer} --enable-foreman-plugin-azure
----

.Verification
. In the {ProjectWebUI}, navigate to *Administer* > *About*.
. Select the *compute resources* tab to verify the installation of the Microsoft Azure plugin.
