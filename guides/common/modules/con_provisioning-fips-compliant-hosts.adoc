= Provisioning FIPS-compliant hosts

[role="_abstract"]
{Project} supports provisioning hosts that comply with the National Institute of Standards and Technology's Security Requirements for Cryptographic Modules standard, reference number FIPS 140-2, referred to here as FIPS.
