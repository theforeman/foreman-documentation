= Creating a user group by using Hammer CLI

[role="_abstract"]
Create a user group by using Hammer CLI and define which roles the group will have.

.Procedure
* Create a user group:
+
[options="nowrap", subs="+quotes,attributes"]
----
$ hammer user-group create \
--name _My_User_Group_Name_ \
--role-ids _My_Role_ID_1_,_My_Role_ID_2_ \
--user-ids _My_User_ID_1_,_My_User_ID_2_
----
