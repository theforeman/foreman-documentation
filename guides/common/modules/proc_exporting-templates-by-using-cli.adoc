= Exporting templates by using Hammer CLI

[role="_abstract"]
You can export templates to an existing repository by using Hammer CLI.
This is useful for backing up customized templates and keeping them synchronized with your version control system.

.Procedure
* Export all templates to a repository:
+
[options="nowrap", subs="+quotes,attributes"]
----
$ hammer export-templates \
--organization "_My_Organization_" \
--repo "_https://git.example.com/path/to/repository_"
----
+
[NOTE]
====
This command clones the repository, makes changes in a commit, and pushes back to the repository.
You can use the `--branch "_My_Branch_"` option to export the templates to a specific branch.
====
