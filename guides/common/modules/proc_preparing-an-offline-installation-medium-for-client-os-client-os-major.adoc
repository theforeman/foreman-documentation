= Preparing an offline installation medium for {client-os} {client-os-major}

[role="_abstract"]
You can add an installation medium for {client-os} to {Project} to provision hosts in disconnected environments.

ifdef::orcharhino[]
include::snip_file-type-repositories-for-installation-media-for-debian-and-ubuntu.adoc[]
endif::[]

[NOTE]
====
This procedure describes distributing the ISO image through {ProjectServer}.
You can also use {SmartProxyServers} or any arbitrary web server that is reachable from your {ProjectServer}.
If you use an alternative web server, adjust the path of the installation medium accordingly.
====

.Prerequisites
ifdef::orcharhino[]
* You have selected at least one {client-os} variant from the list of operating systems during the {Project} installation.
endif::[]
* You have obtained the ISO image for {client-os} {client-os-major}.
* You have created the necessary operating system entries in {Project}.
+
In the {ProjectWebUI}, navigate to *Hosts* > *Provisioning Setup* > *Installation Media* and ensure there are entries for {client-os} {client-os-major}.
* You have installed `p7zip` and `p7zip-plugins` on your {ProjectServer}.

.Procedure
. Copy the {client-os} {client-os-major} ISO image to your {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
$ scp {client-os-context}-{client-os-major-context}.iso root@{foreman-example-com}:/tmp/
----
. Create the directory for your {client-os} medium:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# mkdir --parents /var/www/html/pub/installation_media/{client-target-directory}/
----
. Extract the {client-os} {client-os-major} ISO image to the newly created directory:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# 7z x /tmp/{client-os-context}-{client-os-major-context}.iso -o /var/www/html/pub/installation_media/{client-target-directory}/
----
. Ensure the permissions of your extracted files allow the `apache` user to read them:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# chmod -R go+X+r /var/www/html/pub/installation_media/{client-target-directory}/*
----
. Remove the {client-os} {client-os-major} ISO image from your {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# rm --force /tmp/{client-os-context}-{client-os-major-context}.iso
----

.Verification
* Navigate to `\https://{foreman-example-com}/pub/installation_media/{client-target-directory}/` and verify that the extracted files are available.
