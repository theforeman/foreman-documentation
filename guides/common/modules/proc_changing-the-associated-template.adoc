= Changing the associated template

[role="_abstract"]
You can change the associated template to customize remote execution workflows.

.Procedure
. In the {ProjectWebUI}, navigate to *Administer* > *Remote Execution Features*.
. Select the remote execution feature of which you want to change the associated template.
. In the *Job Template* menu, select a different job template.
+
[CAUTION]
====
Changing the default job templates might break existing {Project} workflows.
Evaluate carefully why you want to change the default job template.

Cloned templates do not receive updates from {Team} when upgrading {Project}.
====
. Click *Submit*.
