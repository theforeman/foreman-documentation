= Upgrade puppet-agent to openvox-agent

[role="_abstract"]
If you use puppet for host management, you have the option to upgrade `puppet-agent`, installed on your hosts, to `openvox-agent`.

.Prerequisites
* *{project-client-name} OpenVox* repository is enabled on the host.

.Procedure
* On your hosts, upgrade the `puppet-agent` agent package to `openvox-agent`:
+
[options="nowrap" subs="+quotes,verbatim,attributes"]
----
# {project-package-update} puppet-agent
----
