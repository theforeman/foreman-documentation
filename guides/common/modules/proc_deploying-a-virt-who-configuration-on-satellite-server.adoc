= Deploying a virt-who configuration on {ProjectServer}

For VMware vSphere and Microsoft Hyper-V, you deploy the virt-who configuration on {ProjectServer}.

Global configuration values apply to all virt-who configurations on {ProjectServer} and are overwritten if you deploy a new virt-who configuration.

.Prerequisites
* Microsoft Hyper-V only: You have enabled remote management on the hypervisor.

.Procedure
. In the {ProjectWebUI}, navigate to *Infrastructure* > *Virt-who Configurations*.
. Click a virt-who configuration.
. Under *Hammer command*, click *Copy to clipboard*.
. Log in to {ProjectServer} and paste the Hammer command into your terminal.
