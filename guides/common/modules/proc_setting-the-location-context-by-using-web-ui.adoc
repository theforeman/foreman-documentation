= Setting the location context by using {ProjectWebUI}

[role="_abstract"]
Set the location context in the {ProjectWebUI} to define which location applies to hosts and their associated resources.

.Procedure
. In the {ProjectWebUI}, navigate to the top menu bar.
The location menu is the second menu item in the menu bar, on the upper left of the {ProjectWebUI}.
. If you have not selected a current location, the menu displays *Any Location*.
Click *Any location* and select the location to use.
