= Viewing resource quotas by using Hammer CLI

[role="_abstract"]
You can use Hammer CLI to view the resource quota of hosts.

.Procedure
. Optional: Retrieve the ID of your host:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
$ hammer host list
----
. Display the resource quota assigned to your host:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
$ hammer host info \
--fields "Id,Name,Resource quota id" \
--id _My_Host_ID_
----
