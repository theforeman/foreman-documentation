= Delete an Ansible lifecycle environment by using Hammer CLI

[role="_abstract"]
You can delete an Ansible lifecycle environment by using Hammer CLI.

.Prerequisites
* You have installed the Hammer CLI plugin for Ansible Director.
For more information, see xref:install-the-hammer-cli-plugin-for-ansible-director[].

.Procedure
* Delete an Ansible lifecycle environment from {Project}:
+
----
$ hammer
----

.Verification
* Verify that your Ansible lifecycle environment path no longer contains the previously deleted Ansible lifecycle environment:
+
----
$ hammer
----
