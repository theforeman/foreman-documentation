= Apply Ansible-based configuration to a host by using Hammer CLI

[role="_abstract"]
You can apply Ansible-based configuration to your host by using Hammer CLI.

.Prerequisites
// TODO: Fix list of permissions
* Your {Project} account has a role that grants the `create_ansible_lifecycle_environment_paths`, `edit_ansible_lifecycle_environment_paths`, `view_ansible_content`, `view_ansible_lifecycle_environment_paths`, `view_ansible_lifecycle_environments`, and `view_organizations` permissions.
* Your host has Ansible roles assigned to it.
* You have installed the Hammer CLI plugin for Ansible Director.
For more information, see xref:install-the-hammer-cli-plugin-for-ansible-director[].

.Procedure
* Apply Ansible-based configuration to a host:
+
----
$ hammer --output json job-invocation create \
--async \
--job-template "Apply Ansible configuration" \
--search-query "_My_Search_Query_" \
| jq ".id"
----
+
Note the ID of the job.

.Verification
* Verify that {Project} finished the remote execution job successfully:
+
----
$ hammer --output=json job-invocation info --id _My_Job_ID_ \
| jq ".Status"
----
+
Verify that the status is `succeeded`.
