= Change an Ansible Execution Environment by using Hammer CLI

[role="_abstract"]
You can change an Ansible Execution Environment in {Project} by using Hammer CLI.
You can change the name, base container image, or version of `ansible-core`.

.Prerequisites
* Your {Project} account has a role that grants the `create_ansible_execution_environments`, `edit_ansible_execution_environments`, `view_ansible_content`, `view_ansible_execution_environments`, and `view_organizations` permissions.
* You have installed the Hammer CLI plugin for Ansible Director.
For more information, see xref:install-the-hammer-cli-plugin-for-ansible-director[].

.Procedure
* Change your Ansible Execution Environment:
+
----
$ hammer
----

.Verification
* Verify that {Project} built the newly modified Ansible Execution Environment successfully:
+
----
$ hammer
----
