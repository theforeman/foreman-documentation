= Increasing the logging level for Hammer

You can find the log in `~/.hammer/log/hammer.log`.

.Procedure

* In `/etc/hammer/cli_config.yml`, set the `:log_level:` option to `debug`:

[options="nowrap", subs="+quotes,verbatim,attributes"]
----
:log_level: 'debug'
----
