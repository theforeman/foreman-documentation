= Uploading content to a Python repository by using {ProjectWebUI}

[role="_abstract"]
You can upload files from your local machine to a Python repository in {Project} when you need to distribute Python content that you cannot synchronize.

.Prerequisites
* You have a `.whl` package or `.tar.gz` archive that contains Python packages.

.Procedure
. In the {ProjectWebUI}, navigate to *Content* > *Products*.
. Select a {customproduct} by name.
. Select a Python type repository by name.
. Click *Browse* to search and select the file you want to upload.
. Click *Upload* to upload the selected file to {ProjectServer}.

.Verification
* Visit the *Published At* URL to see the file.
