= Assigning roles to a user

[role="_abstract"]
Use this procedure to assign roles to a user by using Hammer CLI.

.Procedure
* Assign roles to a user:
+
[options="nowrap", subs="+quotes,attributes"]
----
$ hammer user add-role \
--id _My_User_ID_ \
--role _My_Role_Name_
----
