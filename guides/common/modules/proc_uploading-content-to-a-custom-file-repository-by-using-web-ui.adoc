= Uploading content to a {customfiletype} repository by using {ProjectWebUI}

[role="_abstract"]
If you cannot create a `PULP_MANIFEST` and synchronize your content from a local or remote repository to {Project}, you can manually upload content to a {customfiletype} repository.

.Procedure
. In the {ProjectWebUI}, navigate to *Content* > *Products*.
. Select a {customproduct} by name.
. Select a file type repository by name.
. Click *Browse* to search and select the file you want to upload.
. Click *Upload* to upload the selected file to {ProjectServer}.
. Visit the URL where the repository is published to see the file.
