= Creating architectures by using Hammer CLI

[role="_abstract"]
You can group hosts and operating systems in {Project} by creating custom architectures.
{Project} automatically creates architectures when hosts check in with Puppet, and x86_64 is already preset.

ifdef::satellite[]
[NOTE]
====
Only the Intel x86_64 architecture is supported for provisioning using PXE, Discovery, and boot disk.
For more information, see the Red Hat Knowledgebase solution https://access.redhat.com/solutions/2674001[Supported architectures and provisioning scenarios in Satellite 6].
====
endif::[]

.Procedure
* Create an architecture:
+
[options="nowrap" subs="+quotes"]
----
$ hammer architecture create \
--name "_My_Architecture_" \
--operatingsystems "_My_Operating_System_"
----
