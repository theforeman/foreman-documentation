= Import Ansible content based on a YML file by using Hammer CLI

[role="_abstract"]
You can import Ansible content based on a YML file into {Project} by using {ProjectWebUI}.
Select *YML file* if you want to paste a `requirements.yml` file into the {ProjectWebUI}.

.Prerequisites
* Your {Project} account has a role that grants the `create_ansible_content`, `view_ansible_content`, and `view_organizations` permissions.
* You have installed the Hammer CLI plugin for Ansible Director.
For more information, see xref:install-the-hammer-cli-plugin-for-ansible-director[].

.Procedure
* Import Ansible content based on a YML file:
+
----
$ hammer
----

.Verification
* Verify that {Project} imported the Ansible content successfully:
+
----
$ hammer
----
