= Disassociating a VM from {Project} without removing it from a hypervisor

[role="_abstract"]
You can disassociate a virtual machine (VM) from {Project} if you no longer want to manage it.
This does not delete the VM from its hypervisor.

.Procedure
. In the {ProjectWebUI}, navigate to *Hosts* > *All Hosts*.
. Select the hosts that you want to disassociate.
. From the options menu, select *Disassociate hosts*.
. Click *Disassociate*.
