= Configuring {keycloak-wildfly} authentication for {Project}

{keycloak-wildfly} is an open-source identity and access management solution that provides authentication features, such as single sign-on functionality, user federation, or centralized authentication management.
With {keycloak-wildfly}, you can integrate {ProjectServer} with your existing {keycloak-wildfly} server to delegate user authentication and authorization to {keycloak-wildfly}.
