[id="performing-a-full-backup"]
= Performing a full backup

[role="_abstract"]
ifdef::containerized[]
Run a full offline backup on {ProjectServer} to capture all {Project} content and configuration.
endif::[]
ifndef::containerized[]
Run a full offline backup on {ProjectServer} and {SmartProxyServer} to capture all {Project} content and configuration.
endif::[]
A full backup is useful when you want to prepare for a future restore from scratch.

.Prerequisites
* Your backup location must have sufficient available disk space to store the backup.
ifdef::katello,orcharhino,satellite[]
For more information, see xref:estimating-the-size-of-a-backup[].
endif::[]
ifdef::containerized[]
* To enable {Project} to save the backup to an NFS share, the `root` user of your {ProjectServer} must be able to write to the NFS share.
endif::[]
ifndef::containerized[]
* To enable {Project} to save the backup to an NFS share, the `root` user of your {ProjectServer} or {SmartProxyServer} must be able to write to the NFS share.
endif::[]
NFS export options such as `root_squash` and `all_squash` are known to prevent this.
ifndef::orcharhino[]
For more information, see link:{RHELDocsBaseURL}9/html/configuring_and_using_network_file_services/deploying-an-nfs-server_configuring-and-using-network-file-services[{RHEL} _Configuring and using network files services_] and link:{RHELDocsBaseURL}9/html/securing_networks/securing-network-services_securing-networks#export-options-for-securing-an-nfs-server_securing-the-nfs-service[{RHEL} _Securing network services_].
endif::[]

[WARNING]
====
ifdef::containerized[]
Request other users of {ProjectServer} to save any changes and warn them that {Project} services are unavailable for the duration of the backup.
endif::[]
ifndef::containerized[]
Request other users of {ProjectServer} or {SmartProxyServer} to save any changes and warn them that {Project} services are unavailable for the duration of the backup.
endif::[]
Ensure no other tasks are scheduled for the same time as the backup.
====

.Procedure
ifdef::containerized[]
* Back up your {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foremanctl} backup _/var/{project-context}-backup_
----
endif::[]
ifndef::containerized[]
. Back up your {ProjectServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-maintain} backup offline _/var/{project-context}-backup_
----
. Back up your {SmartProxyServer}:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
# {foreman-maintain} backup offline _/var/foreman-proxy-backup_
----
endif::[]
