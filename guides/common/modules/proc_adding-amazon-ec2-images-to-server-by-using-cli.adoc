= Adding Amazon EC2 images to {ProjectServer} by using Hammer CLI

[role="_abstract"]
You must add Amazon EC2 image details to {ProjectServer} for image-based provisioning.

.Procedure
* Create the image:
+
[options="nowrap" subs="+quotes"]
----
$ hammer compute-resource image create \
--architecture "_My_Architecture_" \
--compute-resource "_My_EC2_Compute_Resource_" \
--name "_My_Amazon_EC2_Image_" \
--operatingsystem "_My_Operating_System_" \
--user-data true \
--username root \
--uuid "ami-_My_AMI_ID_"
----
