= Import Ansible content from Ansible Galaxy by using Hammer CLI

[role="_abstract"]
You can import Ansible content from Ansible Galaxy into {Project} by using Hammer CLI.
Select Ansible Galaxy as content source if you want to import Ansible content from the Ansible Galaxy, from a resource that implements the Ansible Galaxy API, from a Pulp instance, or from a content view.

.Prerequisites
* Your {Project} account has a role that grants the `create_ansible_content`, `view_ansible_content`, and `view_organizations` permissions.
* You have installed the Hammer CLI plugin for Ansible Director.
For more information, see xref:install-the-hammer-cli-plugin-for-ansible-director[].

.Procedure
* Import Ansible content from Ansible Galaxy:
+
----
$ hammer
----

.Verification
* Verify that {Project} imported the Ansible content successfully:
+
----
$ hammer
----
