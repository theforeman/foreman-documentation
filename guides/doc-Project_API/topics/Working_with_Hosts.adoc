=== Working with hosts

include::snip_note-python3-vs-python.adoc[]

.Listing hosts

This example returns a list of Satellite hosts.

Example request:
[options="nowrap" subs="+quotes"]
----
$ curl --request GET --insecure --user _sat_username:sat_password_ \
https://_satellite.example.com_/api/v2/hosts | python3 -m json.tool
----

Example response:
[options="nowrap" subs="+quotes"]
----
{
      ...
       "total" => 2,
    "subtotal" => 2,
        "page" => 1,
    "per_page" => 1000,
      "search" => nil,
        "sort" => {
           "by" => nil,
        "order" => nil
    },
     "results" => [
      ...
}
----

.Requesting information for a host

This request returns information for the host `satellite.example.com`.

Example request:
[options="nowrap" subs="+quotes"]
----
$  curl --request GET --insecure --user _sat_username:sat_password_ \
https://_satellite.example.com_/api/v2/hosts/_satellite.example.com_ \
| python3 -m json.tool
----

Example response:
[options="nowrap" subs="+quotes"]
----
{
    "all_puppetclasses": [],
    "architecture_id": 1,
    "architecture_name": "x86_64",
    "build": false,
    "capabilities": [
        "build"
    ],
    "certname": "_satellite.example.com_",
    "comment": null,
    "compute_profile_id": null,
    ...
}
----

This request returns all facts for the host `satellite.example.com`.

Example request:
[options="nowrap" subs="+quotes"]
----
$ curl --request GET --insecure --user _sat_username:sat_password_ \
https://_satellite.example.com_/api/v2/hosts/_satellite.example.com_/facts \

{
    ...

----

.Searching for hosts with matching patterns

This query returns all hosts that match the pattern "example".

Example request:
[options="nowrap" subs="+quotes"]
----
$ curl --request GET --insecure --user _sat_username:sat_password_ \
https://_satellite.example.com_/api/v2/hosts?search=example \
| python3 -m json.tool
----

Example response:
[options="nowrap" subs="+quotes"]
----
{
    ...
    "results": [
        {
            "name": "_satellite.example.com_",
            ...
        }
    ],
    "search": "example",
    ...
}
----

.Searching for hosts in an environment

This query returns all hosts in the `production` environment.

Example request:
[options="nowrap" subs="+quotes"]
----
$ curl --request GET --insecure --user _sat_username:sat_password_ \
https://_satellite.example.com_/api/v2/hosts?search=environment=production \

Example response:
[options="nowrap" subs="+quotes"]
----

    ...
    "results": [
        {
            "environment_name": "production",

            ...
        }

    ...

----

.Searching for hosts with a specific fact value

Example request:
[options="nowrap" subs="+quotes"]
----

https://_satellite.example.com_/api/v2/hosts?search=model=\"RHEV+Hypervisor\" \

----
{
    ...
    "results": [
        {
            "model_id": 1,
            "model_name": "RHEV Hypervisor",
            "name": "_satellite.example.com_",
            ...
        }
    ],
    "search": "model=\"RHEV Hypervisor\"",
    ...

----

.Deleting a host

This request deletes a host with a name _host1.example.com_.

Example request:
[options="nowrap" subs="+quotes"]
----
$ curl --request DELETE --insecure --user _sat_username:sat_password_ \
https://_satellite.example.com_/api/v2/hosts/_host1.example.com_ \
| python3 -m json.tool
----

.Downloading a full boot disk image

This request downloads a full boot disk image for a host by its ID.

Example request:
[options="nowrap" subs="+quotes"]
----
$ curl --request GET --insecure --user _sat_username:sat_password_ \
https://_satellite.example.com_/api/bootdisk/hosts/_host_ID_?full=true \
--output _image_.iso
----
